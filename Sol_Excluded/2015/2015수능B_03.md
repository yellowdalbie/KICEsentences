## [Step 1] 삼각함수의 합성 공식을 활용하여 함수 식 변형하기

- **Trigger**: [함수 형태] $f(x)=\sin x+\sqrt{7} \cos x-\sqrt{2}$ 분석
- **Action**: [12미적02-01] 삼각함수의 합성 공식($a \sin x + b \cos x = \sqrt{a^2+b^2} \sin(x+\alpha)$)을 적용하여 사인의 단일항으로 변환
- **Result**: $f(x) = \sqrt{1^2 + (\sqrt{7})^2} \sin(x+\alpha) - \sqrt{2} = 2\sqrt{2} \sin(x+\alpha) - \sqrt{2}$ 구조 확보

> **📝 해설 (Explanation)**
> 삼각함수의 합성을 이용하여 함수를 하나의 사인함수로 나타냅니다.
> $f(x) = \sin x + \sqrt{7} \cos x - \sqrt{2} = \sqrt{1^2 + (\sqrt{7})^2} \sin(x+\alpha) - \sqrt{2}$
> 정리하면 $f(x) = \sqrt{8} \sin(x+\alpha) - \sqrt{2} = 2\sqrt{2} \sin(x+\alpha) - \sqrt{2}$ 입니다. (단, $\cos \alpha = \frac{1}{2\sqrt{2}}, \sin \alpha = \frac{\sqrt{7}}{2\sqrt{2}}$)

## [Step 2] 사인의 치역 범위를 고려하여 함수의 최댓값 산출하기

- **Trigger**: [수치 연산] $f(x)$의 최댓값 특정
- **Action**: [12미적02-01] $\sin(x+\alpha)$의 최댓값 1을 대입하여 전체 함숫값의 상한선을 계산
- **Result**: 최댓값 $2\sqrt{2} - \sqrt{2} = \sqrt{2}$ 확정

> **📝 해설 (Explanation)**
> 사인함수의 최댓값은 1이므로, $f(x)$의 최댓값은 $\sin(x+\alpha)=1$ 일 때 발생합니다.
> $f(x)_{\text{max}} = 2\sqrt{2}(1) - \sqrt{2} = \sqrt{2}$ 입니다.
> 정답은 **①**입니다.
