# Trigger Dictionary (조건-반응 카테고리 사전)

에이전트 단독 참조용 문서입니다. 올바른 Trigger 카테고리를 찾기 위해 검색(grep)하십시오.

## 작명 규칙
- **형식**: `[{수학적 상황 명사구} 조건]` (어미는 `~조건` 통일). 
- 새 카테고리 추가 시, 기존 패턴 대체가 불가능한지 1차 확인.

## 주요 승인 카테고리 목록
- **수열**: `[등차수열 조건]`, `[등차수열의 합 조건]`, `[등비수열 조건]`, `[등비급수 조건]`, `[이차방정식의 해와 조건을 이용한 공비 확정]`, `[시그마 기호식 조건]`, `[수열의 귀납적 정의 조건]`, `[수열 조건]`
- **지수/로그**: `[지수함수 조건]`, `[지수함수 평행이동 조건]`, `[로그의 성질 조건]`, `[두 로그 방정식 제시]`
- **삼각/기하**: `[호도법과 부채꼴 넓이 조건]`, `[삼각함수의 각변환 조건]`, `[삼각함수 부호 조건]`, `[삼각형 넓이 조건]`, `[삼각형 조건]`, `[두 점 사이 거리 조건]`
- **미적분Ⅰ**: `[함수의 극한 조건]`, `[함수의 연속 조건]`, `[미분계수 조건]`, `[미분가능성 조건]`, `[도함수의 식 제시]`, `[다항함수의 극값 조건]`, `[다항함수 조건]`, `[정적분을 포함한 함수 조건]`, `[정적분 조건]`, `[적분과 미분의 관계 조건]`, `[두 곡선 사이의 넓이 조건]`, `[적분과 속도·거리 조건]`
- **확률과 통계**: `[경우의 수 조건]`, `[순열 조건]`, `[중복조합 조건]`, `[조합 조건]`, `[이항정리 전개 조건]`, `[확률 덧셈 조건]`, `[배반사건 조건]`, `[여사건 확률 조건]`, `[조건부확률 조건]`, `[이산확률변수 조건]`, `[기댓값 계산 조건]`, `[이산확률변수 분산 조건]`, `[이항분포 조건]`, `[정규분포 조건]`, `[신뢰구간 조건]`, `[모평균과 표본평균 조건]`
- **공통수학/흐름 제어**: `[이차함수의 최대·최소 조건]`, `[두 연속함수의 합·곱 조건]`, `[추가 관계식 조건]`, `[자연수 조건]`, `[최종 구하는 값]`
