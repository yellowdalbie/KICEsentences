# 기출표현 검색 규칙 안내

**기출표현** 검색은 수능·평가원 수학 문항의 원문에서 원하는 표현, 수식, 조건을 정밀하게 찾아주는 기능입니다.
단어나 수식을 그대로 입력하는 것은 물론, 아래의 검색 규칙을 조합하면 훨씬 정교한 검색이 가능합니다.

---

## 1. 기본 단어 검색

검색창에 단어를 그냥 입력하면 해당 단어가 포함된 문항을 모두 찾아줍니다.

| 입력 예시 | 결과 |
|-----------|------|
| 코사인법칙 | "코사인법칙"이 언급된 문항 |
| 역함수 | "역함수"가 언급된 문항 |
| 최솟값 | "최솟값"이 언급된 문항 |

---

## 2. 동시에 포함 (AND 검색)

**단어 사이에 공백**을 두면 두 단어가 **모두 포함**된 문항만 찾습니다. 단어의 순서나 거리와는 무관합니다.

| 입력 예시 | 결과 |
|-----------|------|
| 극댓값 미분계수 | 극댓값과 미분계수가 함께 등장하는 문항 |
| 이차함수 접선 | 이차함수와 접선이 함께 등장하는 문항 |
| 정적분 넓이 삼각함수 | 세 단어가 모두 등장하는 문항 |

> **활용 팁**: 단어를 많이 입력할수록 검색 범위가 좁아져 더 구체적인 문항을 찾을 수 있습니다.

---

## 3. 둘 중 하나 포함 (OR 검색)

단어 사이에 `|` 기호를 넣으면 **둘 중 하나 이상** 포함된 문항을 찾습니다.

| 입력 예시 | 결과 |
|-----------|------|
| 사인법칙 &#124; 코사인법칙 | 사인법칙 또는 코사인법칙이 등장하는 문항 |
| 최댓값 &#124; 최솟값 | 최댓값이나 최솟값이 등장하는 문항 |
| 등차수열 &#124; 등비수열 | 등차수열이나 등비수열이 등장하는 문항 |

---

## 4. 특정 단어 제외 (NOT 검색)

단어 바로 앞에 `-`를 붙이면 해당 단어가 포함된 문항을 **결과에서 제외**합니다.

| 입력 예시 | 결과 |
|-----------|------|
| 주기 -삼각함수 | "주기"가 나오지만 "삼각함수"는 나오지 않는 문항 |
| 접선 -원 | "접선"이 나오지만 "원"은 언급되지 않는 문항 |
| 확률 -조건부 | "확률"이 나오지만 "조건부"는 나오지 않는 문항 |

---

## 5. 정확한 문구 검색

큰따옴표 `"..."` 안에 입력하면 **해당 문구가 그대로** 등장하는 문항만 찾습니다. 띄어쓰기까지 정확히 일치해야 합니다.

| 입력 예시 | 결과 |
|-----------|------|
| "서로 다른 두 실근" | 이 표현이 정확히 포함된 문항 |
| "역함수가 존재" | 이 문구가 그대로 포함된 문항 |
| "최고차항의 계수가 1" | 이 표현이 그대로 포함된 문항 |

> **참고**: 구문 검색은 단어 순서가 중요합니다. `"두 실근 서로 다른"`처럼 순서가 바뀌면 찾지 못합니다.

---

## 6. 괄호로 묶기 (우선순위 지정)

소괄호 `( )`로 묶으면 검색 조건의 **적용 순서**를 직접 지정할 수 있습니다. 복잡한 조건을 조합할 때 유용합니다.

| 입력 예시 | 결과 |
|-----------|------|
| (사인법칙 &#124; 코사인법칙) -외접원 | 사인법칙 또는 코사인법칙이 나오면서 외접원은 나오지 않는 문항 |
| (미분 &#124; 적분) 삼각함수 | 미분이나 적분 중 하나와 삼각함수가 함께 등장하는 문항 |
| (로그 &#124; 지수) (방정식 &#124; 부등식) | 로그나 지수 중 하나, 그리고 방정식이나 부등식 중 하나가 함께 등장하는 문항 |

---

## 7. 연도·시험 범위 지정

`path:` 뒤에 연도나 시험 이름을 입력하면 **특정 연도·시험**의 문항만 검색합니다.

| 입력 예시 | 결과 |
|-----------|------|
| path:2026 극한 | 2026학년도 문항 중 "극한"이 포함된 문항 |
| path:9모 미분 | 9월 모의평가 문항 중 "미분"이 포함된 문항 |
| path:수능 정적분 | 수능 문항 중 "정적분"이 포함된 문항 |
| path:2025 (로그 &#124; 지수) | 2025학년도 중 로그나 지수 관련 문항 |

> **활용 팁**: `path:` 필터는 다른 조건과 자유롭게 조합할 수 있습니다.

---

## 8. 패턴 검색 (고급)

슬래시 `/패턴/` 안에 검색 패턴을 입력하면 **정규식 기반의 유연한 검색**이 가능합니다. 수식처럼 특수문자가 포함된 표현을 찾을 때 특히 유용합니다.

> 정규식에서 `(`, `)`, `^`, `$` 등 특수문자는 앞에 `\`를 붙여야 문자 그대로 인식됩니다.

| 입력 예시 | 결과 |
|-----------|------|
| /f\(x\).*=.*0/ | f(x) = 0 형태의 수식이 포함된 문항 |
| /\d+개의 실근/ | "○개의 실근" 형태 표현이 포함된 문항 |
| /\ba_[0-9]+\b/ | a_1, a_3 등 수열 항이 등장하는 문항 |
| /[가나다라마]함수/ | 가함수, 나함수, 다함수 등 중 하나가 포함된 문항 |

---

## 9. 수식(LaTeX) 검색

문항 원문에는 수식이 LaTeX 형태로 저장되어 있습니다. LaTeX 문법을 그대로 입력하면 특정 수식 표현이 포함된 문항을 정밀하게 찾을 수 있습니다.

> **주의**: LaTeX에서 백슬래시(`\`)는 명령어 앞에 씁니다. 예: \lim, \int, \frac, \sin, \log

### 자주 쓰이는 LaTeX 기호

| LaTeX 입력 | 수식 의미 |
|------------|-----------|
| \lim | 극한 |
| \lim_{n \rightarrow \infty} | 수열의 극한 |
| \int | 정적분 |
| \sum | 시그마(합) |
| \frac | 분수 |
| \sin, \cos, \tan | 삼각함수 |
| \log | 로그 |
| \sqrt | 제곱근 |
| \infty | 무한대 |

### 수식 검색 활용 예시

| 입력 예시 | 결과 |
|-----------|------|
| \lim_{n \rightarrow \infty} S_{n} | 수열의 극한 lim S_n이 등장하는 문항 |
| \int_{0}^{1} | 0에서 1까지 정적분 조건이 있는 문항 |
| \sin 2x | sin 2x 표현이 포함된 삼각함수 문항 |
| \log _{2} | 밑이 2인 로그 조건 문항 |
| \sum_{n=1}^{15} | n=1부터 15까지 합산 조건 문항 |
| \sqrt[3] | 세제곱근이 포함된 문항 |
| \frac{q}{p} | q/p 값을 구하는 문항 |

> **팁**: 수식 검색은 공백 위치에 따라 결과가 달라질 수 있습니다. 원문 표기를 그대로 따르거나, `/패턴/` 정규식 검색으로 공백을 유연하게 처리할 수 있습니다.

### 수식과 조건어를 결합한 검색

```
\lim_{n \rightarrow \infty} S_{n} path:수능
```
→ 수능 문항 중 수열의 극한 lim S_n이 등장하는 문항

```
\int_{0}^{1} 상수
```
→ 0에서 1까지 정적분 조건이 있으면서 "상수"도 언급된 문항

```
\sin path:2025 최댓값
```
→ 2025학년도 문항 중 sin이 포함되면서 최댓값을 구하는 문항

---

## 종합 활용 예시

**예시 1** — 특정 상황이 담긴 문항 찾기

```
"서로 다른 두 실근" 이차함수 path:2025
```
→ 2025학년도 문항 중 이차함수를 다루면서 "서로 다른 두 실근"이라는 표현이 정확히 포함된 문항

---

**예시 2** — 두 개념 중 하나를 포함하면서 특정 개념은 제외

```
(사인법칙 | 코사인법칙) "외접원의 반지름" -내접
```
→ 사인법칙이나 코사인법칙을 쓰고, "외접원의 반지름" 표현이 등장하지만, 내접은 나오지 않는 문항

---

**예시 3** — 연도 범위를 특정한 복합 검색

```
path:수능 (연속 | 미분가능) -다항함수
```
→ 수능 문항 중 연속이나 미분가능성을 다루되, 다항함수 관련 문항은 제외

---

**예시 4** — 수식 패턴 검색

```
/f\([0-9]+\)/ 최솟값
```
→ f(1), f(2) 등 특정 값에서의 함숫값 계산이 포함된 문항 중 최솟값도 함께 등장하는 문항
