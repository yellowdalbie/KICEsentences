# KICE 수학 검색 수식 규칙 가이드 (Expression Search Rules)

본 검색 기능은 평가원(KICE) 수능/평가원 수학 해설 데이터베이스(`MD_Ref`) 내부의 마크다운(.md) 원문 텍스트를 논리 연산자와 정규표현식을 결합하여 강력하게 검색할 수 있도록 특수 규칙들을 제공합니다. 아래의 규칙들을 조합하여 원하는 문항을 세밀하게 필터링해보세요!

---

## 🔍 기본 연산자 (Basic Operators)

### 1. AND 검색 (동시 포함)

* **표기법**: `단어1 단어2` (단어 사이에 공백 하나 이상)
* **설명**: 입력된 **두 단어가 모두** 문서(문항 해설) 내에 존재하는 경우만 찾습니다. 어순이나 거리는 상관없습니다.
* **사용 예시**: `극댓값 미분계수`
  * 해석: "극댓값"과 "미분계수"라는 단어가 둘 다 들어간 문항 검색.

### 2. OR 검색 (택일 포함)

* **표기법**: `단어1 OR 단어2` (대문자 `OR` 필수) 또는 `단어1 | 단어2` (파이프 기호)
* **설명**: 입력된 단어 중 **최소 하나 이상** 포함된 문서를 모두 검색합니다.
* **사용 예시**: `등차수열 OR 등비수열`
  * 해석: 등차수열 문제이거나 등비수열 문제이거나 둘 다 쓰인 문제 검색.

### 3. NOT 검색 (특정 단어 제외)

* **표기법**: `-단어` (단어 바로 앞에 마이너스 하이픈 기호) 또는 `NOT 단어`
* **설명**: 해당 단어가 포함된 문서는 검색 결과에서 **반드시 필터링하여 제외**합니다.
* **사용 예시**: `주기 -삼각함수`
  * 해석: "주기"라는 말이 들어간 문제 중에 "삼각함수"라는 말이 들어간 문항은 모조리 제외하고 검색.

## 🎯 고급 연산자 (Advanced Operators)

### 4. 정확한 구문 검색 (Exact Match)

* **표기법**: `"정확히 일치해야 하는 텍스트"` (큰따옴표로 감쌈)
* **설명**: 공백, 조사, 기호까지 포함하여 **100% 동일하게 연속해서** 등장하는 텍스트 블록만 찾습니다.
* **사용 예시**: `"서로 다른 두 실근"`
  * 해석: 저 문장이 토씨 하나 안 틀리고 그대로 들어간 경우만 검색. ('서로 다른 실근을 두 개 가진다' 같은 문장은 제외됨)

### 5. 그룹화 및 우선순위 (Grouping)

* **표기법**: `( )` (소괄호로 감쌈)
* **설명**: 수학의 사칙연산 괄호처럼 검색 연산자의 **적용 우선순위**를 지정합니다. 복잡한 쿼리를 작성할 때 필수입니다.
* **사용 예시**: `(사인법칙 | 코사인법칙) -외접원`
  * 해석: (사인법칙 혹은 코사인법칙)이 나오는 문서 중에서 "외접원"이란 단어가 나오는 문서는 제거함.

### 6. 정규 표현식 (Regular Expression)

* **표기법**: `/정규식패턴/` (양끝을 슬래시 기호로 감쌈)
* **설명**: 일반 텍스트가 아닌 Python의 정규표현식(`re`) 패턴 매칭 시스템을 사용하여 패턴 기반 문자를 찾습니다. 수식을 잡을 때 특히 유용합니다.
* **사용 예시**: `/f\(x\).*[0-9]+/`
  * 해석: 텍스트 안에 $f(x)$가 등장하고, 그 뒤 어딘가에 숫자가 최소 하나 이상 나오는 패턴의 문장을 검색함.

## 📁 특수 필터 (Special Filters)

### 7. 경로(연도/시험) 필터링 (Path Filter)

* **표기법**: `path:검색경로키워드`
* **설명**: 검색할 파일의 **경로나 제목 이름 자체를 제한**합니다. 특정 연도나 모의고사만 타겟팅할 때 강력합니다.
* **사용 예시 1**: `path:2025 극한`
  * 해석: 2025학년도 하위 폴더/파일 중에서 "극한"이라는 단어가 들어간 텍스트 검색.
* **사용 예시 2**: `path:9모 정적분`
  * 해석: 9월 모의평가 해설 중에서 "정적분" 단어가 들어간 텍스트 검색.

---

## 종합 활용 예시

> `(사인법칙 OR 코사인법칙) "외접원의 반지름" -삼각형 path:2025`

**해석:** 2025학년도 문항(`path:2025`) 중에서, 사인법칙이나 코사인법칙 둘 중 하나 이상 쓰였으면서(`(사인법칙 OR 코사인법칙)`), "외접원의 반지름" 구문이 쓰이되, "삼각형"이라는 단어는 나오지 않는(`-삼각형`) 문항을 찾아라.
