# Phase A 표준 용어 사전 (Vocabulary Memo)
# 최초 작성: 2026-04-10 | 버전: v0.1 (초안 - 점진적 개선 예정)

## 목적
LLM이 `canonical_name`을 작성할 때 반드시 이 사전의 용어를 우선 사용하도록 한다.
`concepts.json`의 `keywords`와 `standard_name`에서 추출한 **교육과정 공식 용어**가 기준이다.

## ⚠️ 주의사항 (LLM 환각 방지 규칙)
1. 아래 목록에 없는 용어는 **새 용어로 간주**하고 별도로 `vocab_candidates.json`에 기록한다.
2. **절대 금지 표현**: "단순화", "도출하기", "계산하여", "정확히", "파악", "확인" → 교육과정 용어가 아닌 서술어 금지.
3. **변수명, 수식, 숫자** 포함 금지 (예: "f(x)", "x=3", "k" 등).
4. **보기/선지 번호** 포함 금지 (예: "보기 ㄱ", "선지 ①" 등).

## ❓ 교육과정에 없으나 해설에 자주 등장하는 용어 후보 (vocab_candidates 예비목록)

*아래는 raw_chunks 분석에서 빈번하게 등장했으나 교육과정 공식 용어에 포함되지 않는 것들.
점진적 검토 후 정식 사전에 편입하거나, 상위 용어로 대체할 예정.*

| 등장 용어 | 처리 방안 제안 |
| :--- | :--- |
| "미정계수" | → 항목 미존재. `항등식의 미정계수 결정법`의 하위 기법. 추가 검토 필요 |
| "부정형(0/0꼴)" | → `극한의 성질` (12미적Ⅰ-01-02) 하위로 편입 |
| "치환" | → 독립 개념이 아닌 '풀이 기법'. 어느 개념의 활용으로 서술할지 검토 필요 |
| "귀류법" | → `수학적 귀납법` (12대수03-06)의 변형. 동일 ID로 처리 가능 |
| "합차공식" | → `인수분해` (10공수1-01-03) 또는 `삼각함수` 합차공식(12대수02-02)으로 분기 |
| "유리화" | → 극한 계산 기법. 독립 항목 불필요, `0/0꼴 극한 계산` 기법으로 통합 |
| "수열의 주기성" | → 표준 용어 없음. `귀납적 정의 + 등비수열` 등의 조합으로 처리 제안 |
