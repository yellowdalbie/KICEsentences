# KICE 수학 문항 해설 작성 지시서 (AI 실행 명령서)

이 문서는 AI가 새 세션에서 해설 작성 작업을 시작할 때 반드시 먼저 읽어야 하는 실행 지시서입니다.
`AI_Instruction_KICE_System.md`(프로젝트 목적/아키텍처)와 `AI_Template_Commentary.md`(기본 템플릿)를 **상위 문서**로 두고, 이 문서는 실제 작업 과정에서 지켜야 할 세부 규칙을 보완합니다.

---

## 0. 작업 시작 전 준비

1. `MD_Ref/{년도}/{파일명}.md`를 반드시 먼저 읽는다.
2. 문항 전체를 머릿속에서 풀고, 논리적 도약이 발생하는 지점마다 Step을 설정한다.
3. 저장 위치: `Sol/{년도}/{파일명}.md`

---

## 1. 파일 헤더 형식

```markdown
# {파일명} 해설
>
> 원본 문항 링크: [{파일명}](../../MD_Ref/{년도}/{파일명}.md)
```

예시:

```markdown
# 2026.9모_12.md 해설
>
> 원본 문항 링크: [2026.9모_12.md](../../MD_Ref/2026/2026.9모_12.md)
```

---

## 2. Step 블록 형식

각 Step은 아래 4개 블록을 순서대로 작성한다. 블록 사이에 빈 줄을 둔다.

```markdown
## [Step N] {타이틀}

- **Trigger**: [{카테고리}] {조건 수식 또는 서술} ({괄호 부가설명})
- **Action**: [{성취기준 코드}] {구체적 수학적 조작}
- **Result**: {결과 수식 또는 서술}

> **📝 해설 (Explanation)**
> {친절한 해설 줄글}
```

---

## 3. Step 타이틀 작성 규칙

**기본 형식**: `[개념/방법]을 이용하여 [대상]을 [결과]하기`

### 4가지 필수 원칙

1. **LaTeX 수식 금지**: 타이틀에 `$...$` 기호 일절 사용 금지. 수학적 대상을 자연어로 표현한다.
   - ❌ `$f(1)$ 계산`, `$a_2$ 산출`, `공차 적용으로 $a_n$ 확정`
   - ✅ `초기 조건 대입으로 특정 함숫값 확정하기`, `공차 적용으로 수열의 특정 항 산출하기`

2. **방법 + 대상 + 결과 3요소 필수**: 방법/개념어 없이 행위만 있는 타이틀 금지.
   - ❌ `합산`, `확률 계산`, `조건 분석`, `최종 연산`
   - ✅ `등비수열 합 공식으로 부분합 산출하기`, `경우의 수 비율로 목표 확률 산출하기`

3. **서사 연결성**: 해당 파일의 Step 타이틀들을 세로로 연결하여 읽으면, Explanation 없이도 풀이의 수학적 논리 흐름이 보여야 한다.

4. **대상어에 수학적 역할 명시**: 단순 "경우의 수"가 아니라, 그 값이 이 단계에서 갖는 역할을 드러낸다.
   - ❌ `경우의 수 산출하기`
   - ✅ `전사건의 경우의 수 산출하기`, `여사건을 이용하여 조건에 위배되는 경우의 수 파악하기`

### 금지 / 허용 대조표

| 금지 (❌) | 허용 (✅) |
|-----------|-----------|
| `$f(1)$ 계산` | `초기 조건 대입으로 특정 함숫값 확정하기` |
| `$a_2$ 계산` | `공차 적용으로 수열의 특정 항 산출하기` |
| `합산` | `등비수열 합 공식으로 부분합 산출하기` |
| `확률 계산` | `경우의 수 비율로 목표 확률 산출하기` |
| `최종 연산` | `확정된 미지수들을 대입하여 최종값 연산하기` |
| `조건 분석` | `약수와 배수 조건을 분석하여 함수의 필수 제한 파악하기` |
| `경우의 수 계산` | `중복조합을 이용하여 조건을 만족시키는 함수의 개수 산출하기` |

> **참조 모범 사례**: `Sol/2025/2025.6모_확28~30.md`, `Sol/2025/2025.9모_확28~30.md`, `Sol/2025/2025수능_확28~30.md`

- 벡터 유사도 기반 시스템이므로 표현의 완전한 표준화는 불필요. 단, 위 4원칙을 위반한 타이틀은 반드시 수정한다.

## 4. Trigger 작성 규칙

**혼합형 필수**: `[카테고리명]` + 구체적 수식/서술 + `(괄호 부가설명)`

```
[삼각함수의 각변환 조건] cos(π/2+θ) = √5/5, tanθ < 0 (각변환 후 사분면 결정)
```

- 카테고리만 있으면 안 됨
- 수식만 있어도 안 됨
- 둘 다 있어야 함

### 자주 쓰이는 Trigger 카테고리

| 카테고리명 | 사용 맥락 |
|-----------|----------|
| `[도함수의 식 제시]` | $f'(x)=\cdots$과 초기값이 주어진 경우 |
| `[등차수열의 조건식 제시]` | 등차수열 조건 2개 이상 |
| `[등비수열의 조건식 제시]` | 등비수열 조건 2개 이상 |
| `[삼각함수의 각변환 조건]` | $\cos(\pi/2+\theta)$ 등 각변환 주어짐 |
| `[삼각함수 부호 조건]` | 사분면 정보로 삼각함수 부호 결정 |
| `[다항함수의 극값 조건]` | 극대/극소 $x$ 좌표 또는 함숫값 조건 |
| `[이차방정식의 해와 조건을 이용한 공비 확정]` | 방정식의 해 중 조건에 맞는 것 선택 |
| `[배반사건 조건]` | $A \cap B = \emptyset$ 또는 이와 동치인 조건 |
| `[여사건 확률 조건]` | $P(A^C) = 1 - P(A)$ 적용 |
| `[확률 덧셈 조건]` | 덧셈정리 또는 분해 공식 적용 |
| `[확률 대수 조건]` | $P(A \cup B)$, $P(A \cap B)$ 등 여러 확률값 제시 |
| `[이산확률변수 조건]` | 확률분포표 또는 확률변수 정의 |
| `[이산확률변수 분산 조건]` | $V(X) = E(X^2) - [E(X)]^2$ 적용 |
| `[시그마 기호식 조건]` | $\sum$ 형태의 조건식이 제시됨 |
| `[두 점 사이 거리 조건]` | $\overline{AB} = \overline{AC}$ 등 거리 등치 조건 |
| `[삼각형 넓이 조건]` | 넓이가 특정 값으로 주어짐 |
| `[두 로그 방정식 제시]` | 로그 방정식 2개 이상이 주어짐 |
| `[지수함수 조건]` | $y = a^x$ 또는 $y = a^x + k$ 등 지수함수 그래프 조건 |
| `[지수함수 평행이동 조건]` | $y = a^x + k$ 꼴의 평행이동된 지수함수 |
| `[기댓값 계산 조건]` | 결정된 파라미터로 $E(X)$, $E(X^2)$ 계산 |
| `[최종 구하는 값]` | 마지막 Step에서 문제가 구하라 한 값 |
| `[적분과 속도·거리 조건]` | 속도를 적분하여 위치 변화량 또는 이동 거리 산출 (`12미적Ⅰ-03-06`) |
| `[정적분을 포함한 함수 조건]` | 함수식 안에 $\int f(t)\,dt$ 형태가 포함되어 적분값을 상수로 치환하는 패턴 (`12미적Ⅰ-03-02`) |
| `[함수의 개수 조건]` | $A \to A$ 등 조건을 만족하는 함수의 개수를 케이스별로 세는 경우 (`10공수1-03-01`) |
| `[등차수열의 합 조건]` | $S_n$ 또는 부분합이 등차수열 조건으로 주어지거나 최대·최소를 구하는 경우 (`12대수03-02`) |
| `[모평균과 표본평균 조건]` | 모분산·표본평균 분산·표본비율 등 표본조사 관련 계산 (`12확통03-06`) |
| `[로그의 성질 조건]` | 로그의 합·차·지수 변환 등 성질을 활용하여 값 산출하는 경우 (`12대수01-02`) |
| `[등비급수 조건]` | 무한등비급수 $\lim S_n$ 또는 $\dfrac{a}{1-r}$ 공식 적용 (`12대수03-03`) |
| `[호도법과 부채꼴 넓이 조건]` | 호도법 반지름·중심각으로 부채꼴 넓이 또는 호의 길이 산출 (`12대수02-01`) |
| `[조합 조건]` | $_nC_r$ 조합으로 선택 경우의 수를 계산하는 경우 (`12확통01-02`) |
| `[순열 조건]` | 같은 것이 있는 순열·다항계수 등 배열 경우의 수 계산 (`12확통01-01`) |
| `[미분가능성 조건]` | 좌·우 미분계수 비교로 미분가능 여부 또는 그 조건 산출 (`12미적Ⅰ-02-02`) |
| `[적분과 미분의 관계 조건]` | $\frac{d}{dx}\int_a^x f(t)\,dt = f(x)$ 관계로 피적분함수 또는 원시함수 도출 (`12미적Ⅰ-03-04`) |
| `[두 곡선 사이의 넓이 조건]` | 두 곡선의 교점을 구간 경계로 하여 정적분으로 넓이를 산출하는 경우 (`12미적Ⅰ-03-05`) |
| `[함수의 극한 조건]` | $\lim_{x\to a}$ 형태의 극한값이 조건으로 주어져 함수 구조(영점·차수)를 결정하는 경우 (`12미적Ⅰ-01-04`) |

> ⚠️ **실수 주의**: 위 목록의 하단 14개 항목은 과거 독단 작성 후 소급 등록된 항목입니다. 유사 상황에서 반드시 참조하세요.

## 6. Result 작성 규칙

- 수식 결과 + 수학적 의미 또는 다음 단계를 위한 해석 병기
- 예: `$r=2$ (모든 항 양수이므로 양의 공비 확정)`
- 예: `$\sin\theta = -\dfrac{\sqrt{3}}{3}$ (4사분면에서 음수임을 확인)`
- 마지막 Step의 Result: 정답값만 간결하게 표기

## 7. 해설(Explanation) 작성 규칙

- 독립적으로 학생에게 보여진다는 것을 가정하고 작성
- 연산 과정 생략 없이 전부 표시
- 해당 Action을 선택한 필연성(왜 이 방법인지) 설명 권장
- 종결어미: **`~입니다` 체** 일관 유지 (`~이다` 금지)
- **문체 (Tone & Manner)**: 철저히 수학적이고 건조한 학술적 문어체를 사용. 비유적 표현이나 구어체("땀 흘려", "친절하게", "관문", "덩어리", "깎아나가며", "외톨이", "진짜로" 등) 절대 사용 금지.
- **교육과정 준수**: 2022 개정 교육과정 기준에 부합하는 수학 용어 사용 (예: 확률과 통계에서 교육과정 외인 '자연수의 분할' 용어 및 개념 접근 지양 $\rightarrow$ '제약 조건에 따른 체계적인 분류' 및 '합의 법칙' 또는 '곱의 법칙'으로 논리 전개).
- **마지막 Step의 마지막 줄**: 정답 명시
  - 선택지형: `정답은 **③**입니다.` 또는 `따라서 정답은 **③**입니다.`
  - 단답형: `정답은 **15**입니다.`

## 8. Step 분리 기준

### 분리 필수

- 서로 다른 수학적 판단이 2개 이상 들어갈 때
  - 조건으로 상수/공비 결정 (Step N) / 결정된 값으로 목표값 계산 (Step N+1)
  - 사분면으로 부호 판단 (Step N) / 항등식으로 크기 계산 (Step N+1)
  - 방정식 수립 (Step N) / 방정식 풀기 (Step N+1) — 단, 단순한 경우 통합 허용

### 통합 허용

- 연산의 흐름상 논리적 단절이 굳이 없는 경우
  - 이차방정식 인수분해 후 두 근 동시 도출
  - 좌극한·우극한 계산 후 연속 조건 수립

### 단순 문항

- 1~2 Step으로 끝나도 반드시 `[Step 1]` 형식 유지

## 10. 교육과정 경계 판정 사례집 (검증 기준)

작성 또는 검증 시 아래 사례를 반드시 참조한다. 의심스러운 표현이 나오면 이 목록을 먼저 확인하고 수정한다.

---

### 10-1. 판별식 관련

**원칙**: 판별식($D$)은 계산 수단이지 상황이 아니다. Trigger 카테고리는 판별식을 사용하게 되는 **상황**에 초점을 맞춰야 한다.

| 상황 | 올바른 카테고리 | 올바른 성취기준 코드 | 잘못된 표현 |
|------|-----------------|---------------------|-------------|
| 이차식이 모든 실수에서 부등식 성립 | `[방정식과 부등식에의 활용 조건]` | `12미적Ⅰ-02-09` | ❌ `[이차함수 부등식 조건]`, `[판별식 조건]` |
| 두 함수 $f$, $g$의 합과 곱이 주어진 경우 | `[두 연속함수의 합·곱 조건]` | `10공수1-02-03` | ❌ `[이차방정식 판별식 조건]` |

> **배경**: 판별식을 활용하는 것은 요령에 가까운 계산 방법이다. 판별식을 사용하게 되는 수학적 상황(이차부등식이 항등적으로 성립, 두 함수의 합·곱 관계 등)이 Trigger의 핵심이다.

---

### 10-2. 최대·최소 관련 — 교육과정 범위 구분 필수

**원칙**: "최대·최소"가 어느 교육과정 단원에서 다뤄지는지에 따라 카테고리와 성취기준 코드가 달라진다.

| 상황 | 올바른 카테고리 | 올바른 성취기준 코드 |
|------|-----------------|---------------------|
| 이차함수의 꼭짓점 또는 치환 후 이차식 최대·최소 (공통수학 범위) | `[이차함수의 최대·최소 조건]` | `10공수1-02-06` |
| 닫힌 구간에서 도함수 부호 분석으로 최대·최소 (미적분Ⅰ 범위) | `[함수의 증가·감소 조건]` | `12미적Ⅰ-02-07` |
| 극값을 이용한 최대·최소 (미적분Ⅰ 범위) | `[다항함수의 극값 조건]` | `12미적Ⅰ-02-07` |

> ❌ `[함수 최댓값 조건]`, `[함수 최솟값 조건]`처럼 범위 구분 없이 일반화된 카테고리명은 사용 금지.

---

### 10-3. 수열의 귀납적 정의 관련

**원칙**: "점화식"은 2022 개정 교육과정에서 퇴출된 용어이다. 일반항 도출 방법(특성방정식 등)도 교육과정 외이다.

- ❌ `[수열 점화식 조건]`, "점화식으로 일반항 도출"
- ✅ `[수열의 귀납적 정의 조건]`, "귀납적 정의로 인접 항의 관계 파악", "연쇄 대입으로 특정 항 결정"
- 성취기준 코드: `12대수03-06` (수열의 귀납적 정의)
- **주의**: 등차·등비수열로 귀결되는 경우 귀납적 정의 단계(도출)와 일반항 계산 단계(적용)를 Step으로 분리하고, 일반항 계산에는 `12대수03-02`(등차) 또는 `12대수03-03`(등비) 사용.

---

### 10-4. 포함-배제 원리 관련

**원칙**: "포함-배제 원리"는 교육과정에 명시되지 않은 개념이다. 카테고리명이나 해설 본문에 직접 사용 금지.

- ❌ `[포함-배제 원리 조건]`, 해설에서 "포함-배제 원리로 …"
- ✅ 교육과정 내 용어로 대체:
  - 카테고리: `[여사건 경우의 수 조건]`
  - 해설 서술: "여사건의 합사건 크기를 합의 법칙으로 산출", "사건 A, B, C의 합사건에서 교사건을 제거"
- **주의**: 여사건이 복잡해져 합의 법칙 반복 적용이 필요한 경우도 교육과정 내 용어(합사건, 교사건, 여사건, 합의 법칙)만 사용.

- 확률과 통계에서 자주 등장하는 표준 개념이므로 독립적 카테고리로 허용.

- 미적분Ⅰ 교육과정에 명시된 개념이므로 허용.

---

### 10-7. 급수와 정적분의 관계 관련 (리만합 용어 금지)

**원칙**: "리만합(Riemann sum)"은 교육과정 용어가 아니다. 카테고리명·해설 본문 어디에도 사용 금지.

- ❌ "리만합으로 변환", "리만합 형태", "리만합 정의에 의해"
- ✅ **교육과정 용어**: "급수와 정적분의 관계"
  - 카테고리: `[시그마 기호식 조건]`, 성취기준 코드 `12미적Ⅰ-03-03`
  - 해설 서술: "$\dfrac{1}{n}\displaystyle\sum_{k=1}^n f\!\left(\dfrac{k}{n}\right)$ 형태를 급수와 정적분의 관계로 $\displaystyle\int_0^1 f(x)\,dx$로 변환"

> **배경**: 리만합은 대학 수준 해석학 용어이며 고등학교 교육과정에 명시되지 않는다. 교육과정은 이 변환을 "급수와 정적분의 관계"로 표현한다.

## [Step 1] 등차수열 조건으로 연립방정식 수립

- **Trigger**: [등차수열의 조건식 제시] $a_1=2a_5$, $a_8+a_{12}=-6$ (두 등차 조건)

- **Result**: $-a_1=8d$에서 $a_1=-8d$. 대입: $-16d+18d=2d=-6 \Rightarrow d=-3$, $a_1=24$

> $a_1=-8d$ 대입: $2d=-6 \Rightarrow d=-3$, $a_1=24$

- **Trigger**: [최종 구하는 값] $a_2$
- **Action**: [12대수03-02] $a_2 = a_1 + d = 24 + (-3) = 21$
- **Result**: $a_2 = 21$

> **📝 해설 (Explanation)**
> $a_2 = 24 + (-3) = 21$이며, 정답은 **③**입니다.
