[ THINK LYNX 윈도우 실행 방법 안내 ]

동봉된 'THINK_LYNX_실행가이드.png' 이미지를 함께 참고하세요.

1단계: 폴더 열기
- ZIP 파일의 압축을 푼 폴더로 들어갑니다. (app 폴더가 보이는 곳)

2단계: 터미널 열기
- 폴더 빈 공간에서 [마우스 우클릭]을 합니다.
- '터미널에서 열기' 또는 'PowerShell 창 열기'를 클릭합니다.

3단계: 명령어 입력 및 실행
- 나타난 터미널 창에 아래 명령어를 직접 입력하고 [엔터]를 칩니다.
  (복사 붙여넣기 시 오류가 날 수 있으니 직접 타이핑을 권장합니다.)

.\app\python\python.exe .\app\launcher.py

[ 실행이 안 될 경우 ]
- 실행 중에는 터미널 창을 닫지 말아주세요. (앱이 함께 종료됩니다.)
- 오류가 발생하면 폴더 이름이나 경로에 한글이나 공백이 있는지 확인하고,
  C:\lynx 와 같이 단순한 경로로 폴더를 옮겨서 실행해 보세요.
