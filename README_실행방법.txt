[ KICE Lynx 윈도우 실행 방법 안내 ]

이 프로그램은 설치 과정 없이 터미널을 통해 직접 실행하는 방식이 가장 안전합니다.
아래의 3단계를 따라해 주세요. (동봉된 'KICE_Lynx_실행가이드.png'를 참고하면 더 쉽습니다.)

1단계: 폴더 열기
- 다운로드한 ZIP 파일의 압축을 푼 폴더로 들어갑니다. (app 폴더가 보이는 위치)

2단계: 터미널 열기
- 폴더 빈 공간에서 [Shift 키]를 누른 상태로 [마우스 우클릭]을 합니다.
- 메뉴에서 '여기서 PowerShell 창 열기'를 클릭합니다.

3단계: 명령어 입력 및 실행
- 나타난 파란색 터미널 창에 아래 명령어를 그대로 복사해서 붙여넣고 [엔터]를 칩니다.

.\app\python\python.exe launcher.py

[ 참고 사항 ]
- 실행 중에는 터미널 창을 닫지 말아주세요. (앱이 함께 종료됩니다.)
