21. 수열 $\left\{a_{n}\right\}$ 이 모든 자연수 $n$ 에 대하여 다음 조건을 만족시킨다.
\[
\begin{array}{l}
\text { (가) } a_{2 n}=a_{n}-1 \\
\text { (나) } a_{2 n+1}=2 a_{n}+1
\end{array}
\]
$a_{20}=1$ 일 때, $\sum_{n=1}^{63} a_{n}$ 의 값은? [4점]
(1) 704
(2) 712
(3) 720
(4) 728
(5) 736
