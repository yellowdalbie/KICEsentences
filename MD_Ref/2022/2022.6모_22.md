22. 삼차함수 $f(x)$ 가 다음 조건을 만족시킨다.
(가) 방정식 $f(x)=0$ 의 서로 다른 실근의 개수는 2 이다. (나) 방정식 $f(x-f(x))=0$ 의 서로 다른 실근의 개수는 3 이다.
$$f(1)=4, f^{\prime}(1)=1, f^{\prime}(0)>1 \text { 일 때, } f(0)=\frac{q}{p} \text { 이다. } p+q \text { 의 }$$

값을 구하시오. (단, $p$ 와 $q$ 는 서로소인 자연수이다.) [4점]
* 확인 사항
- 답안지의 해당란에 필요한 내용을 정확히 기입(표기)했는지 확인 하시오.
- 이어서, 「선택과목(확률과 통계)」 문제가 제시되오니, 자신이 선택한 과목인지 확인하시오.
