30. 좌표평면 위의 네 점 $\mathrm{A}(2,0), \mathrm{B}(0,2), \mathrm{C}(-2,0), \mathrm{D}(0,-2)$ 를 꼭짓점으로 하는 정사각형 ABCD 의 네 변 위의 두 점 $\mathrm{P}, \mathrm{Q}$ 가 다음 조건을 만족시킨다.
\[
\begin{array}{l}
\text { (가) }(\overrightarrow{\mathrm{PQ}} \cdot \overrightarrow{\mathrm{AB}})(\overrightarrow{\mathrm{PQ}} \cdot \overrightarrow{\mathrm{AD}})=0 \\
\text { (나) } \overrightarrow{\mathrm{OA}} \cdot \overrightarrow{\mathrm{OP}} \geq-2 \text { 이고 } \overrightarrow{\mathrm{OB}} \cdot \overrightarrow{\mathrm{OP}} \geq 0 \text { 이다. } \\
\text { (다) } \overrightarrow{\mathrm{OA}} \cdot \overrightarrow{\mathrm{OQ}} \geq-2 \text { 이고 } \overrightarrow{\mathrm{OB}} \cdot \overrightarrow{\mathrm{OQ}} \leq 0 \text { 이다. }
\end{array}
\]

점 $\mathrm{R}(4,4)$ 에 대하여 $\overrightarrow{\mathrm{RP}} \cdot \overrightarrow{\mathrm{RQ}}$ 의 최댓값을 $M$, 최솟값을 $m$ 이라 할 때, $M+m$ 의 값을 구하시오. (단, O 는 원점이다.) [4점]

- 답안지의 해당란에 필요한 내용을 정확히 기입(표기)했는지 확인 하시오.
