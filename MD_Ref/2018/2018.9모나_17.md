17. 실수 전체의 집합에서 정의된 두 함수 $f(x)$ 와 $g(x)$ 에 대하여
\[
\begin{array}{l}
x<0 \text { 일 때, } f(x)+g(x)=x^{2}+4 \\
x>0 \text { 일 때, } f(x)-g(x)=x^{2}+2 x+8
\end{array}
\]

이다. 함수 $f(x)$ 가 $x=0$ 에서 연속이고 $\lim _{x \rightarrow 0-} g(x)-\lim _{x \rightarrow 0+} g(x)=6$ 일 때, $f(0)$ 의 값은? [4점]
(1) -3
(2) -1
(3) 0
(4) 1
(5) 3
