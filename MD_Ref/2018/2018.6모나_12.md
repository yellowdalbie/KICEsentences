12. 실수 $a$ 에 대하여 명제
$$\text { ' } a \geq \sqrt{3} \text { 이면 } a^{2} \geq 3 \text { 이다.' }$$

의 대우는? [3점]
(1) $a^{2}<3$ 이면 $a>\sqrt{3}$ 이다.
(2) $a^{2}<3$ 이면 $a<\sqrt{3}$ 이다.
(3) $a^{2} \leq 3$ 이면 $a \leq \sqrt{3}$ 이다.
(4) $a>\sqrt{3}$ 이면 $a^{2} \leq 3$ 이다.
(5) $a \geq \sqrt{3}$ 이면 $a^{2}<3$ 이다.
