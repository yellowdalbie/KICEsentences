26. 확률변수 $X$ 가 평균이 $m$, 표준편차가 $\sigma$ 인 정규분포를 따르고
$$\mathrm{P}(X \leq 3)=\mathrm{P}(3 \leq X \leq 80)=0.3$$

일 때, $m+\sigma$ 의 값을 구하시오.
(단, $Z$ 가 표준정규분포를 따르는 확률변수일 때,
$\mathrm{P}(0 \leq Z \leq 0.25)=0.1, \mathrm{P}(0 \leq Z \leq 0.52)=0.2$ 로 계산한다. $)$
