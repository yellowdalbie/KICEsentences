20. 첫째항이 $a$ 인 수열 $\left\{a_{n}\right\}$ 은 모든 자연수 $n$ 에 대하여
\[
a_{n+1}=\left\{\begin{array}{ll}
a_{n}+(-1)^{n} \times 2 & (n \text { 이 } 3 \text { 의 배수가 아닌 경우 }) \\
a_{n}+1 & (n \text { 이 } 3 \text { 의 배수인 경우 })
\end{array}\right.
\]

를 만족시킨다. $a_{15}=43$ 일 때, $a$ 의 값은? [4점]
(1) 35
(2) 36
(3) 37
(4) 38
(5) 39
