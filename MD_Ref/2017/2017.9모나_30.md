30. 좌표평면에서 자연수 $n$ 에 대하여 영역
$$\left\{(x, y) \mid 0 \leq x \leq n, 0 \leq y \leq \frac{\sqrt{x+3}}{2}\right\}$$

에 포함되는 정사각형 중에서 다음 조건을 만족시키는 모든 정사각형의 개수를 $f(n)$ 이라 하자.
(가) 각 꼭짓점의 $x$ 좌표, $y$ 좌표가 모두 정수이다.
(나) 한 변의 길이가 $\sqrt{5}$ 이하이다.

예를 들어 $f(14)=15$ 이다. $f(n) \leq 400$ 을 만족시키는 자연수 $n$ 의 최댓값을 구하시오. [4점]
* 확인 사항
- 답안지의 해당란에 필요한 내용을 정확히 기입(표기)했는지 확인 하시오.
