20. 최고차항의 계수가 양수인 삼차함수 $f(x)$ 가 다음 조건을 만족시킨다.
(가) 함수 $f(x)$ 는 $x=0$ 에서 극댓값, $x=k$ 에서 극솟값을 가진다. (단, $k$ 는 상수이다.)
(나) 1 보다 큰 모든 실수 $t$ 에 대하여
\[
\begin{array}{l}
\int_{0}^{t}\left|f^{\prime}(x)\right| d x=f(t)+f(0) \\
\text { 이다. }
\end{array}
\]
<보기>에서 옳은 것만을 있는 대로 고른 것은? [4점]
<보 기>
ㄱ. $\int_{0}^{k} f^{\prime}(x) d x<0$
ㄴ. $0<k \leq 1$
ㄷ. 함수 $f(x)$ 의 극솟값은 0 이다.
