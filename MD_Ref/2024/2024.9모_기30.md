30. 좌표평면에서 $\overline{\mathrm{AB}}=\overline{\mathrm{AC}}$ 이고 $\angle \mathrm{BAC}=\frac{\pi}{2}$ 인 직각삼각형 ABC 에 대하여 두 점 $\mathrm{P}, \mathrm{Q}$ 가 다음 조건을 만족시킨다.
\[
\begin{array}{l}
\text { (가) 삼각형 } \mathrm{APQ} \text { 는 정삼각형이고, } \\
9|\overrightarrow{\mathrm{PQ}}| \overrightarrow{\mathrm{PQ}}=4|\overrightarrow{\mathrm{AB}}| \overrightarrow{\mathrm{AB}} \text { 이다. } \\
\text { (나) } \overrightarrow{\mathrm{AC}} \cdot \overrightarrow{\mathrm{AQ}}<0 \\
\text { (다) } \overrightarrow{\mathrm{PQ}} \cdot \overrightarrow{\mathrm{CB}}=24
\end{array}
\]

선분 AQ 위의 점 X 에 대하여 $|\overrightarrow{\mathrm{XA}}+\overrightarrow{\mathrm{XB}}|$ 의 최솟값을 $m$ 이라 할 때, $m^{2}$ 의 값을 구하시오. [4점]
