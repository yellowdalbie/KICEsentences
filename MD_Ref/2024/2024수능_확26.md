26. 4 개의 동전을 동시에 던져서 앞면이 나오는 동전의 개수를 확률변수 $X$ 라 하고, 이산확률변수 $Y$ 를
\[
Y=\left\{\begin{array}{ll}
X & (X \text { 가 } 0 \text { 또는 } 1 \text { 의 값을 가지는 경우 }) \\
2 & (X \text { 가 } 2 \text { 이상의 값을 가지는 경우 })
\end{array}\right.
\]

라 하자. $\mathrm{E}(Y)$ 의 값은? [3점]
(1) $\frac{25}{16}$
(2) $\frac{13}{8}$
(3) $\frac{27}{16}$
(4) $\frac{7}{4}$
(5) $\frac{29}{16}$
