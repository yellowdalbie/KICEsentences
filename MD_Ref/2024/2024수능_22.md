22. 최고차항의 계수가 1 인 삼차함수 $f(x)$ 가 다음 조건을 만족시킨다.

함수 $f(x)$ 에 대하여
$$f(k-1) f(k+1)<0$$

을 만족시키는 정수 $k$ 는 존재하지 않는다.
$$f^{\prime}\left(-\frac{1}{4}\right)=-\frac{1}{4}, f^{\prime}\left(\frac{1}{4}\right)<0 \text { 일 때, } f(8) \text { 의 값을 구하시오. [4점] }$$
* 확인 사항
- 답안지의 해당란에 필요한 내용을 정확히 기입(표기)했는지 확인 하시오.
- 이어서, 「선택과목(확률과 통계)」 문제가 제시되오니, 자신이 선택한 과목인지 확인하시오.
