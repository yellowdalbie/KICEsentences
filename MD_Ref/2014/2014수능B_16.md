16. 닫힌 구간 $[0, a]$ 에서 정의된 확률변수 $X$ 의 확률밀도함수가 연속이다. 확률변수 $X$ 가 다음 조건을 만족시킬 때, 상수 $k$ 의 값은? [4점]
(가) $0 \leq x \leq a$ 인 모든 $x$ 에 대하여 $\mathrm{P}(0 \leq X \leq x)=k x^{2}$ 이다. (나) $\mathrm{E}(X)=1$
(1) $\frac{9}{16}$
(2) $\frac{4}{9}$
(3) $\frac{1}{4}$
(4) $\frac{1}{9}$
(5) $\frac{1}{16}$
