16. 수열 $\left\{a_{n}\right\}$ 은 $a_{1}=3$ 이고
$$n a_{n+1}-2 n a_{n}+\frac{n+2}{n+1}=0 \quad(n \geq 1)$$

을 만족시킨다. 다음은 일반항 $a_{n}$ 을 구하는 과정이다.
\[
\begin{array}{l}
a_{n+1}-2 a_{n}+\frac{n+2}{n(n+1)}=0 \text { 에서 } \\
a_{n}-2 a_{n-1}+\frac{n+1}{n(n-1)}=0 \quad(n \geq 2) \text { 이므로 } \\
\quad a_{n+1}-a_{n}-2\left(a_{n}-a_{n-1}\right)+\frac{1}{n(n+1)}-(\text { 가 })=0 \quad(n \geq 2)
\end{array}
\]

이다. $b_{n}=a_{n+1}-a_{n}(n \geq 1)$ 이라 놓으면 $b_{1}=\frac{3}{2}$ 이고,
$$b_{n}+\frac{1}{n(n+1)}=2 b_{n-1}+\text { (가) } \quad(n \geq 2)$$

이다. 따라서
$$b_{n}+\frac{1}{n(n+1)}=2^{n} \quad(n \geq 1)$$

이다. 즉, $b_{n}=2^{n}-\frac{1}{n(n+1)} \quad(n \geq 1)$
이므로 $a_{n}=a_{1}+\sum_{k=1}^{n-1} b_{k}=$ (나) $(n \geq 2)$ 이다.
$n=1$ 일 때에도 이 식을 만족시키므로
모든 자연수 $n$ 에 대하여 $a_{n}=$ (나) 이다.

위의 (가), (나)에 알맞은 식을 각각 $f(n), g(n)$ 이라 할 때, $g(6)-f(4)$ 의 값은? [4점]
(1) 64
(2) 66
(3) 68
(4) 70
(5) 72
