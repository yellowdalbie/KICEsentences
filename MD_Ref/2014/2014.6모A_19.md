19. 수열 $\left\{a_{n}\right\}$ 은 $a_{1}=2$ 이고,
$$n^{2} a_{n+1}=\left(n^{2}-1\right) a_{n}+n(n+1) 2^{n} \quad(n \geq 1)$$

을 만족시킨다. 다음은 일반항 $a_{n}$ 을 구하는 과정이다.

주어진 식에 의하여
$$a_{n+1}=\frac{(n+1)(n-1)}{n^{2}} a_{n}+\frac{n+1}{n} 2^{n}$$

이다. $b_{n}=\frac{n-1}{n} a_{n}$ 이라 하면
\[
\begin{array}{l}
\quad b_{n+1}=b_{n}+\text { (가) } \quad(n \geq 1) \\
\text { 이고, } b_{1}=0 \text { 이므로 } \\
\quad b_{n}=\text { (나) }(n \geq 1) \\
\text { 이다. 그러므로 } \\
a_{n}=\left\{\begin{array}{ll}
2 & (n=1) \\
\frac{n}{n-1} \times(\square \text { (나) } & (n \geq 2)
\end{array}\right.
\end{array}
\]

이다.

위의 (가), (나)에 알맞은 식을 각각 $f(n), g(n)$ 이라 할 때, $f(5)+g(10)$ 의 값은? [4점]
(1) 1014
(2) 1024
(3) 1034
(4) 1044
(5) 1054
