28. 방정식 $a+b+c=9$ 를 만족시키는 음이 아닌 정수 $a, b, c$ 의 모든 순서쌍 $(a, b, c)$ 중에서 임의로 한 개를 선택할 때, 선택한 순서쌍 $(a, b, c)$ 가
$$a<2 \text { 또는 } b<2$$

를 만족시킬 확률은 $\frac{q}{p}$ 이다. $p+q$ 의 값을 구하시오. (단, $p$ 와 $q$ 는 서로소인 자연수이다.) [4점]
