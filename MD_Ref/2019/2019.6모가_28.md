28. 자연수 $n(n \geq 3)$ 에 대하여 집합 $A$ 를
$$A=\{(x, y) \mid 1 \leq x \leq y \leq n, x \text { 와 } y \text { 는 자연수 }\}$$

라 하자. 집합 $A$ 에서 임의로 선택된 한 개의 원소 $(a, b)$ 에 대하여 $b$ 가 3 의 배수일 때, $a=b$ 일 확률이 $\frac{1}{9}$ 이 되도록 하는 모든 자연수 $n$ 의 값의 합을 구하시오. [4점]
