30. 최고차항의 계수가 1 인 삼차함수 $f(x)$ 와 최고차항의 계수가 -1 인 이차함수 $g(x)$ 가 다음 조건을 만족시킨다.
(가) 곡선 $y=f(x)$ 위의 점 $(0,0)$ 에서의 접선과 곡선 $y=g(x)$ 위의 점 $(2,0)$ 에서의 접선은 모두 $x$ 축이다.
(나) 점 $(2,0)$ 에서 곡선 $y=f(x)$ 에 그은 접선의 개수는 2 이다.
(다) 방정식 $f(x)=g(x)$ 는 오직 하나의 실근을 가진다.
$x>0$ 인 모든 실수 $x$ 에 대하여
$$g(x) \leq k x-2 \leq f(x)$$

를 만족시키는 실수 $k$ 의 최댓값과 최솟값을 각각 $\alpha, \beta$ 라 할 때, $\alpha-\beta=a+b \sqrt{2}$ 이다. $a^{2}+b^{2}$ 의 값을 구하시오. (단, $a, b$ 는 유리수이다.) [4점]
