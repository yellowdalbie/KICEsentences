20. 점 $\left(-\frac{\pi}{2}, 0\right)$ 에서 곡선 $y=\sin x(x>0)$ 에 접선을 그어 접점의 $x$ 좌표를 작은 수부터 크기순으로 모두 나열할 때, $n$ 번째 수를 $a_{n}$ 이라 하자. 모든 자연수 $n$ 에 대하여 <보기>에서 옳은 것만을 있는 대로 고른 것은? [4점]
<보 기>
\[
\begin{array}{l}
\text { ㄱ. } \tan a_{n}=a_{n}+\frac{\pi}{2} \\
\text { ㄴ. } \tan a_{n+2}-\tan a_{n}>2 \pi \\
\text { ㄷ. } a_{n+1}+a_{n+2}>a_{n}+a_{n+3}
\end{array}
\]
(1) ᄀ
(2) ᄀ, ᄂ
(3) ᄀ, ㄷ
(4) ᄂ, ᄃ
(5) ᄀ, ᄂ, ᄃ
