27. 다음 조건을 만족시키는 전체집합 $U$ 의 두 부분집합 $A, B$ 에 대하여 $n(B-A)$ 의 최댓값을 구하시오. [4점]
\[
\begin{array}{l}
\text { (가) } n(U)=25 \\
\text { (나) } A \cap\left(A^{C} \cup B\right) \neq \varnothing \\
\text { (다) } n(A-B)=11
\end{array}
\]
