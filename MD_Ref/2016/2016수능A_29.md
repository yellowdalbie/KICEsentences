29. 이차함수 $f(x)$ 가 $f(0)=0$ 이고 다음 조건을 만족시킨다.
\[
\begin{array}{l}
\text { (가) } \int_{0}^{2}|f(x)| d x=-\int_{0}^{2} f(x) d x=4 \\
\text { (나) } \int_{2}^{3}|f(x)| d x=\int_{2}^{3} f(x) d x
\end{array}
\]
$f(5)$ 의 값을 구하시오. [4점]
