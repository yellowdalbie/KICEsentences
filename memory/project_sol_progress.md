## a_type_excluded.txt 주의사항

파일 형식: `연도/파일명.md` (예: `2016/2016.6모A_01.md`)
비교 시 반드시 `Path(line).stem`으로 stem 추출 후 NFC 정규화하여 비교해야 함.
단순 `line.strip()`과 `f.stem` 비교 시 형식 불일치로 필터 실패.

**Why:** 이 버그로 인해 행렬/수열극한 문항들이 타겟에 포함됐고, 에이전트가 잘못 작성하는 사고 발생.
**How to apply:** 타겟 계산 스크립트에서 반드시 filter_a_type.py의 EXCLUDE_PATTERNS를 직접 적용하거나, filter_a_type.py를 실행해서 "해설 작성 대상" 섹션을 파싱할 것.
