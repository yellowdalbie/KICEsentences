## [Step 1] 확률의 합이 1인 조건에서 정규분포 대칭성을 이용하여 평균 결정하기

- **Trigger**: [정규분포 대칭 조건] (가) $P(X \geq 1)+P(X \geq 7)=1$
- **Action**: [12확통03-04] $P(X \geq a)+P(X \geq b)=1$이 성립하려면 $a+b=2\mu$여야 함 (대칭성). $1+7=8=2\mu$ → $\mu=4$
- **Result**: $\mu = 4$

> **📝 해설 (Explanation)**
> $X \sim N(\mu, \sigma^2)$에서 $P(X \geq a) = P\left(Z \geq \frac{a-\mu}{\sigma}\right)$
>
> $P(X \geq 1) + P(X \geq 7) = 1$이 되려면
>
> $P(X \geq 1) = 1 - P(X \geq 7) = P(X \leq 7)$
>
> 이는 $X \geq 1$과 $X \leq 7$이 서로 여사건 관계이므로 $\mu$를 기준으로 $1$과 $7$이 대칭이어야 한다.
>
> $$\mu = \frac{1+7}{2} = 4$$

## [Step 2] 정규분포 확률 등식 표준화로 표준편차 역산

- **Trigger**: [정규분포 확률 등식 조건] (나) $P(|X-5| \leq 3)+P(X \leq 0)=P(Z \leq 1)$, $\mu=4$ 적용
- **Action**: [12확통03-04] $P(|X-5| \leq 3) = P(2 \leq X \leq 8)$. 표준화하여 $\sigma$에 대한 방정식 수립
- **Result**: $\sigma = 2$

> **📝 해설 (Explanation)**
> $\mu=4$로 $X \sim N(4, \sigma^2)$
>
> $P(|X-5| \leq 3) = P(2 \leq X \leq 8)$
>
> 표준화: $P\!\left(\frac{2-4}{\sigma} \leq Z \leq \frac{8-4}{\sigma}\right) = P\!\left(\frac{-2}{\sigma} \leq Z \leq \frac{4}{\sigma}\right)$
>
> $P(X \leq 0) = P\!\left(Z \leq \frac{0-4}{\sigma}\right) = P\!\left(Z \leq \frac{-4}{\sigma}\right)$
>
> 우변 $P(Z \leq 1) = 0.5 + P(0 \leq Z \leq 1)$
>
> 조건 (나)를 쓰면:
>
> $$P\!\left(\frac{-2}{\sigma} \leq Z \leq \frac{4}{\sigma}\right) + P\!\left(Z \leq \frac{-4}{\sigma}\right) = P(Z \leq 1)$$
>
> $P\!\left(Z \leq \frac{4}{\sigma}\right) - P\!\left(Z \leq \frac{-2}{\sigma}\right) + P\!\left(Z \leq \frac{-4}{\sigma}\right) = P(Z \leq 1)$
>
> 대칭성: $P(Z \leq -c)=P(Z \geq c)=1-P(Z \leq c)$
>
> $\sigma=2$로 시험: $\frac{-2}{\sigma}=-1$, $\frac{4}{\sigma}=2$, $\frac{-4}{\sigma}=-2$
>
> $P(-1 \leq Z \leq 2) + P(Z \leq -2)$
> $= P(Z \leq 2) - P(Z \leq -1) + P(Z \leq -2)$
> $= P(Z \leq 2) - P(Z \geq 1) + P(Z \geq 2)$
> $= P(Z \leq 2) - (1-P(Z \leq 1)) + (1-P(Z \leq 2))$
> $= P(Z \leq 1)$ ✓
>
> 따라서 $\sigma = 2$이고 정답은 **②**입니다.
