## [Step 1] 코사인값으로 각도를 확정하고 사인법칙으로 변의 길이 비율 설정하기

- **Trigger**: [사인법칙 조건] $\cos A = -\frac{1}{2}$, $\sin B : \sin C = 5:3$인 삼각형 ABC
- **Action**: [12대수02-03] $\cos A = -\frac{1}{2}$ → $A = 120°$. 사인법칙에 의해 $\sin B : \sin C = BC : CA = 5:3$
- **Result**: $A = 120°$, $BC = 5k$, $CA = 3k$ ($k > 0$)로 설정

> **📝 해설 (Explanation)**
> $\cos A = -\frac{1}{2}$이므로 $A = 120°$
>
> 사인법칙에 의해 $\frac{a}{\sin A}=\frac{b}{\sin B}=\frac{c}{\sin C}$ ($a=BC, b=CA, c=AB$)
>
> $\sin B : \sin C = b : c$가 아니라 $\sin B : \sin C = \frac{b}{2R} : \frac{c}{2R}$에서
>
> 사인법칙: $\sin B = \frac{b}{2R}$, $\sin C = \frac{c}{2R}$이 아니라 $a = 2R\sin A$이므로
>
> $\sin B : \sin C = AC : AB = b : c = 5:3$ → 단, 표준 사인법칙: $\frac{a}{\sin A}=\frac{b}{\sin B}=\frac{c}{\sin C}$
>
> 따라서 $\frac{b}{\sin B}=\frac{c}{\sin C}$ → $\sin B : \sin C = b : c$
>
> 즉 $b:c = CA:AB = 5:3$이므로 $CA=5k$, $AB=3k$

## [Step 2] 두 변의 비율과 끼인각을 코사인법칙에 대입하여 대변의 길이 계산하기

- **Trigger**: [코사인법칙 조건] $A=120°$, $b=CA=5k$, $c=AB=3k$
- **Action**: [12대수02-03] $a^2 = b^2+c^2-2bc\cos A = 25k^2+9k^2-2(5k)(3k)(-\frac{1}{2})$
- **Result**: $a^2 = 49k^2$ → $a = BC = 7k$

> **📝 해설 (Explanation)**
> $$a^2 = (5k)^2+(3k)^2-2(5k)(3k)\cos120°$$
>
> $$= 25k^2+9k^2-30k^2\cdot\left(-\frac{1}{2}\right) = 34k^2+15k^2 = 49k^2$$
>
> $$a = 7k$$

## [Step 3] 이등변삼각형 코사인법칙으로 PB 계산 후 스케일 인수 확정

- **Trigger**: [원의 교점 조건] A를 중심으로 B를 지나는 원이 AC, BC와 만나는 점 P, Q 설정, PB=3√3
- **Action**: [12대수02-03] 원의 반지름 $r=AB=3k$. P는 AC 위에서 AP=r=3k이므로 $PC=5k-3k=2k$. 삼각형 BPC에서 코사인법칙으로 PB 계산
- **Result**: $PB = 3\sqrt{3}$에서 $k$ 결정

> **📝 해설 (Explanation)**
> 원의 반지름 = $AB = c = 3k$이므로 $AP = AB = 3k$
>
> $P$는 $AC$ 위의 점으로 $AP=3k$, $PC=5k-3k=2k$
>
> 삼각형 BPC에서 $\angle BCA = C$를 사인법칙으로 구한다.
>
> $\cos C = \frac{b^2+a^2-c^2}{2ab} = \frac{(5k)^2+(7k)^2-(3k)^2}{2(5k)(7k)} = \frac{25+49-9}{70}k^2/(k^2) = \frac{65}{70} = \frac{13}{14}$
>
> 삼각형 BPC에서 $PC=2k$, $BC=7k$, $\angle PCB = C$:
>
> $$PB^2 = PC^2+BC^2-2\cdot PC\cdot BC\cdot\cos C$$
>
> $$= 4k^2+49k^2-2(2k)(7k)\cdot\frac{13}{14}$$
>
> $$= 53k^2-28k^2\cdot\frac{13}{14} = 53k^2-26k^2 = 27k^2$$
>
> $$PB = 3\sqrt{3}k$$
>
> $3\sqrt{3}k = 3\sqrt{3}$ → $k=1$
>
> 따라서 $a=7$, $b=5$, $c=3$

## [Step 4] 이등변삼각형 성질로 BQ 산출 후 삼각형 PQC 코사인법칙으로 PQ 계산

- **Trigger**: [원과 직선 교점 조건] 원(중심 A, 반지름 3)이 BC와 만나는 점 Q (B가 아닌 점)
- **Action**: [12대수02-03] 삼각형 ABQ에서 AB=AQ=3(반지름), $\angle ABQ = B$ 이용하여 BQ 계산, 이후 사인법칙으로 PQ 결정
- **Result**: $PQ = \frac{9}{14}$

> **📝 해설 (Explanation)**
> $k=1$로 $AB=AQ=3$ (반지름), $BC=7$
>
> 이등변삼각형 ABQ: $AB=AQ=3$, $\angle ABQ=B$
>
> $\cos B = \frac{a^2+c^2-b^2}{2ac} = \frac{49+9-25}{42} = \frac{33}{42} = \frac{11}{14}$
>
> 삼각형 ABQ에서 $AB=AQ=3$, $\angle ABQ=B$이면 $\angle AQB=B$도 동일 (이등변삼각형).
>
> $\angle BAQ = \pi-2B$
>
> 코사인법칙으로 BQ:
>
> $BQ^2 = AB^2+AQ^2-2\cdot AB\cdot AQ\cdot\cos(\angle BAQ)$
>
> $= 9+9-18\cos(\pi-2B) = 18+18\cos2B = 18(1+2\cos^2 B-1) = 36\cos^2 B$
>
> $BQ = 6\cos B = 6\cdot\frac{11}{14} = \frac{33}{7}$
>
> $QC = BC-BQ = 7-\frac{33}{7} = \frac{16}{7}$
>
> 삼각형 PQC에서 $PC=2$, $QC=\frac{16}{7}$, $\angle PCQ=C$:
>
> $\cos C = \frac{13}{14}$
>
> $PQ^2 = PC^2+QC^2-2\cdot PC\cdot QC\cdot\cos C$
>
> $= 4+\frac{256}{49}-2\cdot2\cdot\frac{16}{7}\cdot\frac{13}{14}$
>
> $= 4+\frac{256}{49}-\frac{832}{98}$
>
> $= 4+\frac{256}{49}-\frac{416}{49}$
>
> $= 4-\frac{160}{49} = \frac{196-160}{49} = \frac{36}{49}$
>
> $PQ = \frac{6}{7}$
>
> 따라서 정답은 **$\dfrac{6}{7}$**입니다.
