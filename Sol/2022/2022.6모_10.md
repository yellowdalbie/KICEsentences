## [Step 1] 로그의 성질을 이용하여 로그방정식을 다항방정식으로 구하기

- **Trigger**: [로그방정식의 교점 제시] $y=\log _{n} x$, $y=-\log _{n}(x+3)+1$ (교점의 $x$좌표 조건)
- **Action**: [12대수01-04] 로그의 성질($\log_a M + \log_a N = \log_a MN$)을 적용하여 두 곡선의 식을 $x$에 관한 다항방정식으로 변환
- **Result**: $x^2+3x=n$ ($x>0$ 범위의 실근)

> **📝 해설 (Explanation)**
> 두 곡선이 만나는 점의 $x$좌표를 구하기 위해 두 함숫값이 같다고 놓습니다.
> $\log_n x = -\log_n(x+3)+1$
> 로그 항을 좌변으로 모으면 다음과 같습니다.
> $\log_n x + \log_n(x+3) = 1$
> 로그의 성질을 이용하여 좌변을 하나의 로그로 합칩니다.
> $\log_n \{x(x+3)\} = 1$
> 로그의 정의에 따라 $x(x+3) = n^1$ 이므로, $x^2+3x=n$ 이 성립합니다.
> 이때 로그의 진수 조건에 의해 $x>0$ 이고 $x+3>0$ 이어야 하므로, 구하는 교점의 $x$좌표는 양수입니다.

## [Step 2] 이차함수의 증가 성질을 이용하여 실근의 조건을 만족하는 자연수 구하기

- **Trigger**: [실근의 위치 조건] 교점의 $x$좌표가 $1 < x < 2$ 사이에 존재
- **Action**: [10공수1-02-04] $f(x)=x^2+3x$가 해당 구간에서 증가함을 이용하여 경계에서의 함숫값 범위 $f(1) < n < f(2)$를 만족하는 자연수 구하기
- **Result**: $35$

> **📝 해설 (Explanation)**
> 교점의 $x$좌표가 $1 < x < 2$ 사이에 존재할 조건을 구하기 위해 함수 $f(x)=x^2+3x$를 이용합니다.
> 이 함수는 $x > -1.5$인 영역에서 $x$의 값이 커지면 $f(x)$의 값도 커지는 증가하는 성질을 가집니다.
> 따라서 실근 $x$가 $1 < x < 2$ 범위에 존재하려면, 상수 $n$의 값은 해당 구간의 양 끝점에서의 함숫값 사이에 위치해야 합니다.
> $f(1) < n < f(2)$
> 각 경계값을 대입하면 $f(1) = 1^2 + 3(1) = 4$, $f(2) = 2^2 + 3(2) = 10$ 이므로, 자연수 $n$이 만족해야 하는 범위는 $4 < n < 10$ 입니다.
> 이를 만족하는 자연수 $n$은 $5, 6, 7, 8, 9$ 이며, 이들의 합은 $5+6+7+8+9 = 35$ 입니다.
> 따라서 정답은 **②**입니다.
