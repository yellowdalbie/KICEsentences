## [Step 1] 확률밀도함수의 성질을 이용하여 미정계수의 값 결정하기

- **Trigger**: [확률밀도함수의 성질] $0 \le x \le 6$에서 정의된 $f(x), g(x)$에 대하여 전 구간의 넓이의 합은 1
- **Action**: [CPT-STA-STA-001] $f(x)+g(x)=k$ 관계를 이용하여 두 함수 그래프와 $x$축 사이의 넓이의 합을 $k$에 관한 식으로 표현
- **Result**: $k = \frac{1}{3}$

> **📝 해설 (Explanation)**
> 확률변수 $X$와 $Y$의 확률밀도함수 $f(x), g(x)$의 그래프와 $x$축 사이의 넓이는 각각 1입니다.
> 문제에서 주어진 조건 $f(x)+g(x)=k$는 두 함수의 함숫값의 합이 항상 $k$임을 의미합니다.
> $0 \le x \le 6$ 구간에서 두 함수 그래프와 $x$축 사이의 넓이의 합은 다음과 같습니다.
> (함수 $f(x)$의 넓이) + (함수 $g(x)$의 넓이) = (가로 6, 세로 $k$인 직사각형의 넓이)
> $$1 + 1 = 6 \times k$$
> 그러므로 $6k = 2$에서 $k = \frac{1}{3}$입니다.

## [Step 2] 결정된 미정계수를 이용하여 확률변수의 특정 구간에 대한 확률 표현하기

- **Trigger**: [확률 계산 조건] $P(6k \le Y \le 15k)$ 산출 요구
- **Action**: [CPT-STA-STA-001] $k=1/3$을 대입하여 $P(2 \le Y \le 5)$를 구간 $[2, 5]$에서의 $g(x)$ 그래프 넓이로 해석
- **Result**: $P(2 \le Y \le 5) = (\text{가로 } 3, \text{세로 } k \text{인 직사각형 넓이}) - (\text{구간 } [2, 5] \text{에서 } f(x) \text{의 넓이})$

> **📝 해설 (Explanation)**
> 구하는 확률은 $P(6k \le Y \le 15k)$이고 $k = \frac{1}{3}$이므로 $P(2 \le Y \le 5)$입니다.
> 조건 $g(x) = k - f(x)$에 의해 구간 $[2, 5]$에서 $g(x)$의 넓이는 다음과 같습니다.
> $$P(2 \le Y \le 5) = (\text{가로 } 3, \text{세로 } k \text{인 직사각형 넓이}) - (\text{구간 } [2, 5] \text{에서 } f(x) \text{의 넓이})$$
> $$= 3 \times \frac{1}{3} - (\text{구간 } [2, 5] \text{에서 } f(x) \text{의 넓이}) = 1 - (\text{구간 } [2, 5] \text{에서 } f(x) \text{의 넓이})$$
> 그러므로 구간 $[2, 5]$에서 $f(x)$ 그래프와 $x$축 사이의 넓이를 구하면 됩니다.

## [Step 3] 함수의 그래프 개형 분석을 이용하여 특정 구간의 확률값 산출하기

- **Trigger**: [도형의 넓이 활용] $f(x)$가 점 (0,0), (3, 1/4), (5, 1/4), (6,0)을 지나는 직선들로 구성됨
- **Action**: [CPT-STA-STA-001] 구간 $[2, 3]$의 사다리꼴 넓이와 구간 $[3, 5]$의 직사각형 넓이 계산 및 합산
- **Result**: $P = \frac{7}{24}, p+q=31$

> **📝 해설 (Explanation)**
> $f(x)$의 그래프에서 구간 $[2, 5]$의 넓이를 구하기 위해 나누어 분석합니다.
> 1) **구간 $[2, 3]$의 넓이**: $(0,0)$과 $(3, 1/4)$을 잇는 직선의 방정식은 $y = \frac{1}{12}x$입니다.
>    - $x=2$일 때 $f(2) = \frac{2}{12} = \frac{1}{6}$이고, $x=3$일 때 $f(3) = \frac{1}{4}$입니다.
>    - 윗변 $\frac{1}{6}$, 아랫변 $\frac{1}{4}$, 높이 1인 사다리꼴의 넓이는 $\frac{1}{2} \times \left(\frac{1}{6} + \frac{1}{4}\right) \times 1 = \frac{5}{24}$입니다.
> 2) **구간 $[3, 5]$의 넓이**: 가로 $5-3=2$, 세로 $\frac{1}{4}$인 직사각형의 넓이는 $2 \times \frac{1}{4} = \frac{1}{2} = \frac{12}{24}$입니다.
>
> 그러므로 구간 $[2, 5]$에서 $f(x)$의 넓이의 합은 $\frac{5}{24} + \frac{12}{24} = \frac{17}{24}$입니다.
> 이를 Step 2의 식에 대입하면 우리가 구하는 확률은 다음과 같습니다.
> $$P(2 \le Y \le 5) = 1 - \frac{17}{24} = \frac{7}{24}$$
> 따라서 $p=24, q=7$이고 두 수는 서로소인 자연수이므로 $p+q = 24+7 = 31$입니다.
>
> 따라서 정답은 **31**입니다.
