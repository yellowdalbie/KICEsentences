## [Step 1] 등차수열의 대칭성을 이용하여 첫째항과 공차 사이의 관계식 수립하기

- **Trigger**: [등차수열의 조건] $a_3+a_5=0$
- **Action**: [CPT-AL1-SEQ-001] 등차중항의 성질을 이용하여 $a_4$의 값을 구하고, 이를 첫째항 $a$와 공차 $d$에 관한 식으로 표현
- **Result**: $a_4=0 \implies a = -3d$

> **📝 해설 (Explanation)**
> 등차수열 $\{a_n\}$의 공차를 $d$라 할 때, 등차중항의 성질에 의해
> $$a_3 + a_5 = 2a_4 = 0 \implies a_4 = 0$$
> 입니다. 일반항 $a_n = a + (n-1)d$를 적용하면 $a_4 = a + 3d = 0$이므로 $a = -3d$가 성립합니다.
> 이때 수열의 항들은 $-3d, -2d, -d, 0, d, 2d, 3d, \dots$와 같은 형태를 가집니다.

## [Step 2] 절댓값이 포함된 합의 조건을 분석하여 공차의 값 결정하기

- **Trigger**: [수열의 합 조건] $\sum_{k=1}^{6}(|a_k|+a_k)=30$
- **Action**: [CPT-AL1-SEQ-001] $a_k$의 부호에 따라 $|a_k|+a_k$가 0이 되는 지점을 파악하고, $d$의 범위를 고려하여 식 계산
- **Result**: $d=3$ (공차는 정수)

> **📝 해설 (Explanation)**
> $|a_k| + a_k$는 $a_k \ge 0$이면 $2a_k$이고, $a_k < 0$이면 $0$입니다.
> 1) **$d > 0$일 때**: $a_1, a_2, a_3$은 음수이고 $a_4=0$, $a_5, a_6$은 양수입니다.
>    $$\sum_{k=1}^{6}(|a_k|+a_k) = 0 + 0 + 0 + 2a_4 + 2a_5 + 2a_6 = 0 + 2(d) + 2(2d) = 6d$$
>    $6d = 30$에서 $d = 5$입니다. (정수 조건 만족)
> 2) **$d < 0$일 때**: $a_1, a_2, a_3$은 양수이고 $a_4=0$, $a_5, a_6$은 음수입니다.
>    $$\sum_{k=1}^{6}(|a_k|+a_k) = 2a_1 + 2a_2 + 2a_3 + 2a_4 + 0 + 0 = 2(-3d) + 2(-2d) + 2(-d) + 0 = -12d$$
>    $-12d = 30$에서 $d = -2.5$입니다. 이는 공차가 정수라는 조건에 모순됩니다.
> 그러므로 공차 $d=5$입니다. (참고: $d=0$이면 모든 항이 0이 되어 합이 30일 수 없습니다.)

## [Step 3] 일반항의 공식을 적용하여 수열의 특정 번째 항의 값 계산하기

- **Trigger**: [구하고자 하는 값] $a_9$
- **Action**: [CPT-AL1-SEQ-001] 구한 $a$와 $d$를 일반항 공식에 대입하여 계산
- **Result**: 25

> **📝 해설 (Explanation)**
> $d=5$이면 단계 1에 의해 $a = -3(5) = -15$입니다.
> 구하고자 하는 아홉 번째 항은 다음과 같습니다.
> $$a_9 = a + 8d = -15 + 8(5) = -15 + 40 = 25$$
> 입니다.
>
> 따라서 정답은 **25**입니다.
