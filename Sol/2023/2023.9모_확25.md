# 2023.9모_확25.md 해설

## [Step 1] 정규분포의 성질과 표준화를 활용하여 조건을 만족시키는 상수 구하기

- **Trigger**: [정규분포 표준화 조건] 서로 다른 정규분포를 따르는 두 확률변수의 구간 확률이 같다는 조건
- **Action**: [12확통03-04] 표준화 변수 $Z = \frac{X-m}{\sigma}$ 를 이용하여 두 확률을 표준정규분포상의 확률로 변환하고, 표준정규분포의 대칭성을 이용하여 미지수를 확정함
- **Result**: $20.25$ (상수)

> **📝 해설 (Explanation)**
> 두 확률변수 $X \sim N(9, 0.4^2), Y \sim N(20, 1^2)$ 에 대하여 각각의 확률을 표준화합니다.
> 1) $P(8.9 \le X \le 9.4) = P\left(\frac{8.9-9}{0.4} \le Z \le \frac{9.4-9}{0.4}\right) = P(-0.25 \le Z \le 1)$
> 2) $P(19 \le Y \le k) = P\left(\frac{19-20}{1} \le Z \le \frac{k-20}{1}\right) = P(-1 \le Z \le k-20)$
> 표준정규분포의 대칭성에 의해 $P(-0.25 \le Z \le 1) = P(-1 \le Z \le 0.25)$ 이므로
> $P(-1 \le Z \le k-20) = P(-1 \le Z \le 0.25)$ 에서 $k-20 = 0.25$ 임을 알 수 있습니다.
> 그러므로 $k = 20.25$ 입니다.
> 
> 따라서 정답은 **④**입니다.
