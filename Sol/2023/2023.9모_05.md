## [Step 1] 등차수열의 일반항을 이용하여 특정 항의 값 구하기

- **Trigger**: [등차수열 조건] 첫째항과 공차에 관한 관계식이 주어지는 상황
- **Action**: [12대수03-02] 등차수열의 일반항 $a_n = a_1 + (n-1)d$ 를 대입하여 연립방정식을 풀고 목표 항의 값을 구함
- **Result**: $21$

> **📝 해설 (Explanation)**
> $a_1 = 2a_5 = 2(a_1 + 4d)$ 에서 $a_1 = -8d$ 입니다.
> $a_8 + a_{12} = (a_1 + 7d) + (a_1 + 11d) = 2a_1 + 18d = -6$ 입니다.
> $a_1 = -8d$ 를 대입하면 $2d = -6$ 이므로 $d = -3, a_1 = 24$ 입니다.
> 결과적으로 $a_2 = a_1 + d = 24 - 3 = 21$ 입니다.
> 
> 따라서 정답은 **③**입니다.
