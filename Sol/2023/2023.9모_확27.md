## [Step 1] 이산확률변수의 성질과 기댓값의 정의를 활용하여 식의 값 구하기

- **Trigger**: [이산확률변수 조건] 확률변수의 기댓값과 표준편차가 같다는 조건이 주어지는 상황
- **Action**: [12확통03-02] $V(X) = \{E(X)\}^2$ 관계를 이용하여 확률변수의 미정계수를 구하고, 기댓값의 정의를 적용하여 최종 식의 값을 도출함
- **Result**: $45$ (식의 값)

> **📝 해설 (Explanation)**
> $\sigma(X) = E(X)$ 이므로 $V(X) = E(X^2) - \{E(X)\}^2 = \{E(X)\}^2$ 에서 $E(X^2) = 2\{E(X)\}^2$ 가 성립합니다.
> $E(X) = \frac{1}{2} + \frac{2a}{5}$ 이고 $E(X^2) = \frac{1}{2} + \frac{2a^2}{5}$ 이므로
> $\frac{1}{2} + \frac{2a^2}{5} = 2\left(\frac{1}{2} + \frac{2a}{5}\right)^2 = \frac{1}{2} + \frac{4a}{5} + \frac{8a^2}{25}$ 가 성립합니다.
> 이를 정리하면 $\frac{2a^2}{25} - \frac{4a}{5} = 0 \Rightarrow \frac{2a}{25}(a - 10) = 0$ 이므로 $a=10$ ($a>1$) 입니다.
> $a=10$ 을 대입하면 $E(X) = \frac{9}{2}, E(X^2) = \frac{81}{2}$ 입니다.
> 그러므로 $E(X^2) + E(X) = \frac{81+9}{2} = 45$ 입니다.
> 
> 따라서 정답은 **⑤**입니다.
