## [Step 1] 항등식을 이용하여 도함수의 관계식 도출하기

- **Trigger**: [다항식의 항등식 조건] 모든 실수 $x$ 에 대하여 $f(x)=f(1)+(x-1) f^{\prime}(g(x))$ (미분계수를 포함한 항등식)
- **Action**: [CPT-CM1-POL-002] $f(x) = x^3 + ax^2 + bx + c$로 두고 $x \neq 1$일 때 양변을 $x-1$로 나누어 $f'(g(x))$를 $x$에 대한 이차식으로 표현
- **Result**: $3\{g(x)\}^2 + 2ag(x) = x^2 + (a+1)x + a+1$

> **📝 해설 (Explanation)**
> 최고차항의 계수가 $1$인 삼차함수를 $f(x) = x^3 + a x^2 + b x + c$라고 합시다.
> 도함수는 $f'(x) = 3x^2 + 2ax + b$입니다.
> 조건 (가)의 식에서 $f(1)$을 좌변으로 이항하고 $x \neq 1$일 때 양변을 $x-1$로 나누면 $\frac{f(x)-f(1)}{x-1} = f'(g(x))$가 됩니다.
> 좌변을 계산하면 $\frac{x^3 - 1 + a(x^2 - 1) + b(x-1)}{x-1} = x^2 + x + 1 + a(x+1) + b = x^2 + (a+1)x + a + b + 1$입니다.
> 우변은 $f'(g(x)) = 3\{g(x)\}^2 + 2ag(x) + b$입니다.
> 양변의 $b$를 소거하면 $3\{g(x)\}^2 + 2ag(x) = x^2 + (a+1)x + a + 1$이라는 항등식을 얻을 수 있습니다.

## [Step 2] 최솟값 조건을 분석하여 함수의 미지수 확정하기

- **Trigger**: [다항함수의 극값 조건] 함수 $g(x)$의 최솟값은 $\frac{5}{2}$ (합성함수 구조에서의 최솟값)
- **Action**: [CPT-CA1-DIF-007] 양변의 함수의 최솟값이 같아야 함을 바탕으로 $a$에 대한 이차방정식을 수립하고, 대칭축 조건을 통해 유효한 해 선택
- **Result**: $a = -6$

> **📝 해설 (Explanation)**
> 우변의 이차식 $x^2 + (a+1)x + a + 1$은 완전제곱식으로 변형하면 $(x + \frac{a+1}{2})^2 - \frac{(a+1)^2}{4} + a + 1$이 되므로, $x = -\frac{a+1}{2}$에서 최솟값 $-\frac{(a+1)^2}{4} + a + 1$을 가집니다.
> 조건 (나)에서 실수 전체에서 연속인 함수 $g(x)$의 최솟값이 $\frac{5}{2}$입니다.
> 따라서 좌변의 함수를 $k(t) = 3t^2 + 2at$라 할 때, $g(x)$가 취할 수 있는 값의 범위인 $t \ge \frac{5}{2}$에서 $k(t)$의 최솟값을 구해야 합니다.
> $k(t)$의 대칭축은 $t = -\frac{a}{3}$입니다. 만약 $-\frac{a}{3} \le \frac{5}{2}$라면, $k(t)$는 $t \ge \frac{5}{2}$ 구간에서 단조 증가하므로 $t = \frac{5}{2}$일 때 최솟값을 가집니다.
> 양변의 최솟값이 일치해야 하므로 $k(\frac{5}{2}) = 3(\frac{25}{4}) + 2a(\frac{5}{2}) = \frac{75}{4} + 5a$가 우변의 최솟값 $-\frac{a^2 + 2a + 1}{4} + \frac{4a + 4}{4}$와 같아야 합니다.
> 식을 정리하면 $75 + 20a = -a^2 + 2a + 3$에서 $a^2 + 18a + 72 = 0$이 되어 $(a+6)(a+12) = 0$입니다.
> $a = -6$일 때 $-\frac{a}{3} = 2 \le \frac{5}{2}$ 조건을 만족하지만, $a = -12$일 때는 $-\frac{a}{3} = 4 \le \frac{5}{2}$가 거짓이 되어 모순이 발생합니다.
> 따라서 $a = -6$입니다.

## [Step 3] 주어진 함숫값 조건을 대입하여 함수의 나머지 계수 산출하기

- **Trigger**: [주어진 함숫값 조건] $f(0)=-3, f(g(1))=6$ (특정 함숫값 조건)
- **Action**: [CPT-CM1-EQU-002] 도출된 관계식에 $x=1$을 대입하여 이차방정식을 풀고 조건에 맞는 $g(1)$을 구한 뒤, 이를 활용하여 $b$와 $c$ 연산
- **Result**: $b = 12$, $c = -3$ (함수식 $f(x) = x^3 - 6x^2 + 12x - 3$ 확정)

> **📝 해설 (Explanation)**
> 조건 (다)에서 $f(0) = -3$이므로 $c = -3$입니다.
> Step 1에서 구한 관계식에 $x=1$, $a=-6$을 대입하면 $3\{g(1)\}^2 - 12g(1) = 1 - 5 - 5$가 되어 $3\{g(1)\}^2 - 12g(1) + 9 = 0$입니다.
> 양변을 $3$으로 나누어 인수분해하면 $(g(1)-1)(g(1)-3) = 0$이 되어 $g(1) = 1$ 또는 $g(1) = 3$입니다.
> $g(x)$의 최솟값이 $\frac{5}{2}$이므로 $g(1) = 3$이어야 합니다.
> 다시 조건 (다)에 의해 $f(g(1)) = 6$이므로 $f(3) = 6$입니다.
> $f(3) = 27 - 6(9) + 3b - 3 = 3b - 30$이므로 $3b - 30 = 6$을 풀면 $3b = 36$, 즉 $b = 12$입니다.
> 따라서 함수는 $f(x) = x^3 - 6x^2 + 12x - 3$으로 완전히 확정됩니다.

## [Step 4] 확정된 함수식에 대입하여 최종 함숫값 연산하기

- **Trigger**: [최종 구하는 값] $f(4)$
- **Action**: [CPT-CM1-POL-001] 확정된 $f(x)$ 식에 $x=4$를 대입하여 사칙연산 수행
- **Result**: $13$

> **📝 해설 (Explanation)**
> 앞선 과정을 통해 삼차함수의 식이 $f(x) = x^3 - 6x^2 + 12x - 3$임을 구했습니다.
> 최종적으로 구하고자 하는 값은 $f(4)$이므로, 식에 $x=4$를 대입하여 계산합니다.
> $f(4) = 4^3 - 6(4^2) + 12(4) - 3 = 64 - 96 + 48 - 3$입니다.
> 덧셈과 뺄셈을 순서대로 연산하면 $112 - 99 = 13$입니다.
> 정답은 **13**입니다.
