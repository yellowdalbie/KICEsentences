## [Step 1] 등차수열의 성질을 이용하여 특정 항의 값 구하기

- **Trigger**: [등차수열 조건] 공차 $d=3, a_5 \times a_7 < 0, \sum_{k=1}^{6}|a_{k+6}| = 6 + \sum_{k=1}^{6}|a_{2k}|$
- **Action**: [12대수03-02] $a_5$와 $a_7$의 부호 조건을 분석하여 $a_6$의 범위를 파악하고, 주어진 절댓값 합의 관계식을 $a_6$에 대한 방정식으로 세워 $a_{10}$의 값을 구함.
- **Result**: $a_{10} = \frac{23}{2}$

> **📝 해설 (Explanation)**
> 공차가 $3$인 등차수열 $\{a_n\}$에서 $a_5 \times a_7 < 0$이므로 $a_5 < 0, a_7 > 0$입니다.
> $a_6 = a_5 + 3, a_7 = a_6 + 3$에서 $a_6 - 3 < 0, a_6 + 3 > 0$이므로 $-3 < a_6 < 3$입니다.
>
> 1. 좌변의 식 정리:
> $\sum_{k=1}^{6}|a_{k+6}| = |a_7| + |a_8| + \dots + |a_{12}|$
> $a_7 > 0$이므로 모든 항이 양수입니다.
> $\sum_{k=1}^{6} (a_6 + k \times 3) = 6a_6 + 3(1+2+3+4+5+6) = 6a_6 + 63$입니다.
>
> 2. 우변의 식 정리:
> $6 + \sum_{k=1}^{6}|a_{2k}| = 6 + (|a_2| + |a_4| + |a_6| + |a_8| + |a_{10}| + |a_{12}|)$
> $a_5 < 0$이므로 $a_2, a_4$는 음수이고, $a_7 > 0$이므로 $a_8, a_{10}, a_{12}$는 양수입니다.
> $6 + \{-(a_6-12) - (a_6-6) + |a_6| + (a_6+6) + (a_6+12) + (a_6+18)\}$
> $= 6 + (a_6 + 54 + |a_6|) = a_6 + |a_6| + 60$입니다.
>
> 3. 방정식 풀이:
> $6a_6 + 63 = a_6 + |a_6| + 60 \Rightarrow 5a_6 + 3 = |a_6|$
> - $a_6 \ge 0$이면 $5a_6+3=a_6$에서 $a_6 = -0.75$ (모순)
> - $a_6 < 0$이면 $5a_6+3=-a_6$에서 $a_6 = -0.5$ (적합)
>
> 결과적으로 $a_{10} = a_6 + 4d = -0.5 + 4 \times 3 = 11.5 = \frac{23}{2}$입니다.
> 따라서 정답은 **③**입니다.
