## [Step 1] 공차와 부호 조건을 분석하여 특정 항의 범위 파악하기

- **Trigger**: [등차수열의 조건식 제시] 공차 $d=3$, $a_5 \times a_7 < 0$ (부호 변화 조건)
- **Action**: [CPT-ALG-SEQ-001] $a_6$이 부호가 바뀌는 지점임을 파약하고, $a_5$와 $a_7$ 사이의 관계를 통해 $a_6$의 가능한 부호 탐색
- **Result**: $a_5 < 0, a_7 > 0$ 이며 $a_6$은 $-3 < a_6 < 3$ 범위에 존재

> **📝 해설 (Explanation)**
> 등차수열 $\{a_n\}$의 공차가 $3$으로 양수이므로, 항의 번호가 커질수록 값도 커집니다.
> $a_5 \times a_7 < 0$이므로 $a_5 < 0$이고 $a_7 > 0$이어야 합니다.
> $a_6 = a_5 + 3$이고 $a_7 = a_6 + 3$입니다.
> $a_5 < 0 \Rightarrow a_6 - 3 < 0 \Rightarrow a_6 < 3$
> $a_7 > 0 \Rightarrow a_6 + 3 > 0 \Rightarrow a_6 > -3$
> 따라서 $a_6$의 범위는 $-3 < a_6 < 3$입니다. $a_6$이 $0$인지 아닌지에 따라 절댓값의 처리가 달라질 수 있음을 유의합니다.

## [Step 2] 등차수열의 성질을 이용하여 절댓값 합의 관계식 정리하기

- **Trigger**: [시그마 기호식 조건] $\sum_{k=1}^{6}|a_{k+6}| = 6 + \sum_{k=1}^{6}|a_{2k}|$
- **Action**: [CPT-ALG-SEQ-001] 각 항을 $a_6$과 공차 $d=3$에 대한 식으로 표현하고, 각 항의 부호를 판별하여 절댓값 기호를 풀이
- **Result**: $a_6 = -\frac{1}{2}$

> **📝 해설 (Explanation)**
> 좌측 합: $|a_7| + |a_8| + |a_9| + |a_{10}| + |a_{11}| + |a_{12}|$
> $a_7 > 0$이므로 이후의 모든 항은 양수입니다.
> $\text{LHS} = a_7 + a_8 + a_9 + a_{10} + a_{11} + a_{12} = 6a_6 + (1+2+3+4+5+6) \cdot 3 = 6a_6 + 63$입니다.
> (또는 $a_7 + a_{12} = 2a_6 + 7 \cdot 3$ 등을 이용)
>
> 우측 합: $6 + (|a_2| + |a_4| + |a_6| + |a_8| + |a_{10}| + |a_{12}|)$
> $a_5 < 0$이므로 $a_2, a_4$는 음수입니다. $a_8, a_{10}, a_{12}$는 양수입니다. $a_6$의 부호는 아직 모릅니다.
> $\text{RHS} = 6 + (-a_2 - a_4 + |a_6| + a_8 + a_{10} + a_{12})$
> $\text{RHS} = 6 + (-(a_6-12) - (a_6-6) + |a_6| + (a_6+6) + (a_6+12) + (a_6+18))$
> $\text{RHS} = 6 + (a_6 + 54 + |a_6|)$입니다.
>
> 두 식을 같게 놓으면:
> $6a_6 + 63 = 60 + a_6 + |a_6|$
> $5a_6 + 3 = |a_6|$
>
> 1) $a_6 \ge 0$일 때: $5a_6 + 3 = a_6 \Rightarrow 4a_6 = -3 \Rightarrow a_6 = -0.75$ (모순)
> 2) $a_6 < 0$일 때: $5a_6 + 3 = -a_6 \Rightarrow 6a_6 = -3 \Rightarrow a_6 = -0.5$ (성립)
> 따라서 $a_6 = -0.5$입니다.

## [Step 3] 일반항 공식을 이용하여 특정 항의 값 산출하기

- **Trigger**: [최종 구하는 값] $a_{10}$
- **Action**: [CPT-ALG-SEQ-001] $a_{10} = a_6 + 4d$ 식에 구한 값을 대입하여 계산
- **Result**: $a_{10} = 11.5$ (또는 $\frac{23}{2}$)

> **📝 해설 (Explanation)**
> 구하고자 하는 값은 $a_{10}$입니다.
> $a_{10} = a_6 + (10-6)d = -0.5 + 4 \times 3 = -0.5 + 12 = 11.5$입니다.
> 분수 형태로 나타내면 $\frac{23}{2}$입니다.
> 정답은 **③**입니다.
