## [Step 1] 정규분포를 표준화하여 구간 확률을 표준정규분포 $Z$ 의 확률로 변환하기

- **Trigger**: [통계 조건] $X \sim N(1.5, 2^2)$ 및 $H(t) = P(t \le X \le t+1)$
- **Action**: [12확통03-04] 표준화 공식 $Z = (X-1.5)/2$ 를 사용하여 $H(0)$ 과 $H(2)$ 가 각각 어느 $Z$ 의 구간에 해당하는지 계산하기
- **Result**: $H(0) = P(-0.75 \le Z \le -0.25)$, $H(2) = P(0.25 \le Z \le 0.75)$

## [Step 2] 표준정규분포표의 대칭성을 활용하여 확률값 합산하기

- **Trigger**: [확률 계산] $H(0) + H(2)$ 산출
- **Action**: [12확통03-04] $H(0) = H(2) = P(0.25 \le Z \le 0.75)$ 인 대칭성을 인지하고, 표의 $P(0 \le Z \le 0.75) - P(0 \le Z \le 0.25)$ 연산 후 2배 하기
- **Result**: 0.3494 (최종 수술)

> **📝 해설 (Explanation)**
> $X \sim N(1.5, 2^2)$ 이므로 $Z = \frac{X-1.5}{2}$ 입니다.
> $H(0) = P(0 \le X \le 1) = P(\frac{0-1.5}{2} \le Z \le \frac{1-1.5}{2}) = P(-0.75 \le Z \le -0.25)$
> $H(2) = P(2 \le X \le 3) = P(\frac{2-1.5}{2} \le Z \le \frac{3-1.5}{2}) = P(0.25 \le Z \le 0.75)$
> 표준정규분포의 대칭성에 의해 $H(0) = H(2) = P(0 \le Z \le 0.75) - P(0 \le Z \le 0.25)$ 입니다.
> $P = 0.2734 - 0.0987 = 0.1747$
> 구하는 값은 $0.1747 \times 2 = 0.3494$ 입니다.
> 정답은 **①**입니다.
