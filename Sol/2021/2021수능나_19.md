## [Step 1] 정규분포 평균 대칭성으로 조건식을 변환하여 Y의 상방 확률 결정

- **Trigger**: [두 정규분포 변수의 확률 조건 제시] $\mathrm{P}(4 \leq X \leq 8) + \mathrm{P}(Y \geq 8) = \dfrac{1}{2}$ (X의 평균 8을 이용해 좌변 재분해)
- **Action**: [12확통03-04] $X$의 평균이 8이므로 $\mathrm{P}(X \leq 8) = \dfrac{1}{2}$. 따라서 $\mathrm{P}(4 \leq X \leq 8) = \dfrac{1}{2} - \mathrm{P}(X < 4)$로 바꾸면, $\mathrm{P}(Y \geq 8) = \mathrm{P}(X < 4)$
- **Result**: $\mathrm{P}(Y \geq 8) = \mathrm{P}(X < 4)$

> **📝 해설 (Explanation)**
> $X \sim N(8, 3^2)$이므로 $\mathrm{P}(X \leq 8) = \dfrac{1}{2}$.
> 조건식에 대입하면:
> $\left[\dfrac{1}{2} - \mathrm{P}(X < 4)\right] + \mathrm{P}(Y \geq 8) = \dfrac{1}{2}$
> $\therefore\ \mathrm{P}(Y \geq 8) = \mathrm{P}(X < 4)$

## [Step 2] 확률 등치를 표준화하여 두 정규분포 매개변수 간 관계식 추출

- **Trigger**: [이산확률변수 조건] Y의 상방 확률이 X의 좌방 확률과 같은 조건 (표준화로 z값 등치 설정)
- **Action**: [12확통03-04] 표준화: $\mathrm{P}\!\left(Z \geq \dfrac{8-m}{\sigma}\right) = \mathrm{P}\!\left(Z < \dfrac{4-8}{3}\right) = \mathrm{P}(Z < -\tfrac{4}{3})$이므로 $\dfrac{8-m}{\sigma} = \dfrac{4}{3}$
- **Result**: $8 - m = \dfrac{4\sigma}{3}$, 즉 $m = 8 - \dfrac{4\sigma}{3}$

> **📝 해설 (Explanation)**
> $\mathrm{P}(X < 4) = \mathrm{P}\!\left(Z < \dfrac{4-8}{3}\right) = \mathrm{P}\!\left(Z < -\dfrac{4}{3}\right) = \mathrm{P}\!\left(Z > \dfrac{4}{3}\right)$
>
> $\mathrm{P}(Y \geq 8) = \mathrm{P}\!\left(Z \geq \dfrac{8-m}{\sigma}\right)$이 같으려면:
> $\dfrac{8-m}{\sigma} = \dfrac{4}{3}$

## [Step 3] 구하는 확률의 표준화와 분포표 참조로 최종값 산출

- **Trigger**: [정규분포 조건] $\mathrm{P}\!\left(Y \leq 8 + \dfrac{2\sigma}{3}\right)$
- **Action**: [12확통03-04] $\dfrac{8+\frac{2\sigma}{3}-m}{\sigma} = \dfrac{8-m}{\sigma} + \dfrac{2}{3} = \dfrac{4}{3} + \dfrac{2}{3} = 2$ 이용, 표준정규분포표 참조
- **Result**: $\mathrm{P}(Z \leq 2) = 0.5 + 0.4772 = 0.9772$

> **📝 해설 (Explanation)**
> $\mathrm{P}\!\left(Y \leq 8 + \dfrac{2\sigma}{3}\right) = \mathrm{P}\!\left(Z \leq \dfrac{8 + \frac{2\sigma}{3} - m}{\sigma}\right)$
>
> $= \mathrm{P}\!\left(Z \leq \dfrac{8-m}{\sigma} + \dfrac{2}{3}\right) = \mathrm{P}\!\left(Z \leq \dfrac{4}{3} + \dfrac{2}{3}\right) = \mathrm{P}(Z \leq 2)$
>
> $= 0.5 + \mathrm{P}(0 \leq Z \leq 2) = 0.5 + 0.4772 = \mathbf{0.9772}$
> 정답은 **④**입니다.
