## [Step 1] 삼각함수 각변환 공식으로 방정식을 사인 단일 변수의 이차식으로 변환

- **Trigger**: [삼각함수 방정식 조건 제시] $4\sin^2 x - 4\cos\!\left(\frac{\pi}{2}+x\right) - 3 = 0$ (이종 삼각함수 혼재로 변환 필요)
- **Action**: [12대수02-02] 보각 공식 $\cos\!\left(\frac{\pi}{2}+x\right) = -\sin x$ 적용 후 정리: $4\sin^2 x + 4\sin x - 3 = 0$
- **Result**: $(2\sin x - 1)(2\sin x + 3) = 0$ → $\sin x = \dfrac{1}{2}$ (범위 내 유일한 해 조건)

> **📝 해설 (Explanation)**
> 보각 공식에 의해 $\cos\!\left(\dfrac{\pi}{2}+x\right) = -\sin x$이므로 주어진 방정식은
> $4\sin^2 x + 4\sin x - 3 = 0$
> $(2\sin x - 1)(2\sin x + 3) = 0$
> $\sin x = -1$ 이상이어야 하므로 $\sin x = -\dfrac{3}{2}$는 불가. 따라서 $\sin x = \dfrac{1}{2}$입니다.

## [Step 2] 사인 그래프의 주기성으로 전체 구간 내 모든 해를 열거하여 합산

- **Trigger**: [삼각함수 조건] $0 \leq x < 4\pi$에서 모든 해의 합 (주기 $2\pi$로 두 주기 적용)
- **Action**: [12대수02-02] 사인 그래프에서 $0 \leq x < 4\pi$ 구간 내 $\sin x = \frac{1}{2}$의 해 4개를 열거하여 합산
- **Result**: $\dfrac{\pi}{6} + \dfrac{5\pi}{6} + \dfrac{13\pi}{6} + \dfrac{17\pi}{6} = 6\pi$

> **📝 해설 (Explanation)**
> $\sin x = \dfrac{1}{2}$인 $x$의 해:
> - $0 \leq x < 2\pi$: $x = \dfrac{\pi}{6},\ \dfrac{5\pi}{6}$
> - $2\pi \leq x < 4\pi$: $x = 2\pi + \dfrac{\pi}{6} = \dfrac{13\pi}{6},\ 2\pi + \dfrac{5\pi}{6} = \dfrac{17\pi}{6}$
>
> 모든 해의 합 $= \dfrac{\pi + 5\pi + 13\pi + 17\pi}{6} = \dfrac{36\pi}{6} = \mathbf{6\pi}$
> 정답은 **②**입니다.
