## [Step 1] 평균값 정리를 이용하여 도함수의 부호 변화와 극값의 위치 구하기

- **Trigger**: [평균변화율의 부호 조건] 두 평균변화율의 곱이 음수인 세 실수 $x_1, x_2, x_3$ 이 열린구간 $(k, k+3/2)$ 에 존재함
- **Action**: [12미적Ⅰ-02-06] 평균값 정리에 의해 구간 내에 $f'(c_1)f'(c_2) < 0$ 을 만족하는 지점이 존재해야 함을 파악하고, 도함수의 부호 변화(극값)가 구간 내에 있음을 연역
- **Result**: $f'(x) = 3x^2 - 4ax = x(3x - 4a)$ 이므로 $x=0$ 또는 $x=4a/3$ 중 적어도 하나가 구간 $(k, k+3/2)$ 에 포함되어야 함

> **📝 해설 (Explanation)**
> 주어진 조건은 구간 $(k, k+3/2)$ 내에서 함수 $f(x)$ 의 증가와 감소가 동시에 일어남을 의미합니다.
> 평균값 정리에 의해 $\frac{f(x_1)-f(x_2)}{x_1-x_2} = f'(c_1)$ 이고 $\frac{f(x_2)-f(x_3)}{x_2-x_3} = f'(c_2)$ 인 $c_1, c_2$ 가 해당 구간에 존재하므로, $f'(c_1)f'(c_2) < 0$ 입니다.
> 다항함수의 도함수는 연속이므로, 사잇값 정리에 의해 구간 내에 $f'(x)=0$ 인 지점, 즉 함수의 극값 지점이 존재해야 합니다.
> $f'(x) = 3x^2 - 4ax = x(3x - 4a)$ 이므로 극값은 $x=0$ 과 $x=4a/3$ 에서 발생합니다.
> 그러므로 열린구간 $(k, k+3/2)$ 에 $0$ 또는 $4a/3$ 이 적어도 하나 포함되도록 하는 정수 $k$ 를 찾아야 합니다.

## [Step 2] 정수 상수의 조건과 부등식을 이용하여 함수의 식 구하기

- **Trigger**: [방정식의 해 조건] 조건을 만족하는 모든 정수 $k$ 의 곱이 $-12$
- **Action**: [10공수1-02-11] $x=0$ 에 의해 결정되는 $k=-1$ 을 제외한 나머지 $k$ 들의 곱이 $12$ 가 되는 $a$ 의 범위를 분석하여 정수 $a$ 를 확정하고 $f'(10)$ 산출
- **Result**: $a = -2$ 이므로 $f'(10) = 380$

> **📝 해설 (Explanation)**
> 먼저 극값 $x=0$ 이 구간 $(k, k+3/2)$ 에 포함될 조건은 $k < 0 < k+1.5$ 입니다.
> 이를 만족하는 정수 $k$ 는 $-1.5 < k < 0$ 에서 $k = -1$ 뿐입니다.
> 모든 $k$ 의 곱이 $-12$ 이므로, 또 다른 극값 $x = 4a/3$ 을 포함하게 하는 $k$ 들의 곱은 $12$ 가 되어야 합니다.
> $x = 4a/3$ 이 구간에 포함될 조건은 $k < 4a/3 < k+1.5$ 이며, 정리하면 $4a/3 - 1.5 < k < 4a/3$ 입니다.
> 구간의 길이가 $1.5$ 이므로 정수 $k$ 는 $1$ 개 또는 $2$ 개 존재할 수 있습니다.
> 곱이 $12$ 가 되는 정수 세트는 $\{12\}$ 또는 $\{-3, -4\}$ 입니다.
> 만약 $k=12$ 가 유일한 정수라면 $12 < 4a/3 < 13.5$ 에서 $9 < a < 10.125$ 이며, 정수 $a=10$ 일 때 $4a/3 = 13.33...$ 이 되어 구간 내 정수가 $12, 13$ 두 개가 되어 모순입니다.
> 반면 $k = -4, -3$ 인 경우를 검토하면 $-3 < 4a/3 < -2.5$ 의 공통 범위를 만족해야 합니다.
> 이를 통해 $-2.25 < a < -1.875$ 를 얻으며, 정수 $a = -2$ 가 확정됩니다.
> 실제로 $a = -2$ 이면 $4a/3 = -8/3 \approx -2.67$ 이고, $k$ 의 구간은 $(-4.17, -2.67)$ 이 되어 만족하는 정수 $k$ 는 $-4, -3$ 이 됩니다.
> 이제 $f'(x) = 3x^2 - 4(-2)x = 3x^2 + 8x$ 에 $x=10$ 을 대입하여 값을 구하면 $f'(10) = 3(10)^2 + 8(10) = 380$ 입니다.
> 따라서 정답은 **380**입니다.
