## [Step 1] 도함수를 부정적분하여 결정

- **Trigger**: [도함수의 식 제시] 도함수의 식 $f'(x) = 8x^3 - 1$ 이 주어짐
- **Action**: [CPT-CA1-INT-002] 다항함수의 부정적분 공식을 적용하여 $f(x)$ 형태 결정
- **Result**: $f(x) = 2x^4 - x + C$ (단, $C$ 는 적분 상수)

> **📝 해설 (Explanation)**
> $f'(x) = 8x^3 - 1$ 을 항별로 적분합니다.
> $f(x) = \int(8x^3 - 1)dx = 2x^4 - x + C$

## [Step 2] 초기 조건으로 결정 후 계산

- **Trigger**: [특정 x값 대입 조건] 문제 조건 $f(0) = 3$
- **Action**: [CPT-CA1-INT-002] $x=0$ 대입으로 $C$ 를 결정하고, $x=2$ 를 대입하여 $f(2)$ 계산
- **Result**: $C = 3$ 이므로 $f(x) = 2x^4 - x + 3$ 이고, $f(2) = 32 - 2 + 3 = 33$

> **📝 해설 (Explanation)**
> $f(0) = 2(0)^4 - 0 + C = C = 3$ 이므로 $C = 3$ 입니다.
>
> 그러므로 $f(x) = 2x^4 - x + 3$ 이고,
> $f(2)$ $= 2(2)^4$ $- 2$ $+ 3$ $= 2 \times 16$ $- 2$ $+ 3$ $= 32$ $- 2$ $+ 3$ $= 33$
> 따라서 정답은 **33**입니다.
