## [Step 1] 부정적분을 이용하여 도함수로부터 원함수의 일반형을 결정하기

- **Trigger**: [정적분 조건] 도함수의 식 $f'(x) = 8x^3 - 1$ 이 주어짐
- **Action**: [12미적Ⅰ-03-02] 다항함수의 부정적분 공식을 적용하여 $f(x)$ 형태 결정
- **Result**: $f(x) = 2x^4 - x + C$ (단, $C$ 는 적분 상수)

> **📝 해설 (Explanation)**
> $f'(x) = 8x^3 - 1$ 을 항별로 적분합니다.
> $f(x) = \int(8x^3 - 1)dx = 2x^4 - x + C$

## [Step 2] 초기 조건으로 적분상수를 결정하고 목표 함숫값 계산하기

- **Trigger**: [정적분 조건] 문제 조건 $f(0) = 3$
- **Action**: [12미적Ⅰ-03-02] $x=0$ 대입으로 $C$ 를 결정하고, $x=2$ 를 대입하여 $f(2)$ 계산
- **Result**: $C = 3$ 이므로 $f(x) = 2x^4 - x + 3$ 이고, $f(2) = 32 - 2 + 3 = 33$

> **📝 해설 (Explanation)**
> $f(0) = 2(0)^4 - 0 + C = C = 3$ 이므로 $C = 3$ 입니다.
>
> 그러므로 $f(x) = 2x^4 - x + 3$ 이고,
> $f(2)$ $= 2(2)^4$ $- 2$ $+ 3$ $= 2 \times 16$ $- 2$ $+ 3$ $= 32$ $- 2$ $+ 3$ $= 33$
> 따라서 정답은 **33**입니다.
