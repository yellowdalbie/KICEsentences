## [Step 1] 삼각함수의 각변환을 이용하여 부등식 구하기

- **Trigger**: [삼각함수의 각변환 조건] $\cos x \leq \sin \frac{\pi}{7}$ (사인과 코사인이 혼재된 부등식)
- **Action**: [12대수02-02] 삼각함수의 각변환 성질 $\sin \theta = \cos(\frac{\pi}{2}-\theta)$ 를 이용하여 부등식의 양변을 코사인으로 통일함
- **Result**: $\sin\frac{\pi}{7} = \cos\left(\frac{\pi}{2}-\frac{\pi}{7}\right) = \cos\frac{5\pi}{14}$ 이므로 부등식은 $\cos x \leq \cos\frac{5\pi}{14}$

> **📝 해설 (Explanation)**
> 주어진 부등식의 우변을 코사인함수로 변형하기 위해 각변환 성질을 이용합니다.
> $\sin \frac{\pi}{7} = \cos \left( \frac{\pi}{2} - \frac{\pi}{7} \right) = \cos \frac{5\pi}{14}$
> 그러므로 주어진 부등식은 $0 \leq x \leq 2\pi$ 범위에서 $\cos x \leq \cos \frac{5\pi}{14}$ 와 같습니다.

## [Step 2] 삼각함수의 그래프와 성질을 이용하여 부등식의 해 구하기

- **Trigger**: [삼각함수 주기 조건] $0 \leq x \leq 2\pi, \cos x \leq \cos\frac{5\pi}{14}$
- **Action**: [12대수02-02] 코사인함수의 그래프 개형과 대칭성을 이용하여 부등식을 만족시키는 $x$ 의 범위를 확정하고 $\beta-\alpha$ 를 구함
- **Result**: $\alpha = \frac{5\pi}{14}, \beta = 2\pi - \frac{5\pi}{14} = \frac{23\pi}{14}$ 이므로 $\beta-\alpha = \frac{18\pi}{14} = \frac{9}{7}\pi$

> **📝 해설 (Explanation)**
> 코사인함수 $y = \cos x$ 의 그래프는 $x = \pi$ 에 대하여 대칭입니다.
> $0 \leq x \leq 2\pi$ 에서 $\cos x \leq \cos \frac{5\pi}{14}$ 를 만족시키는 $x$ 의 범위는 $\frac{5\pi}{14} \leq x \leq 2\pi - \frac{5\pi}{14}$ 입니다.
> 즉, $\alpha = \frac{5\pi}{14}$ 이고 $\beta = \frac{23\pi}{14}$ 임을 알 수 있습니다.
> 그러므로 $\beta - \alpha = \frac{23\pi}{14} - \frac{5\pi}{14} = \frac{18\pi}{14} = \frac{9}{7}\pi$ 입니다.
>
> 따라서 정답은 **③**입니다.
