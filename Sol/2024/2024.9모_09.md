## [Step 1] $\cos x \leq \sin\frac{\pi}{7}$을 코사인 부등식으로 변환

- **Trigger**: [삼각함수의 각변환 조건] $\sin\frac{\pi}{7} = \cos\left(\frac{\pi}{2} - \frac{\pi}{7}\right) = \cos\frac{5\pi}{14}$ (각변환으로 동일 삼각함수로 통일)
- **Action**: [CPT-ALG-TRI-002] $\sin\frac{\pi}{7} = \cos\frac{5\pi}{14}$로 변환하여 $\cos x \leq \cos\frac{5\pi}{14}$ 형태로 재표현
- **Result**: 부등식 $\cos x \leq \cos\frac{5\pi}{14}$

> **📝 해설 (Explanation)**
> $\sin\frac{\pi}{7} = \cos\left(\frac{\pi}{2} - \frac{\pi}{7}\right) = \cos\frac{5\pi}{14}$이므로,
> 부등식 $\cos x \leq \sin\frac{\pi}{7}$은 $\cos x \leq \cos\frac{5\pi}{14}$와 동치입니다.

## [Step 2] 코사인 부등식의 해 범위 및 $\beta - \alpha$ 계산

- **Trigger**: [삼각함수 주기 조건] $0 \leq x \leq 2\pi$ 범위에서 $\cos x \leq \cos\frac{5\pi}{14}$의 해 범위
- **Action**: [CPT-ALG-TRI-002] $[0, 2\pi]$에서 $\cos x \leq c$ (단, $c = \cos\frac{5\pi}{14} > 0$)의 해 범위: $\alpha = \frac{5\pi}{14}$, $\beta = 2\pi - \frac{5\pi}{14} = \frac{23\pi}{14}$
- **Result**: $\beta - \alpha = \frac{23\pi}{14} - \frac{5\pi}{14} = \frac{18\pi}{14} = \frac{9\pi}{7}$

> **📝 해설 (Explanation)**
> $0 \leq x \leq 2\pi$에서 $\cos x \leq \cos\theta_0$ (단, $0 < \theta_0 < \pi$)의 해는 $\theta_0 \leq x \leq 2\pi - \theta_0$입니다.
> $\theta_0 = \frac{5\pi}{14}$이므로 해 범위: $\frac{5\pi}{14} \leq x \leq 2\pi - \frac{5\pi}{14} = \frac{23\pi}{14}$
> 따라서 $\alpha = \frac{5\pi}{14}$, $\beta = \frac{23\pi}{14}$이고 $\beta - \alpha = \frac{18\pi}{14} = \frac{9\pi}{7}$
> 정답은 ③번입니다.
