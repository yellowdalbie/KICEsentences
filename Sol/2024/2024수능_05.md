## [Step 1] 도함수를 부정적분하여 함수의 일반형을 결정하기

- **Trigger**: [도함수의 식 제시] 도함수의 식 $f'(x) = 3x(x-2) = 3x^2 - 6x$ 가 주어짐
- **Action**: [CPT-CA1-INT-002] $f'(x)$ 를 부정적분하여 $f(x)$ 의 일반형 결정
- **Result**: $f(x) = x^3 - 3x^2 + C$ (단, $C$ 는 적분 상수)

> **📝 해설 (Explanation)**
> $f'(x) = 3x^2 - 6x$ 를 적분합니다.
> $f(x) = x^3 - 3x^2 + C$

## [Step 2] 초기 조건으로 결정 후 계산하기

- **Trigger**: [특정 x값 대입 조건] 문제 조건 $f(1) = 6$
- **Action**: [CPT-CA1-INT-002] $x=1$ 대입으로 $C$ 를 결정하고, $x=2$ 를 대입하여 $f(2)$ 계산
- **Result**: $f(1) = 1-3+C = C-2 = 6$ 이므로 $C = 8$, $f(2) = 8-12+8 = 4$

> **📝 해설 (Explanation)**
> $f(1) = 1^3 - 3(1)^2 + C = 1 - 3 + C = C - 2 = 6$ 이므로 $C = 8$ 입니다.
> $f(x) = x^3 - 3x^2 + 8$ 이고, $f(2) = 8 - 12 + 8 = 4$
> 따라서 정답은 **④번**입니다.
