## [Step 1] 정수의 성질을 이용하여 조건을 만족시키는 사건 구하기

- **Trigger**: [정수의 성질 조건] 곱이 홀수, 합이 짝수인 자연수 순서쌍
- **Action**: [10공수1-03-01] 각 변수의 홀짝성 대응 관계를 분석하여 중복조합을 적용할 수 있는 배반사건들로 분류함
- **Result**: 두 양 끝 항은 홀수이며, 가운데 두 항은 모두 홀수이거나 모두 짝수인 경우로 나뉨

> **📝 해설 (Explanation)**
> 조건 (나)에서 $ad$ 가 홀수이므로 $a$ 와 $d$ 는 모두 홀수입니다.
> $b+c$ 가 짝수이려면 $b$ 와 $c$ 의 홀짝성이 같아야 합니다.
> 그러므로 전체 순서쌍의 경우는 다음과 같은 두 가지 배반사건으로 분류됩니다.
> 1) $a, b, c, d$ 가 모두 홀수인 경우
> 2) $a, d$ 는 홀수이고 $b, c$ 는 모두 짝수인 경우

## [Step 2] 중복조합을 이용하여 조건을 만족시키는 경우의 수 구하기

- **Trigger**: [중복조합 조건] 크기 순서가 정해진 자연수 선택 상황
- **Action**: [12확통01-02] 각 분류된 사건에 대하여 중복조합 $_nH_r$ 공식을 적용하고, 합의 법칙을 통해 전체 경우의 수를 계산함
- **Result**: $336$

> **📝 해설 (Explanation)**
> $13$ 이하의 자연수 중 홀수는 $7$개, 짝수는 $6$개입니다.
> 1) 네 수가 모두 홀수인 경우: $1 \le a \le b \le c \le d \le 13$ 인 홀수를 선택하므로 $_7H_4 = 210$ 가지입니다.
> 2) $a, d$ 는 홀수, $b, c$ 는 짝수인 경우: 홀수와 짝수는 같을 수 없으므로 $a < b \le c < d$ 가 성립합니다.
> $a = 2A-1, b = 2B, c = 2C, d = 2D-1$ 이라 하면 $2A-1 < 2B \Rightarrow A \le B$ 이고 $2C < 2D-1 \Rightarrow C \le D-1$ 입니다.
> 즉 $1 \le A \le B \le C \le D-1 \le 6$ 을 만족하는 자연수를 선택하는 것과 같으므로 $_6H_4 = 126$ 가지입니다.
> 합의 법칙에 의해 전체 경우의 수는 $210 + 126 = 336$ 입니다.
>
> 따라서 정답은 **336**입니다.
