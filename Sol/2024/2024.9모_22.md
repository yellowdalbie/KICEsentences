## [Step 1] 정적분으로 정의된 함수를 미분하여 특정 함수의 식 확정하기

- **Trigger**: [정적분으로 정의된 함수 조건] $\int_{1}^{x} f(t) d t=x f(x)-2 x^{2}-1$ (미분과 대입을 통한 다항함수 추론)
- **Action**: [12미적Ⅰ-03-04] 양변을 $x$에 대하여 미분하여 도함수를 구하고, $x=1$을 대입하여 함숫값을 구한 뒤 식을 완성
- **Result**: $f(x) = 4x - 1$

> **📝 해설 (Explanation)**
> 조건 (가)에 주어진 등식 $\int_{1}^{x} f(t) d t = x f(x) - 2x^2 - 1$의 양변을 $x$에 대하여 미분합니다.
> 좌변은 $f(x)$가 되고, 우변은 곱의 미분법에 의해 $f(x) + x f'(x) - 4x$가 됩니다.
> 즉, $f(x) = f(x) + x f'(x) - 4x$에서 $x f'(x) = 4x$를 얻습니다.
> 이 식이 모든 실수 $x$에 대하여 성립하므로 $f'(x) = 4$입니다.
> 따라서 함수 $f(x)$는 일차함수이며 $f(x) = 4x + C$로 둘 수 있습니다.
> 등식의 양변에 $x=1$을 대입하면 $\int_{1}^{1} f(t) d t = 0 = f(1) - 2 - 1$이므로 $f(1) = 3$입니다.
> $f(1) = 4 + C = 3$에서 $C = -1$이므로, $f(x) = 4x - 1$로 확정됩니다.

## [Step 2] 곱의 미분법 역연산을 이용하여 두 부정적분의 곱의 형태 파악하기

- **Trigger**: [미분 조건] $f(x) G(x)+F(x) g(x)=8 x^{3}+3 x^{2}+1$ (두 함수의 곱의 미분 형태 인식)
- **Action**: [12미적Ⅰ-02-04] 좌변이 $\{F(x)G(x)\}'$임을 파악하고 양변을 부정적분하여 $F(x)G(x)$에 대한 다항식 수립
- **Result**: $F(x)G(x) = 2x^4 + x^3 + x + C_2$ (단, $C_2$는 적분상수)

> **📝 해설 (Explanation)**
> $F(x)$와 $G(x)$는 각각 $f(x)$와 $g(x)$의 부정적분이므로 $F'(x) = f(x)$, $G'(x) = g(x)$입니다.
> 조건 (나)의 좌변 $f(x)G(x) + F(x)g(x)$는 곱의 미분법에 의해 $\{F(x)G(x)\}'$과 같습니다.
> 따라서 $\{F(x)G(x)\}' = 8x^3 + 3x^2 + 1$이 성립합니다.
> 양변을 $x$에 대하여 부정적분하면 $F(x)G(x) = 2x^4 + x^3 + x + C_2$ ($C_2$는 적분상수)라는 식을 얻을 수 있습니다.

## [Step 3] 항등식의 계수비교를 통하여 각 함수의 부정적분 식 확정하기

- **Trigger**: [항등식 조건] $F(x)$와 $G(x)$의 곱이 사차식임과 $f(x) = 4x-1$ 조건
- **Action**: [10공수1-01-02] $F(x) = 2x^2-x+C_1$, $G(x) = x^2+ax+b$로 두고 곱셈 전개 후 우변과 계수비교
- **Result**: $F(x) = 2x^2 - x + 1$, $G(x) = x^2 + x$

> **📝 해설 (Explanation)**
> 앞서 $f(x) = 4x - 1$을 구했으므로, 이 함수의 부정적분인 $F(x)$는 $F(x) = 2x^2 - x + C_1$ ($C_1$은 적분상수) 꼴입니다.
> $F(x)$가 이차식이므로 두 다항식의 곱 $F(x)G(x)$가 사차식이 되려면 $G(x)$ 역시 이차식이어야 합니다. 최고차항의 계수를 맞추기 위해 $G(x) = x^2 + ax + b$로 둡니다.
> 두 식을 곱하면 $(2x^2 - x + C_1)(x^2 + ax + b) = 2x^4 + (2a-1)x^3 + (2b-a+C_1)x^2 + (-b+C_1a)x + C_1b$입니다.
> 이 식이 $2x^4 + x^3 + 0x^2 + x + C_2$와 같아야 하므로 각 차수의 계수를 비교합니다.
> 삼차항 계수: $2a - 1 = 1$에서 $a = 1$입니다.
> 이차항 계수: $2b - a + C_1 = 0$에서 $2b - 1 + C_1 = 0$, 즉 $C_1 = 1 - 2b$입니다.
> 일차항 계수: $-b + C_1a = 1$에서 $-b + C_1 = 1$입니다.
> 위 두 식을 연립하여 대입하면 $-b + (1 - 2b) = 1$에서 $-3b = 0$이므로 $b = 0$입니다.
> $b = 0$이므로 $C_1 = 1$이 되며, $F(x) = 2x^2 - x + 1$, $G(x) = x^2 + x$로 완전히 확정됩니다.

## [Step 4] 정적분과 부정적분의 관계를 이용하여 최종 정적분 값 산출하기

- **Trigger**: [정적분 조건] $\int_{1}^{3} g(x) d x$
- **Action**: [12미적Ⅰ-03-04] $G(x)$가 $g(x)$의 부정적분임을 이용하여 미적분학의 기본정리인 $G(3) - G(1)$ 연산
- **Result**: $10$

> **📝 해설 (Explanation)**
> 구하고자 하는 값은 정적분 $\int_{1}^{3} g(x) d x$입니다.
> $G(x)$가 $g(x)$의 한 부정적분이므로, 정적분의 정의에 따라 이 값은 $G(3) - G(1)$과 같습니다.
> 앞서 구한 $G(x) = x^2 + x$ 식에 $x=3$과 $x=1$을 각각 대입합니다.
> $G(3) = 3^2 + 3 = 12$
> $G(1) = 1^2 + 1 = 2$
> 따라서 $G(3) - G(1) = 12 - 2 = 10$입니다.
> 정답은 **10**입니다.
