# 2018.9모가_12.md 해설
>
> 원본 문항 링크: [2018.9모가_12.md](../../MD_Ref/2018/2018.9모가_12.md)

## [Step 1] 정규분포의 대칭성을 활용하여 확률의 영역 차이를 표준화 변수로 정규화하기

- **Trigger**: [확률변수 조건] $P(m \leq X \leq m+12) - P(X \leq m-12) = 0.3664$ 제시
- **Action**: [12확통03-04] $P(X \leq m-12) = 0.5 - P(m \leq X \leq m+12)$ 등의 대칭성을 활용하여 주어진 식을 $2P(0 \leq Z \leq 12/\sigma) - 0.5 = 0.3664$ 형태로 변환하고 표준정규분포표 수치 도출
- **Result**: $P(0 \leq Z \leq 12/\sigma) = 0.4332$ 확정

## [Step 2] 표준정규분포표의 지표와 변수를 대조하여 미지 표준편차 산출하기

- **Trigger**: [확률변수 조건] $\sigma$의 값 요구
- **Action**: [12확통03-04] 표에서 $0.4332$에 대응하는 $z=1.5$임을 확인하고 $12/\sigma = 1.5$ 방정식을 수립하여 정량 연산
- **Result**: 최종 결과값 8 확정

> **📝 해설 (Explanation)**
> 종 모양의 분포(정규분포)가 가진 좌우 대칭 특징을 이용해 흩어짐의 정도(표준편차)를 알아내는 문제입니다.
> 복잡하게 적힌 확률 수치를 표의 형식에 맞게 하나로 정리한 뒤, 약속된 표준 자투리(Z-값)와 실제 값의 간격을 비교하여 정답을 확정합니다.
> 정답은 **③**입니다.
