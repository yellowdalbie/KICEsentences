## [Step 1] 점화식에 항을 순차 대입하여 수열의 반복 주기와 규칙을 확인하기

- **Trigger**: [수열의 귀납적 정의] 두 수열 $a_n, b_n$ 의 초기항과 점화식 $a_{n+2} = a_{n+1}^2 - a_n^2$, $b_{n+1} = a_n - b_n + n$ 제시
- **Action**: [CPT-ALG-SEQ-006] 점화식 $a_{n+2} = a_{n+1}^2 - a_n^2$ 에 항을 차례로 대입하여 $a_n$ 항이 $1, 1, 0, -1, 1, 0, -1 \dots$ 주기 $3$ 의 패턴을 나타냄을 파악하고, $b_{n}$ 식을 $b_{n+1} + b_n = a_n + n$ 형태로 정리하여 활용 기반 마련
- **Result**: 수열 $a_n$ 은 $n \geq 2$ 일 때 $\{1, 0, -1\}$ 주기로 반복됨

> **📝 해설 (Explanation)**
> 두 수열 $a_n, b_n$ 이 귀납적으로 정의되어 있습니다.
> 수열 $a_n$ 의 규칙을 파악하기 위해 점화식 $a_{n+2} = a_{n+1}^2 - a_n^2$ 에 순차적으로 대입합니다.
> $a_1 = 1, a_2 = 1$
> $a_3 = 1^2 - 1^2 = 0$
> $a_4 = 0^2 - 1^2 = -1$
> $a_5 = (-1)^2 - 0^2 = 1$
> $a_6 = 1^2 - (-1)^2 = 0$
> $a_7 = 0^2 - 1^2 = -1$
> 이로부터 수열 $a_n$ 의 제2항부터 값 $1, 0, -1$ 이 반복되는 주기성을 갖는 것을 확인할 수 있습니다.
> 이 규칙성을 이용하여 수열 $b_n$ 에 대한 연산을 수행합니다.

## [Step 2] 합의 교차 상쇄를 이용하여 도출하기

- **Trigger**: [특수항 도출 요구] 조건 $b_{20} = 14$ 일 때, 초기항 $b_1 = k$ 의 값 식별
- **Action**: [CPT-ALG-SEQ-006] 정리된 점화식 $b_{n+1} + b_n = a_n + n$ 에 $n$ 값을 차례로 대입하고 교대부호합 연산을 적용하여 $b_n$ 을 소거시키고, $b_{20}$ 과 $b_1$ 만 남는 식 도출 및 $a_n$ 항 합산
- **Result**: $b_{20} = -k + 11$ 산출, 그러므로 상수 $k = -3$

> **📝 해설 (Explanation)**
> 수열 $b_n$ 에 관한 점화식을 $b_{n+1} + b_n = a_n + n$ 의 형태로 정리합니다.
> 이웃한 두 항의 합 형태이므로, 좌변의 특정 항들을 소거하기 위해 $n=1$ 부터 차례로 대입한 후 교대부호를 적용해 합산합니다.
> $b_2 + b_1 = a_1 + 1$
> $b_3 + b_2 = a_2 + 2$
> $\dots$
> 이 연산을 통해 중간 항들이 교차 소거되고 양 끝 항인 $b_{20}$ 과 $b_1$ 에 대한 등식을 얻을 수 있으며, 우변은 확인된 $a_n$ 항 변동 값을 대입하여 합산합니다.
> 계산을 수행하면 등식 $b_{20} = -b_1 + 11$ 로 정리됩니다.
>
> 조건에서 $b_{20} = 14$ 로 주어졌고 초기항 $b_1 = k$ 이므로 다음 방정식을 세울 수 있습니다.
> $14 = -k + 11$
> 이를 계산하면 $k = -3$ 이 됩니다.
>
> 따라서 정답은 **①**입니다.
