## [Step 1] 도함수의 부호 분석을 통하여 닫힌구간에서 다항함수의 최댓값과 최솟값 구하기

- **Trigger**: [닫힌 구간 부등식 항상 성립 조건] $-2 \le x \le 2$ 에서 부등식 $-k \le f(x) \le k$ 가 성립하기 위한 $f(x) = 2x^3+3x^2-12x-8$ 의 치역 파악
- **Action**: [12미적Ⅰ-02-08] 도함수 $f'(x) = 6(x+2)(x-1)$ 의 영점과 구간의 끝점을 대입하여 함숫값들을 비교하고 최댓값과 최솟값 결정하기
- **Result**: 최댓값 $12$, 최솟값 $-15$

> **📝 해설 (Explanation)**
> 주어진 부등식이 항상 성립하려면 함수 $f(x) = 2x^3+3x^2-12x-8$ 의 최댓값과 최솟값을 알아야 합니다.
> 먼저 극점의 위치를 찾기 위해 함수를 미분합니다.
> $f'(x) = 6x^2 + 6x - 12 = 6(x^2 + x - 2) = 6(x+2)(x-1)$
> 도함수가 $0$ 이 되는 점은 $x = -2$ 와 $x = 1$ 입니다.
> 이제 닫힌구간 $[-2, 2]$ 에서 극점과 구간의 양 끝점의 함숫값을 각각 구하여 비교합니다.
> 1. $x=-2$ 일 때(극대이자 왼쪽 끝점): $f(-2) = 2(-8) + 3(4) - 12(-2) - 8 = 12$
> 2. $x=1$ 일 때(극소): $f(1) = 2(1) + 3(1) - 12(1) - 8 = -15$
> 3. $x=2$ 일 때(오른쪽 끝점): $f(2) = 2(8) + 3(4) - 12(2) - 8 = -4$
> 세 값을 비교하면 주어진 구간에서 함수의 최댓값은 $12$, 최솟값은 $-15$ 임을 알 수 있습니다.

## [Step 2] 함수의 치역을 활용하여 부등식을 만족시키는 상수의 최솟값 구하기

- **Trigger**: [방정식과 부등식에의 활용 조건] $-15 \le f(x) \le 12$ 일 때 $-k \le f(x) \le k$ 가 성립할 조건
- **Action**: [12미적Ⅰ-02-09] 절댓값 부등식 $|f(x)| \le k$ 의 성질을 이용하여 $k$ 가 만족해야 하는 최소 범위를 분석하여 값 구하기
- **Result**: $15$

> **📝 해설 (Explanation)**
> 앞선 단계에서 함수의 치역이 $-15 \le f(x) \le 12$ 임을 확인하였습니다.
> 문제의 부등식 $-k \le f(x) \le k$ 가 모든 $x$ 에 대해 성립하려면, 함수의 최솟값보다 $-k$ 가 작거나 같아야 하고 최댓값보다 $k$ 가 크거나 같아야 합니다.
> 즉, $-k \le -15$ 이고 $12 \le k$ 여야 합니다.
> 이를 정리하면 $k \ge 15$ 이고 $k \ge 12$ 이므로, 두 조건을 동시에 만족하는 양수 $k$ 의 범위는 $k \ge 15$ 입니다.
> 그러므로 $k$ 의 최솟값은 $15$ 입니다.
>
> 따라서 정답은 **15**입니다.
