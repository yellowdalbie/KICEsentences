## [Step 1] 다항함수의 부정적분과 적분상수의 결정을 활용하여 함수의 함숫값 구하기

- **Trigger**: [부정적분 조건] 도함수 $f'(x) = 3x^2+2x+1$ 와 초기 조건 $f(1)=6$ 제시 (부정적분과 적분상수 특정 필요)
- **Action**: [12미적Ⅰ-03-02] 다항함수의 부정적분을 계산하여 원함수를 구하고, 주어진 함숫값 조건을 대입하여 적분상수를 결정한 뒤 $f(2)$ 의 값 구하기
- **Result**: $17$

> **📝 해설 (Explanation)**
> 도함수 $f'(x) = 3x^2 + 2x + 1$ 의 부정적분 $f(x)$ 는 다음과 같습니다.
> $f(x) = \int (3x^2 + 2x + 1) dx = x^3 + x^2 + x + C$ (단, $C$ 는 적분상수)
> 조건 $f(1) = 6$ 에서 $1^3 + 1^2 + 1 + C = 3 + C = 6$ 이므로 적분상수 $C = 3$ 입니다.
> 따라서 완성된 다항함수는 $f(x) = x^3 + x^2 + x + 3$ 이며, $x=2$ 를 대입하여 함숫값을 구하면 다음과 같습니다.
> $f(2) = 2^3 + 2^2 + 2 + 3 = 8 + 4 + 2 + 3 = 17$
>
> 따라서 정답은 **17**입니다.
