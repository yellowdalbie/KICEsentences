## [Step 1] 부정적분의 뜻과 성질을 이용하여 함숫값 구하기

- **Trigger**: [부정적분의 뜻과 성질 조건]
- **Action**: [12미적Ⅰ-03-02] $f'(x) = 3x^2+2x+1$ 의 부정적분을 구하고 $f(1)=6$ 을 활용하여 적분상수를 정한 후 $f(2)$ 의 값 구하기
- **Result**: $17$

> **📝 해설 (Explanation)**
> 도함수 $f'(x) = 3x^2 + 2x + 1$ 의 부정적분 $f(x)$ 는 다음과 같습니다.
> $f(x) = \int (3x^2 + 2x + 1) dx = x^3 + x^2 + x + C$ (단, $C$ 는 적분상수)
> 조건 $f(1) = 6$ 에서 $1^3 + 1^2 + 1 + C = 3 + C = 6$ 이므로 $C = 3$ 입니다.
> 그러므로 $f(x) = x^3 + x^2 + x + 3$ 이며, $x=2$ 를 대입하여 함숫값을 구하면 다음과 같습니다.
> $f(2) = 2^3 + 2^2 + 2 + 3 = 8 + 4 + 2 + 3 = 17$
>
> 따라서 정답은 **17**입니다.
