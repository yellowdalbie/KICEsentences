## [Step 1] 도함수를 적분하여 원함수의 부정적분 구하기

- **Trigger**: 도함수의 식 $f'(x)=3x^2+2x+1$ 이 주어짐
- **Action**: [CPT-CA1-INT-002] 다항함수의 부정적분 계산
- **Result**: $f(x) = x^3 + x^2 + x + C$

> **📝 해설 (Explanation)**
> 함수 $f(x)$ 를 구하기 위해 도함수 $f'(x)$ 의 부정적분을 구합니다.
> $f(x) = \int f'(x) dx = \int (3x^2 + 2x + 1) dx$
> 다항함수의 적분법을 사용하면 $f(x) = x^3 + x^2 + x + C$ (단, $C$ 는 적분상수) 가 됩니다.

## [Step 2] 초기 조건을 이용한 적분상수 결정

- **Trigger**: [Step 1의 Result: $f(x)$ 식] + [문제 조건: $f(1)=6$]
- **Action**: [CPT-CA1-INT-001] 부정적분의 뜻 (초기 조건을 대입하여 적분상수 도출)
- **Result**: $C = 3$, $f(x) = x^3 + x^2 + x + 3$

> **📝 해설 (Explanation)**
> 문제에서 주어진 초기 조건 $f(1) = 6$ 을 방금 구한 $f(x)$ 식에 대입하여 적분상수 $C$ 의 값을 찾습니다.
> $f(1) = 1^3 + 1^2 + 1 + C = 3 + C$
> 이 값이 $6$ 이어야 하므로, $3 + C = 6 \implies C = 3$ 임을 알 수 있습니다.
>
> 그러므로 함수는 $f(x) = x^3 + x^2 + x + 3$ 으로 완전히 결정됩니다.

## [Step 3] 특정 값 대입을 통한 최종 함숫값 계산

- **Trigger**: [Step 2의 Result: 결정된 $f(x)$ 식] + [문제 조건: 구하고자 하는 값 $f(2)$]
- **Action**: [CPT-CM1-POL-001] 식의 값 구하기 (다항식에 특정 $x$ 값 대입)
- **Result**: $17$

> **📝 해설 (Explanation)**
> 완성된 함수식에 $x=2$ 를 대입하여 문제에서 구하고자 하는 값인 $f(2)$ 를 계산합니다.
> $f(2) = 2^3 + 2^2 + 2 + 3 = 8 + 4 + 2 + 3 = 17$
>
> 따라서 정답은 **$17$**입니다.
