## [Step 1] 삼각함수의 정의를 이용하여 각 변의 길이를 변수로 표현하기

- **Trigger**: [삼각형 넓이 조건] $\overline{\mathrm{AB}}:\overline{\mathrm{AC}}=\sqrt{2}:1, \overline{\mathrm{AH}}=2, \angle A > \pi/2$
- **Action**: [12대수02-03] $\angle \mathrm{ABC} = \beta, \angle \mathrm{ACB} = \gamma$라고 하면, 직각삼각형 ABH와 ACH에서 $\sin \beta = 2/c, \sin \gamma = 2/b$임을 이용
- **Result**: $c = \sqrt{2}b$ 이므로 $\sin \beta : \sin \gamma = 1 : \sqrt{2}$

> **📝 해설 (Explanation)**
> 삼각형 $\mathrm{ABC}$에서 꼭짓점 $\mathrm{A}$에서 $\mathrm{BC}$에 내린 수선의 발을 $\mathrm{H}$라 합시다.
> 직각삼각형 $\mathrm{ABH}$에서 $\overline{\mathrm{AB}} = c$, 직각삼각형 $\mathrm{ACH}$에서 $\overline{\mathrm{AC}} = b$라고 하면 $c : b = \sqrt{2} : 1$입니다.
> $\overline{\mathrm{AH}} = 2$이므로 $\sin B = \frac{2}{c}, \sin C = \frac{2}{b}$입니다.
> $\frac{\sin B}{\sin C} = \frac{2/c}{2/b} = \frac{b}{c} = \frac{1}{\sqrt{2}}$이므로 $\sin C = \sqrt{2} \sin B$입니다.

## [Step 2] 사인법칙과 외접원 반지름을 이용하여 변의 길이 산출하기

- **Trigger**: [지수함수 조건] 외접원 넓이가 $50\pi$ ($R=5\sqrt{2}$)
- **Action**: [12대수02-03] 사인법칙 $b = 2R \sin B, c = 2R \sin C$를 이용하여 구체적인 변의 길이 산출
- **Result**: $b = 2\sqrt{5}, c = 2\sqrt{10}$ (아님 $b=2\sqrt{5}$면 $\sin C$ 등이 어긋날 수 있음. 다시 계산) $\rightarrow$ $b=2\sqrt{5}$ 이고 $\sin B = 2/c = 2/(b\sqrt{2})$ 이용

> **📝 해설 (Explanation)**
> 외접원의 반지름 $R = \sqrt{50} = 5\sqrt{2}$입니다.
> 사인법칙에 의해 $c = 2R \sin C = 10\sqrt{2} \sin C$입니다.
> 앞서 $c = 2/\sin B$이므로 $2/\sin B = 10\sqrt{2} \sin C$입니다.
> $\frac{1}{5\sqrt{2}} = \sin B \sin C = \sin B (\sqrt{2} \sin B) = \sqrt{2} \sin^2 B$
> $\sin^2 B = \frac{1}{5\sqrt{2} \cdot \sqrt{2}} = \frac{1}{10}$입니다.
> $B$는 삼각형의 내각이므로 $\sin B = \frac{1}{\sqrt{10}}$입니다.
> 따라서 $c = \frac{2}{1/\sqrt{10}} = 2\sqrt{10}$이고, $b = \frac{c}{\sqrt{2}} = \frac{2\sqrt{10}}{\sqrt{2}} = 2\sqrt{5}$입니다.

## [Step 3] 피타고라스 정리를 이용하여 목적 선분의 길이 산출하기

- **Trigger**: [최종 구하는 값] 선분 BH의 길이
- **Action**: [10공수2-01-01] 직각삼각형 ABH 또는 ACH에서 피타고라스 정리를 적용하여 변의 길이 연산
- **Result**: $\overline{\mathrm{BH}} = 6$

> **📝 해설 (Explanation)**
> 직각삼각형 $\mathrm{ABH}$에서 피타고라스 정리를 적용합니다.
> $\overline{\mathrm{AB}}^2 = \overline{\mathrm{AH}}^2 + \overline{\mathrm{BH}}^2$
> $(2\sqrt{10})^2 = 2^2 + \overline{\mathrm{BH}}^2$
> $40 = 4 + \overline{\mathrm{BH}}^2 \Rightarrow \overline{\mathrm{BH}}^2 = 36$
> 따라서 $\overline{\mathrm{BH}} = 6$입니다.
> 참고로 $\overline{\mathrm{CH}} = \sqrt{b^2 - 4} = \sqrt{20-4} = 4$입니다.
> $\angle A > \pi/2$ 조건에 따라 $H$가 $BC$ 사이에 있는지 외부에 있는지 확인해야 하나, 수치상 $H$가 내부에 있는 둔각삼각형도 가능합니다.
> 정답은 **①**입니다.
