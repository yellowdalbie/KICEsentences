## [Step 1] 다항함수의 식의 전개와 부등식의 성질을 활용하여 다항함수 사이의 관계 도출하기

- **Trigger**: [다항함수 조건] 최고차항의 계수가 $1$ 인 삼차함수 $f(x)$ 가 모든 정수 $k$ 에 대하여 $2 k-8 \leq \frac{f(k+2)-f(k)}{2} \leq 4 k^{2}+14 k$
- **Action**: [12미적Ⅰ-02-04] 다항함수의 미분법 (삼차함수의 식을 전개하여 모든 정수 $k$ 에 대해 성립하는 두 이차부등식을 유도)
- **Result**: $P(k) = 3k^2 + (2a+4)k + 2a+b+12 \ge 0$, $Q(k) = k^2 + (8-2a)k - 2a-b-4 \ge 0$

> **📝 해설 (Explanation)**
> 최고차항의 계수가 $1$ 인 삼차함수 $f(x) = x^3 + ax^2 + bx + c$ 로 두면, 부등식의 가운데 항인 평균변화율 형태의 식은 다음과 같이 정리됩니다.
> $\frac{f(k+2)-f(k)}{2} = \frac{\{ (k+2)^3 + a(k+2)^2 + b(k+2) + c \} - \{ k^3 + ak^2 + bk + c \}}{2}$
> $= \frac{6k^2 + (4a+12)k + 4a + 2b + 8}{2} = 3k^2 + (2a+6)k + 2a + b + 4$
> 주어진 연립부등식을 두 부분으로 나누어 정리하면 모든 정수 $k$ 에 대하여 다음 두 식 $P(k), Q(k)$ 가 항상 $0$ 이상이어야 함을 알 수 있습니다.
> 1) $2k-8 \le 3k^2 + (2a+6)k + 2a + b + 4 \implies 3k^2 + (2a+4)k + 2a + b + 12 \ge 0 \cdots P(k)$
> 2) $3k^2 + (2a+6)k + 2a + b + 4 \le 4k^2 + 14k \implies k^2 + (8-2a)k - 2a - b - 4 \ge 0 \cdots Q(k)$

## [Step 2] 연립이차부등식의 합의 성질을 활용하여 조건을 만족시키는 값 구하기

- **Trigger**: [추가 관계식 조건] 모든 정수 $k$ 에 대하여 $P(k) \ge 0, Q(k) \ge 0$ 임
- **Action**: [12미적Ⅰ-02-09] 방정식과 부등식에의 활용 (두 부등식의 합이 $0$ 이 되는 정수 지점에서 미정계수를 결정하고 최종 미분계수 산출)
- **Result**: $31$

> **📝 해설 (Explanation)**
> 모든 정수 $k$ 에 대하여 $P(k) \ge 0$ 이고 $Q(k) \ge 0$ 이므로, 두 식의 합 $P(k) + Q(k)$ 역시 항상 $0$ 이상이어야 합니다.
> $P(k) + Q(k) = 4k^2 + 12k + 8 = 4(k+1)(k+2) \ge 0$
> 이 식의 좌변은 정수 $k=-1$ 또는 $k=-2$ 일 때 정확히 $0$ 이 됩니다.
> 각각 $0$ 이상인 두 값 $P(k)$ 와 $Q(k)$ 를 더해 $0$ 이 되려면, $k=-1, -2$ 에서 $P(k)=0$ 이고 $Q(k)=0$ 이어야 합니다.
> $P(-1) = 3 - (2a+4) + 2a + b + 12 = b + 11 = 0$ 이므로 $b = -11$ 입니다.
> $P(-2) = 12 - 2(2a+4) + 2a + b + 12 = -2a + b + 16 = 0$ 이므로 $-2a + 5 = 0$, 즉 $a = \frac{5}{2}$ 입니다.
> 이를 통해 구하는 함수는 $f(x) = x^3 + \frac{5}{2}x^2 - 11x + c$ 이며, 미분하면 $f^{\prime}(x) = 3x^2 + 5x - 11$ 입니다.
> 결과적으로 $f^{\prime}(3) = 3(9) + 5(3) - 11 = 27 + 15 - 11 = 31$ 임을 알 수 있습니다.
>
> 따라서 정답은 **31**입니다.
