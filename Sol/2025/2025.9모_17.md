## [Step 1] 부정적분을 이용하여 도함수로부터 원함수 형태를 도출하기

- **Trigger**: [도함수의 식] $f'(x)=6x^2+2x+1$
- **Action**: [12미적Ⅰ-03-02] 다항함수의 부정적분 (적분 수행)
- **Result**: $f(x) = 2x^3 + x^2 + x + C$

> **📝 해설 (Explanation)**
> 주어진 도함수 $f'(x)$ 를 부정적분하여 원래 함수 $f(x)$ 의 식을 $C$(적분상수)를 포함하여 구합니다.
> $f(x) = \int (6x^2+2x+1) dx = 2x^3 + x^2 + x + C$ 입니다.

## [Step 2] 초기 조건을 이용하여 적분 상수 도출하기

- **Trigger**: [Step 1의 Result: 적분된 함수식] $f(x) = 2x^3 + x^2 + x + C$ + [초기 조건] $f(0)=1$
- **Action**: [12미적Ⅰ-03-02] 다항함수의 부정적분 (초기조건 대입하여 상수 도출). $x=1$ 대입: $f(1)=2+1+1+1=5$
- **Result**: $f(x) = 2x^3 + x^2 + x + 1$, $f(1)=5$

> **📝 해설 (Explanation)**
> 문제에서 $f(0)=1$ 이라고 주어졌으므로, 앞서 구한 식에 $x=0$ 을 대입합니다.
> $f(0) = 2(0)^3 + 0^2 + 0 + C = C$ 이고 이 값이 $1$ 이므로 $C=1$ 입니다.
>
> $f(x) = 2x^3 + x^2 + x + 1$ 에 $x=1$ 을 대입하면 $f(1) = 2+1+1+1 = 5$ 입니다.
>
> 따라서 정답은 **$5$**입니다.
