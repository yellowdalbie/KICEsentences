## [Step 1] 미분가능성 조건으로 의 차 상수항 계수 결정

- **Trigger**: [도함수의 식 제시] 함수 $g(x)$ 는 실수 전체의 집합에서 미분가능 ($x=0$ 을 기준으로 나뉜 함수)
- **Action**: [CPT-CA1-DIF-002] $x=0$ 에서 연속 및 좌우 미분계수가 일치함을 이용하여 $f(x)$ 관계식 수립
- **Result**: $f(0)=7$, $f'(0)=15$ ($f'(x) = 2px+15$ 형태로 도출)

> **📝 해설 (Explanation)**
> 함수 $g(x)$ 가 실수 전체의 집합에서 미분가능하므로, 경계인 $x=0$ 에서 연속이고 미분가능해야 합니다.
> $x \le 0$ 일 때 $g(x) = x^3 + ax^2 + 15x + 7$ 이므로 $\lim_{x \to 0^-} g(x) = 7$ 이고, 도함수 $g'(x) = 3x^2 + 2ax + 15$ 에서 $\lim_{x \to 0^-} g'(x) = 15$ 입니다.
> $x>0$ 에서 $g(x)=f(x)$ 이고, $f(x)$ 는 최고차항의 계수가 음수인 이차함수이므로 $f(x) = px^2 + qx + r$ ($p<0$)로 둘 수 있습니다.
> $x=0$ 에서의 연속 조건에 의해 $f(0) = r = 7$ 이고, 미분가능성 조건에 의해 $f'(0) = q = 15$ 입니다.
> 그러므로 $x>0$ 일 때 $g'(x) = f'(x) = 2px + 15$ 입니다.

## [Step 2] 실근 개수 조건과 평행이동으로 도함수의 근 확정

- **Trigger**: [방정식의 해 조건] 방정식 $g'(x) \times g'(x-4)=0$ 의 서로 다른 실근의 개수가 4 (집합의 합집합 원소 개수로 근의 간격 파악)
- **Action**: [CPT-CM1-EQU-003] $g'(x)=0$ 의 근이 등차수열을 이룸을 파악하고 이차방정식의 근과 계수의 관계 적용
- **Result**: $a=9$, $p=-\dfrac{5}{2}$ ($g(x)$ 함수식 완전 확정)

> **📝 해설 (Explanation)**
> $g'(x)=0$ 의 서로 다른 실근 집합을 $S$ 라 하면, $g'(x-4)=0$ 의 실근 집합은 $S$ 의 모든 원소를 $x$ 축 방향으로 $4$ 만큼 평행이동한 $S+4$ 입니다.
> 방정식 $g'(x)g'(x-4)=0$ 의 실근 집합은 $S \cup (S+4)$ 이며, 조건 (나)에 의해 이 집합의 원소 개수가 4개입니다.
> $g'(x)=0$ 을 만족하는 $x$ 를 살펴보면, $x>0$ 에서는 $2px+15=0$ 에서 $x = -\frac{15}{2p}$ 입니다. $p<0$ 이므로 양의 실근 1개를 가집니다. 이를 $\alpha$ 라 합시다.
> $x \le 0$ 에서는 $3x^2+2ax+15=0$ 입니다. 이 이차방정식의 두 근의 곱은 $\frac{15}{3}=5>0$ 이므로 두 실근을 갖는다면 부호가 같습니다. $\alpha>0$ 하나만으로는 집합 원소 4개를 만들 수 없으므로, 음의 실근 2개를 가져야 합니다. 이를 $\beta_1, \beta_2$ ($\beta_1 < \beta_2 < 0$)라 하면 $S = \{\beta_1, \beta_2, \alpha\}$ 입니다.
> 원소가 3개인 두 집합 $S$ 와 $S+4$ 의 합집합 원소가 4개가 되려면 교집합의 원소가 2개여야 합니다. 즉 평행이동 구조상 $\beta_1+4 = \beta_2$ 이고 $\beta_2+4 = \alpha$ 여야 성립합니다.
> $\beta_2 - \beta_1 = 4$ 이므로 $3x^2+2ax+15=0$ 에서 이차방정식의 두 근의 차 공식을 이용하면 $\frac{\sqrt{4a^2-180}}{3} = 4$ 에서 $a^2-45 = 36$, 즉 $a^2=81$ 입니다. 두 근이 음수이려면 두 근의 합 $-\frac{2a}{3} < 0$ 이어야 하므로 $a=9$ 입니다.
> $a=9$ 일 때 $3x^2+18x+15=0$ 을 풀면 $x^2+6x+5=0$ 이므로 두 근은 $\beta_1=-5, \beta_2=-1$ 입니다.
> 그러므로 $\alpha = \beta_2+4 = 3$ 이며, 앞서 구한 $-\frac{15}{2p} = 3$ 에 대입하면 $p=-\frac{5}{2}$ 로 확정됩니다.

## [Step 3] 구역별 함수식에 대입하여 최종값 산출

- **Trigger**: [최종 구하는 값] $g(-2)+g(2)$
- **Action**: [CPT-CM2-FUN-001] 완성된 $g(x)$ 의 각 범위에 해당하는 식에 $x=-2, x=2$ 대입
- **Result**: $32$

> **📝 해설 (Explanation)**
> 앞선 과정을 통해 함수 $g(x)$ 의 식을 완전히 구했습니다.
> $x \le 0$ 일 때 $g(x) = x^3 + 9x^2 + 15x + 7$ 이고,
> $x>0$ 일 때 $g(x) = -\frac{5}{2}x^2 + 15x + 7$ 입니다.
> $x=-2$ 는 $x \le 0$ 범위에 속하므로 $g(-2) = (-2)^3 + 9(-2)^2 + 15(-2) + 7 = -8 + 36 - 30 + 7 = 5$ 입니다.
> $x=2$ 는 $x>0$ 범위에 속하므로 $g(2) = -\frac{5}{2}(2)^2 + 15(2) + 7 = -10 + 30 + 7 = 27$ 입니다.
> 그러므로 $g(-2)+g(2) = 5 + 27 = 32$ 이며입니다.
> 따라서 정답은 **②**입니다.
