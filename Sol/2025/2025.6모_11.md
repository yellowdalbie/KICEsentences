## [Step 1] 미분계수의 정의와 접선의 방정식을 활용하여 함수의 함숫값과 미분계수 구하기

- **Trigger**: [함수의 극한 조건] $\lim _{x \rightarrow a} \frac{f(x)-1}{x-a}=3$, 점 $(a, f(a))$ 에서의 접선의 $y$ 절편이 $4$ 제시 (미분계수 정의와 접선의 성질 활용 필요)
- **Action**: [12미적Ⅰ-02-05] 극한의 수렴 조건으로부터 도출한 $f(a), f'(a)$ 를 접선의 방정식에 적용하고, $y$ 절편 조건을 활용하여 상수 $a$ 의 값 구하기
- **Result**: $a = -1, f(-1) = 1, f'(-1) = 3$

> **📝 해설 (Explanation)**
> $\lim _{x \rightarrow a} \frac{f(x)-1}{x-a}=3$ 에서 $x \to a$ 일 때 분모가 $0$ 으로 수렴하므로, 극한값이 존재하기 위해서는 분자도 $0$ 으로 수렴해야 합니다.
> 그러므로 $\lim_{x \to a} (f(x)-1) = f(a)-1 = 0$ 에서 $f(a) = 1$ 임을 알 수 있습니다.
> 이때 주어진 극한식은 미분계수의 정의에 의해 $f'(a) = 3$ 을 의미합니다.
> 점 $(a, f(a))$, 즉 $(a, 1)$ 에서의 접선의 방정식은 기울기가 $f'(a)=3$ 이므로 $y - 1 = 3(x - a)$ 에서 $y = 3x - 3a + 1$ 입니다.
> 이 접선의 $y$ 절편이 $4$ 이므로 $-3a + 1 = 4$ 가 성립하며, 이를 풀면 $3a = -3$ 에서 $a = -1$ 입니다.
> 결과적으로 $f(-1) = 1, f'(-1) = 3$ 임을 얻습니다.

## [Step 2] 다항함수의 미분법과 연립방정식을 활용하여 삼차함수의 미정계수와 함숫값 구하기

- **Trigger**: [다항함수 조건] 최고차항의 계수가 $1$, $f(0)=0$, $f(-1)=1$, $f'(-1)=3$ (함수의 미정계수 결정 필요)
- **Action**: [12미적Ⅰ-02-04] $f(x) = x^3 + bx^2 + cx$ 에 다항함수의 미분법을 적용하고 미분계수 조건을 연립하여 미정계수를 구한 뒤 $f(1)$ 계산하기
- **Result**: $-5$

> **📝 해설 (Explanation)**
> $f(0)=0$ 이고 최고차항의 계수가 $1$ 인 삼차함수이므로 $f(x) = x^3 + bx^2 + cx$ 라 두면, 다항함수의 미분법에 의해 도함수는 $f'(x) = 3x^2 + 2bx + c$ 입니다.
> 앞서 구한 조건을 대입하여 미정계수 $b, c$ 에 관한 연립방정식을 세우면 다음과 같습니다.
> 1) $f(-1) = (-1)^3 + b(-1)^2 + c(-1) = -1 + b - c = 1 \Rightarrow b - c = 2$
> 2) $f'(-1) = 3(-1)^2 + 2b(-1) + c = 3 - 2b + c = 3 \Rightarrow -2b + c = 0$
> (2)에서 $c = 2b$ 이므로 이를 (1)에 대입하면 $b - 2b = -b = 2$ 에서 $b = -2$ 입니다.
> 이를 통해 $c = 2(-2) = -4$ 임을 알 수 있으며, 삼차함수의 식은 $f(x) = x^3 - 2x^2 - 4x$ 가 됩니다.
> 구하고자 하는 $f(1)$ 의 값은 $f(1) = 1^3 - 2(1^2) - 4(1) = 1 - 2 - 4 = -5$ 입니다.
> 따라서 정답은 **⑤**입니다.
