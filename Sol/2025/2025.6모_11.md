## [Step 1] 극한의 성질을 이용하여 함숫값과 미분계수 산출하기

- **Trigger**: [지수함수 조건] $\lim _{x \rightarrow a} \frac{f(x)-1}{x-a}=3$
- **Action**: [CPT-CA1-DIF-001] 극한의 존재 조건으로부터 분자가 0이 되어야 함을 이용하여 $f(a)$를 구하고, 미분계수의 정의를 통해 $f'(a)$ 산출
- **Result**: $f(a) = 1, f'(a) = 3$

> **📝 해설 (Explanation)**
> $\lim _{x \rightarrow a} \frac{f(x)-1}{x-a}=3$ 에서 분모가 $0$으로 수렴하므로 분자도 $0$으로 수렴해야 합니다.
> 따라서 $f(a) - 1 = 0 \Rightarrow f(a) = 1$ 입니다.
> 또한 미분계수의 정의에 의해 $f'(a) = 3$ 입니다.
> 따라서 곡선 $y=f(x)$ 위의 점 $(a, 1)$ 에서의 접선의 기울기는 $3$ 입니다.

## [Step 2] 접선의 방정식을 이용하여 변수 산출하기

- **Trigger**: [지수함수 조건] 점 $(a, f(a))$에서의 접선의 $y$절편이 4
- **Action**: [CPT-CA1-DIF-003] 접점 $(a, 1)$과 기울기 $3$을 이용하여 접선의 방정식을 구하고, $y$절편 조건을 대입하여 $a$의 값 산출
- **Result**: $a = -1$

> **📝 해설 (Explanation)**
> 점 $(a, 1)$ 에서의 접선의 방정식은 $y - 1 = 3(x - a) \Rightarrow y = 3x - 3a + 1$ 입니다.
> 이 직선의 $y$절편이 $4$ 이므로 $-3a + 1 = 4 \Rightarrow -3a = 3 \Rightarrow a = -1$ 입니다.
> 따라서 $f(-1) = 1$ 이고 $f'(-1) = 3$ 입니다.

## [Step 3] 삼차함수의 식을 완성하고 최종 결과 도출하기

- **Trigger**: [최종 구하는 값] $f(1)$
- **Action**: [CPT-CA1-DIF-003] 최고차항 계수가 1이고 $f(0)=0, f(-1)=1, f'(-1)=3$ 임을 이용하여 $f(x) = x^3 + bx^2 + cx$ 식의 미지수 산출
- **Result**: $f(1) = -3$

> **📝 해설 (Explanation)**
> $f(0)=0$ 이고 최고차항의 계수가 $1$ 인 삼차함수이므로 $f(x) = x^3 + bx^2 + cx$ 라고 합합시다.
> 이를 미분하면 $f'(x) = 3x^2 + 2bx + c$ 입니다.
> $f(-1) = (-1)^3 + b(-1)^2 + c(-1) = -1 + b - c = 1 \Rightarrow b - c = 2$ ⋯ (1)
> $f'(-1) = 3(-1)^2 + 2b(-1) + c = 3 - 2b + c = 3 \Rightarrow -2b + c = 0 \Rightarrow c = 2b$ ⋯ (2)
> (2)를 (1)에 대입하면 $b - 2b = -b = 2 \Rightarrow b = -2, c = -4$ 입니다.
> 따라서 $f(x) = x^3 - 2x^2 - 4x$ 입니다.
> 구하고자 하는 $f(1)$ 의 값은 $1 - 2 - 4 = -5$ 입니다. (아, 다시 확인: $1-2-4=-5$. 선택지에 -5가 있습니다.)
> 정답은 **⑤**입니다.
