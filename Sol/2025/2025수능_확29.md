## [Step 1] 정규분포의 대칭성을 이용하여 확률변수의 평균 추론하기

- **Trigger**: [정규분포 조건] $P(X \le x) = P(X \ge 40-x)$ 가 모든 실수 $x$에서 성립 (대칭축 $m_1=20$ 도출)
- **Action**: [12확통03-04] 정규분포 곡선이 평균에 대해 대칭임을 이용하여 평균 $m_1$ 도출
- **Result**: $m_1 = 20$

> **📝 해설 (Explanation)**
> 확률변수 $X$의 정규분포 곡선은 $x=m_1$에 대하여 대칭입니다.
> 주어진 확률 조건 $P(X \le x) = P(X \ge 40-x)$가 모든 실수 $x$에 대해 성립하려면, 두 확률의 기준이 되는 $x$와 $40-x$의 중점이 항상 정규분포의 평균 $m_1$과 일치해야 합니다.
> 즉, $m_1 = \frac{x + (40-x)}{2} = \frac{40}{2} = 20$입니다.

## [Step 2] 확률변수의 관계식을 표준화하여 두 정규분포의 평균과 표준편차 비교하기

- **Trigger**: [정규분포 조건] $P(Y \le x) = P(X \le x+10)$ 가 모든 실수 $x$에서 성립 (표준화 항등식으로 $m_2=10$, $\sigma_1=\sigma_2$ 도출)
- **Action**: [12확통03-04] $X$와 $Y$를 각각 표준화하여 얻은 $Z$의 부등식이 모든 $x$에 대해 항등식이 됨을 이용하여 $m_2$와 $\sigma_1$, $\sigma_2$의 관계 도출
- **Result**: $m_2 = 10$, $\sigma_1 = \sigma_2$ (공통 표준편차를 $\sigma$로 설정)

> **📝 해설 (Explanation)**
> 두 확률변수 $X \sim N(20, \sigma_1^2)$과 $Y \sim N(m_2, \sigma_2^2)$의 확률을 각각 표준화 공식을 이용해 정리합니다.
> $P(Y \le x) = P(Z \le \frac{x-m_2}{\sigma_2})$ 입니다.
> $P(X \le x+10) = P(Z \le \frac{(x+10)-20}{\sigma_1}) = P(Z \le \frac{x-10}{\sigma_1})$ 입니다.
> 이 두 확률이 모든 실수 $x$에 대하여 같으려면 표준화된 $Z$의 범위가 항상 같아야 하므로, $\frac{x-m_2}{\sigma_2} = \frac{x-10}{\sigma_1}$가 $x$에 대한 항등식이어야 합니다.
> $x$의 계수와 상수항을 각각 비교합니다.
> $\frac{1}{\sigma_2} = \frac{1}{\sigma_1}$에서 $\sigma_1 = \sigma_2$입니다. (이후 공통된 표준편차를 $\sigma$라고 하겠습니다.)
> $-\frac{m_2}{\sigma_2} = -\frac{10}{\sigma_1}$에서 $\sigma_1 = \sigma_2$이므로 $m_2 = 10$입니다.

## [Step 3] 표준정규분포표를 이용하여 공통 표준편차 산출하기

- **Trigger**: [최종 구하는 값] $m_1 + \sigma_2$ ($P(15 \le X \le 20) + P(15 \le Y \le 20) = 0.4772$로 $\sigma=5$ 확정 후 산출)
- **Action**: [12확통03-04] 주어진 확률을 표준화하여 $Z$의 확률로 변환한 뒤, 두 확률의 구간을 더하여 표준정규분포표의 값과 비교
- **Result**: $\sigma = 5$, $m_1 + \sigma_2 = 25$

> **📝 해설 (Explanation)**
> 주어진 두 확률을 표준편차 $\sigma$를 사용하여 차례대로 표준화합니다.
> $P(15 \le X \le 20) = P(\frac{15-20}{\sigma} \le Z \le \frac{20-20}{\sigma}) = P(-\frac{5}{\sigma} \le Z \le 0)$ 입니다. 표준정규분포의 대칭성에 의해 이는 $P(0 \le Z \le \frac{5}{\sigma})$ 와 같습니다.
> $P(15 \le Y \le 20) = P(\frac{15-10}{\sigma} \le Z \le \frac{20-10}{\sigma}) = P(\frac{5}{\sigma} \le Z \le \frac{10}{\sigma})$ 입니다.
> 이 두 확률을 더하면 꼬리를 무는 연속된 구간이 되어 $P(0 \le Z \le \frac{10}{\sigma})$ 가 됩니다.
> 주어진 표에서 $P(0 \le Z \le 2.0) = 0.4772$이므로 $\sigma = 5$이며, $m_1+\sigma_2 = 20+5 = 25$입니다.
> 정답은 **25**입니다.
