## [Step 1] 정규분포의 대칭성을 활용하여 확률변수의 평균 구하기

- **Trigger**: [정규분포 조건] $P(X \le x) = P(X \ge 40-x)$ 가 모든 실수 $x$ 에 대하여 성립 (확률밀도함수의 대칭축 분석 필요)
- **Action**: [12확통03-04] 정규분포 곡선이 평균 $m_1$ 에 대하여 대칭임을 이용하여 평균의 값을 산출함
- **Result**: $m_1 = 20$

> **📝 해설 (Explanation)**
> 모든 실수 $x$ 에 대하여 $\mathrm{P}(X \le x) = \mathrm{P}(X \ge 40-x)$ 가 성립한다는 것은, 확률변수 $X$ 의 분포가 $x$ 와 $40-x$ 의 중점에 대하여 대칭임을 의미합니다.
> 따라서 정규분포의 평균 $m_1$ 은 $\frac{x + (40-x)}{2} = 20$ 입니다.

## [Step 2] 확률변수의 표준화를 활용하여 평균과 표준편차의 관계 분석하기

- **Trigger**: [정규분포 조건] $P(Y \le x) = P(X \le x+10)$ 가 모든 실수 $x$ 에 대하여 성립 (표준화 식의 항등 성질 분석 필요)
- **Action**: [12확통03-04] 두 확률변수를 각각 표준화하여 $Z$ 에 관한 식으로 변환하고, 계수 비교를 통해 $m_2$ 와 $\sigma_2$ 의 관계를 도출함
- **Result**: $m_2 = 10, \sigma_1 = \sigma_2$ (공통 표준편차 $\sigma$ 로 설정)

> **📝 해설 (Explanation)**
> 두 확률변수 $X \sim \mathrm{N}(20, \sigma_1^2)$ 과 $Y \sim \mathrm{N}(m_2, \sigma_2^2)$ 를 표준화하여 조건을 재구성합니다.
> $\mathrm{P}(Y \le x) = \mathrm{P}\left(Z \le \frac{x-m_2}{\sigma_2}\right)$
> $\mathrm{P}(X \le x+10) = \mathrm{P}\left(Z \le \frac{(x+10)-20}{\sigma_1}\right) = \mathrm{P}\left(Z \le \frac{x-10}{\sigma_1}\right)$
> 이 두 확률이 모든 $x$ 에 대해 같으려면 $\frac{x-m_2}{\sigma_2} = \frac{x-10}{\sigma_1}$ 이 $x$ 에 대한 항등식이어야 하므로, $\sigma_1 = \sigma_2$ 이고 $m_2 = 10$ 임을 알 수 있습니다. 이후 공통 표준편차를 $\sigma$ 로 정의합니다.

## [Step 3] 표준정규분포를 활용하여 상수의 값 구하기

- **Trigger**: [최종 구하는 값] $m_1 + \sigma_2$ ($\mathrm{P}(15 \le X \le 20) + \mathrm{P}(15 \le Y \le 20) = 0.4772$ 활용)
- **Action**: [12확통03-04] 주어진 확률을 표준화하여 $Z$ 의 확률로 변환하고, 구간의 병합을 통해 표준정규분포표의 수치와 비교하여 $\sigma$ 를 계산함
- **Result**: $25$

> **📝 해설 (Explanation)**
> 주어진 확률의 합을 표준화하면 다음과 같습니다.
> $\mathrm{P}(15 \le X \le 20) = \mathrm{P}\left(\frac{15-20}{\sigma} \le Z \le 0\right) = \mathrm{P}\left(0 \le Z \le \frac{5}{\sigma}\right)$
> $\mathrm{P}(15 \le Y \le 20) = \mathrm{P}\left(\frac{15-10}{\sigma} \le Z \le \frac{20-10}{\sigma}\right) = \mathrm{P}\left(\frac{5}{\sigma} \le Z \le \frac{10}{\sigma}\right)$
> 두 확률을 더하면 $\mathrm{P}\left(0 \le Z \le \frac{10}{\sigma}\right) = 0.4772$ 가 됩니다.
> 표준정규분포표에서 $\mathrm{P}(0 \le Z \le 2.0) = 0.4772$ 이므로 $\frac{10}{\sigma} = 2$, 즉 $\sigma = 5$ 입니다.
> 따라서 $m_1 + \sigma_2 = 20 + 5 = 25$ 입니다.
>
> 정답은 **25**입니다.
