## [Step 1] 구간별 함수의 식을 분석하여 그래프의 개형과 치역 파악하기

- **Trigger**: [삼각함수 부호 조건] 구간 $[0,\pi)$에서 $\sin x \ge 0$, $[\pi, 2\pi]$에서 $\sin x \le 0$ (구간별 $f(x)$ 치역과 그래프 개형 파악)
- **Action**: [12대수02-02] 각 구간에서 $\sin x$의 치역을 바탕으로 $f(x)$의 최댓값과 최솟값 및 그래프의 개형 도출
- **Result**: $0 \le x < \pi$에서 최댓값 $0$, 최솟값 $-1$ / $\pi \le x \le 2\pi$에서 최댓값 $\sqrt{2}-1$, 최솟값 $-1$

> **📝 해설 (Explanation)**
> 함수 $f(x)$의 그래프를 파악하기 위해 구간을 나누어 최댓값과 최솟값을 확인합니다.
> 첫째, $0 \le x < \pi$에서 $0 \le \sin x \le 1$이므로, $f(x) = \sin x - 1$의 범위는 $-1 \le f(x) \le 0$입니다. 이 구간에서 $x = \frac{\pi}{2}$일 때 최댓값 $0$을 가지고, $x = 0$일 때 $-1$을 가집니다. ($x \to \pi$일 때 $-1$에 한없이 가까워집니다.)
> 둘째, $\pi \le x \le 2\pi$에서 $-1 \le \sin x \le 0$이므로, $0 \le -\sqrt{2}\sin x \le \sqrt{2}$가 됩니다. 따라서 $f(x) = -\sqrt{2}\sin x - 1$의 범위는 $-1 \le f(x) \le \sqrt{2}-1$입니다. 이 구간에서 $x = \frac{3\pi}{2}$일 때 최댓값 $\sqrt{2}-1$을 가지고, $x = \pi$와 $x = 2\pi$일 때 최솟값 $-1$을 가집니다.
> 즉, $f(x)$의 그래프는 $x$축 아래에 머물다가 두 번 위로 볼록하게 솟아오르는 형태이며, 두 번째 봉우리가 더 높습니다.

## [Step 2] 그래프 교점 개수 조건으로 목표 함숫값의 범위 결정하기

- **Trigger**: [추가 관계식 조건] 방정식 $f(x) = f(t)$의 실근이 $3$개 (직선 $y=k$와의 교점 개수 분석으로 $k=0$ 또는 $k=-1$ 확정)
- **Action**: [12대수02-02] 그래프의 개형을 바탕으로 직선 $y=k$ (단, $k=f(t)$)를 위아래로 움직이며 교점의 개수가 $3$개가 되는 상수 $k$의 값 조사
- **Result**: 교점이 3개인 상수 $k$의 값은 $0$, $-1$

> **📝 해설 (Explanation)**
> 방정식 $f(x) = f(t)$의 실근의 개수는 $y=f(x)$의 그래프와 상수함수인 직선 $y=k$ (단, $k=f(t)$)가 만나는 교점의 개수와 같습니다.
> 교점의 개수를 $k$의 값에 따라 분류해보면 다음과 같습니다.
> $k = \sqrt{2}-1$일 때, 교점은 $1$개입니다.
> $0 < k < \sqrt{2}-1$일 때, 교점은 $2$개입니다.
> $k = 0$일 때, 첫 번째 구간의 봉우리 꼭대기에서 $1$개, 두 번째 구간에서 $2$개 만나므로 교점은 총 $3$개입니다.
> $-1 < k < 0$일 때, 첫 번째 구간에서 $2$개, 두 번째 구간에서 $2$개 만나므로 교점은 총 $4$개입니다.
> $k = -1$일 때, 양 끝점과 파동의 경계에서 만나므로 교점은 $x = 0$, $x = \pi$, $x = 2\pi$로 총 $3$개입니다.
> 따라서 서로 다른 실근의 개수가 $3$개가 되도록 하는 함숫값 $f(t)$는 $0$ 또는 $-1$이어야 합니다.

## [Step 3] 삼각방정식의 실근 조건을 분석하여 조건을 만족시키는 변수의 값을 탐색하고 합산하기

- **Trigger**: [최종 구하는 값] $f(t)=0$ 또는 $f(t)=-1$ ($t \in [0, 2\pi]$에서 모든 해를 구하여 $p+q$ 산출)
- **Action**: [12대수02-02] 각 구간의 함수식에 $f(t)=0$과 $f(t)=-1$을 대입하여 삼각방정식을 풀고 해를 모두 도출한 뒤 합산하여 $p+q$ 계산
- **Result**: $t \in \left\{0, \frac{\pi}{2}, \pi, \frac{5\pi}{4}, \frac{7\pi}{4}, 2\pi\right\}$, 합 $= \frac{13}{2}\pi$, $p=2, q=13$, $p+q=15$

> **📝 해설 (Explanation)**
> 이제 $f(t) = 0$과 $f(t) = -1$을 만족하는 실수 $t$를 $0 \le t \le 2\pi$ 범위에서 각각 찾습니다.
>
> 1) $f(t) = 0$인 경우:
> $0 \le t < \pi$ 범위에서는 $\sin t - 1 = 0$, 즉 $\sin t = 1$이므로 $t = \frac{\pi}{2}$입니다.
> $\pi \le t \le 2\pi$ 범위에서는 $-\sqrt{2}\sin t - 1 = 0$, 즉 $\sin t = -\frac{\sqrt{2}}{2}$이므로 $t = \frac{5\pi}{4}$, $t = \frac{7\pi}{4}$입니다.
>
> 2) $f(t) = -1$인 경우:
> 앞서 그래프의 개형에서 확인했듯이, $f(t)$의 최솟값이 $-1$이 되는 지점은 $t = 0$, $t = \pi$, $t = 2\pi$의 세 군데입니다.
>
> 따라서 조건을 만족하는 모든 $t$의 값은 $0$, $\frac{\pi}{2}$, $\pi$, $\frac{5\pi}{4}$, $\frac{7\pi}{4}$, $2\pi$입니다.
> 모든 $t$의 합 $= 0 + \frac{\pi}{2} + \pi + \frac{5\pi}{4} + \frac{7\pi}{4} + 2\pi = \frac{\pi}{2} + 3\pi + 3\pi = 6\pi + \frac{\pi}{2} = \frac{13}{2}\pi$입니다.
> $\frac{13}{2}\pi = \frac{q}{p}\pi$이고 $p$, $q$는 서로소인 자연수이므로 $p=2$, $q=13$입니다.
> $p+q = 2+13 = 15$이며, 정답은 **15**입니다.
