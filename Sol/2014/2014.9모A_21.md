## [Step 1] 도함수의 부호 조건을 분석하여 미정계수의 관계식 구하기

- **Trigger**: [증감 조건] $(-\infty, 0)$ 에서 감소, $(2, \infty)$ 에서 증가
- **Action**: [12미적Ⅰ-02-07] $f'(x) = (x+1)(x^2+ax+b)$ 가 $x < 0$ 영역에서 부호가 변하지 않고 항상 $\le 0$ 이어야 함을 이용하여 $(x+1)$ 의 인수가 짝수 차수가 되어야 함을 파악하고 $a, b$ 를 하나의 변수로 표현하기
- **Result**: $a = 1-\alpha, b = -\alpha$ ($0 \le \alpha \le 2$)

> **📝 해설 (Explanation)**
> 사차함수 $f(x)$ 의 도함수가 $f'(x) = (x+1)(x^2+ax+b)$ 입니다.
> 1. $x < 0$ 구간에서 함수가 감소하려면 $f'(x) \le 0$ 이어야 합니다. 인수인 $(x+1)$ 은 $x=-1$ 을 기준으로 부호가 변하므로, 전체 부호가 음을 유지하려면 $x^2+ax+b$ 역시 $x=-1$ 을 인수로 가져야 합니다.
>    따라서 $x^2+ax+b = (x+1)(x-\alpha)$ 라 할 수 있으며, $f'(x) = (x+1)^2(x-\alpha)$ 입니다.
> 2. $x < 0$ 에서 $f'(x) = (x+1)^2(x-\alpha) \le 0$ 이어야 하므로 $x-\alpha \le 0 \implies x \le \alpha$ 입니다. 모든 $x < 0$ 에 대해 성립하려면 $\alpha \ge 0$ 이어야 합니다.
> 3. $x > 2$ 에서 $f'(x) = (x+1)^2(x-\alpha) \ge 0$ 이어야 하므로 $x-\alpha \ge 0 \implies x \ge \alpha$ 입니다. 모든 $x > 2$ 에 대해 성립하려면 $\alpha \le 2$ 이어야 합니다.
> 결과적으로 $0 \le \alpha \le 2$ 이며, $a = 1-\alpha, b = -\alpha$ 가 성립합니다.

## [Step 2] 이차함수의 최대 및 최소 성질을 이용하여 결과값 구하기

- **Trigger**: [구하는 값의 최댓값/최솟값] $a^2+b^2$ 의 범위
- **Action**: [10공수02-01-02] $a, b$ 를 $\alpha$ 에 관한 식으로 변환하여 이차함수 $g(\alpha)$ 를 수립하고, 제한된 범위 $0 \le \alpha \le 2$ 에서의 최대와 최소 구하기
- **Result**: $M+m = \frac{11}{2}$ (최종 정답)

> **📝 해설 (Explanation)**
> $a^2 + b^2 = (1-\alpha)^2 + (-\alpha)^2 = 2\alpha^2 - 2\alpha + 1$ 입니다.
> 이 이차함수를 $g(\alpha) = 2(\alpha - \frac{1}{2})^2 + \frac{1}{2}$ 로 변형하면 다음과 같습니다.
> 1. 최솟값 ($m$): $\alpha = \frac{1}{2}$ 일 때, $m = \frac{1}{2}$ 입니다.
> 2. 최댓값 ($M$): 범위의 양 끝점인 $\alpha=0$ 또는 $\alpha=2$ 를 확인합니다.
>    - $g(0) = 1$
>    - $g(2) = 2(4) - 2(2) + 1 = 5$
>    따라서 $M = 5$ 입니다.
> 최종적으로 $M + m = 5 + \frac{1}{2} = \frac{11}{2}$ 입니다.
>
> 따라서 정답은 **③**입니다.
