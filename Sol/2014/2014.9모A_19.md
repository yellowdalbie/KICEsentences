## [Step 1] 표준정규분포를 이용하여 확률 구하기

- **Trigger**: [정규분포 조건] 정규분포 $X \sim \mathrm{N}(\frac{3}{2}, 2^2)$ 에서 $H(0)$ 과 $H(2)$ 의 합산
- **Action**: [12확통03-04] $Z = \frac{X - 1.5}{2}$ 로 표준화하여 각각의 확률 $H(0) = \mathrm{P}(0 \le X \le 1)$ 과 $H(2) = \mathrm{P}(2 \le X \le 3)$ 도출
- **Result**: $H(0) = 0.1747$, $H(2) = 0.1747$, 총합 $0.3494$

> **📝 해설 (Explanation)**
> $H(t)=\mathrm{P}(t \leq X \leq t+1)$ 이므로, $H(0)$ 은 $\mathrm{P}(0 \leq X \leq 1)$ 입니다.
> 정규분포의 평균이 $1.5$, 표준편차가 $2$ 이므로 확륣변수 $Z=\frac{X-1.5}{2}$ 로 표준화합니다.
> $H(0)$ $= \mathrm{P}\left(\frac{0-1.5}{2} \leq Z \leq \frac{1-1.5}{2}\right)$ $= \mathrm{P}(-0.75 \leq Z \leq -0.25)$
> 표준정규분포의 대칭성에 의해 이는 $\mathrm{P}(0.25 \leq Z \leq 0.75)$ 와 같으며, $\mathrm{P}(0 \leq Z \leq 0.75)$ $- \mathrm{P}(0 \leq Z \leq 0.25)$ 로 구합니다.
> 표의 값을 대입하면 $0.2734 - 0.0987 = 0.1747$ 입니다.
> 다음으로 $H(2)$ 는 $\mathrm{P}(2 \leq X \leq 3)$ 이며, 이를 표준화하면
> $H(2)$ $= \mathrm{P}\left(\frac{2-1.5}{2} \leq Z \leq \frac{3-1.5}{2}\right)$ $= \mathrm{P}(0.25 \leq Z \leq 0.75)$ 입니다.
> 앞서 구한 값과 동일하게 $0.1747$ 입니다.
> $H(0) + H(2) = 0.1747 + 0.1747 = 0.3494$ 입니다.
> 따라서 정답은 **①**입니다.
