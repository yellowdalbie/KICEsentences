## [Step 1] 정규분포의 표준화를 활용하여 구간 확률을 표준정규분포로 변환하기

- **Trigger**: [정규분포 조건] $X \sim N(1.5, 2^2)$, $H(t) = P(t \le X \le t+1)$
- **Action**: [12확통03-04] $H(0) = P(0 \le X \le 1)$ 과 $H(2) = P(2 \le X \le 3)$ 을 각각 표준화하여 $Z$ 의 확률로 표현
- **Result**: $H(0) = P(-0.75 \le Z \le -0.25)$, $H(2) = P(0.25 \le Z \le 0.75)$

## [Step 2] 표준정규분포의 대칭성을 이용하여 합산 확률의 수치 산출하기

- **Trigger**: [정규분포 조건] $H(0) + H(2)$ 의 합산
- **Action**: [12확통03-04] $H(0) = P(0.25 \le Z \le 0.75)$ 임을 파악하고 $H(0)+H(2) = 2 \times P(0.25 \le Z \le 0.75)$ 계산
- **Result**: 0.3494 (최종 수치 확정)

> **📝 해설 (Explanation)**
> $X \sim N(1.5, 2^2)$ 이므로 $Z = \frac{X-1.5}{2}$ 입니다.
> $H(0) = P(0 \le X \le 1) = P(\frac{0-1.5}{2} \le Z \le \frac{1-1.5}{2}) = P(-0.75 \le Z \le -0.25)$ 입니다.
> $H(2) = P(2 \le X \le 3) = P(\frac{2-1.5}{2} \le Z \le \frac{3-1.5}{2}) = P(0.25 \le Z \le 0.75)$ 입니다.
> 표준정규분포의 대칭성에 의해 $H(0) = P(0.25 \le Z \le 0.75)$ 이 성립합니다.
> $H(0) + H(2) = 2 \times \{P(0 \le Z \le 0.75) - P(0 \le Z \le 0.25)\}$
> $2 \times (0.2734 - 0.0987) = 2 \times 0.1747 = 0.3494$ 입니다.
> 정답은 **①**입니다.
