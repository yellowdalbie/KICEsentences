## [Step 1] 정규분포의 표준화와 대칭성을 활용한 확률값의 합산 산출하기

- **Trigger**: [정규분포 조건] $X \sim N(1.5, 2^2)$이고 확률 함수 $H(k) = \mathrm{P}(k \leq X \leq k+1)$로 정의됨
- **Action**: [12확통03-04] 표준화 변수 $Z = \frac{X-1.5}{2}$를 이용하여 $H(0)$과 $H(2)$를 표준정규분포로 변환하고 대칭성을 확인하여 합산
- **Result**: $0.3494$ (두 영역의 확률 합)

> **📝 해설 (Explanation)**
> 확률변수 $X$가 평균 $m=1.5$, 표준편차 $\sigma=2$인 정규분포를 따르므로, $Z = \frac{X-1.5}{2}$로 표준화를 수행합니다. 문제에서 구하고자 하는 값은 $H(0) + H(2)$입니다.
>
> 1. $H(0)$의 계산:
> $H(0) = \mathrm{P}(0 \leq X \leq 1)$이며 이를 표준화하면 다음과 같습니다.
> $\mathrm{P}\left(\frac{0-1.5}{2} \leq Z \leq \frac{1-1.5}{2}\right) = \mathrm{P}(-0.75 \leq Z \leq -0.25)$
> 표준정규분포의 대칭성에 의해 이는 $\mathrm{P}(0.25 \leq Z \leq 0.75)$와 값이 같습니다.
>
> 2. $H(2)$의 계산:
> $H(2) = \mathrm{P}(2 \leq X \leq 3)$이며 이를 표준화하면 다음과 같습니다.
> $\mathrm{P}\left(\frac{2-1.5}{2} \leq Z \leq \frac{3-1.5}{2}\right) = \mathrm{P}(0.25 \leq Z \leq 0.75)$
>
> 계산 결과 $H(0)$과 $H(2)$는 서로 같은 확률값을 가집니다. 표준정규분포표를 이용하여 값을 구합니다.
> $\mathrm{P}(0.25 \leq Z \leq 0.75) = \mathrm{P}(0 \leq Z \leq 0.75) - \mathrm{P}(0 \leq Z \leq 0.25)$
> $= 0.2734 - 0.0987 = 0.1747$
>
> 최종적으로 $H(0) + H(2) = 0.1747 + 0.1747 = 0.3494$입니다.
> 따라서 정답은 **①**입니다.
