## [Step 1] 표본평균의 분포를 이용하여 새로운 확률변수의 정규분포 구하기

- **Trigger**: [표본 추출 조건] 모평균 $m=20$, 모표준편차 $\sigma=5$ 인 모집단에서 크기 $n=16$ 인 표본 추출
- **Action**: [12확통03-05] 표본평균 $\bar{X}$ 의 분포가 평균 $20$, 표준편차 $\frac{5}{\sqrt{16}}$ 인 정규분포 $\mathrm{N}\left(20, \left(\frac{5}{4}\right)^2\right)$ 를 따름을 식별
- **Result**: $\bar{X} \sim \mathrm{N}(20, 1.25^2)$

> **📝 해설 (Explanation)**
> 모집단의 상담 시간이 정규분포 $\mathrm{N}(20, 5^2)$ 을 따릅니다.
> 크기가 $16$ 인 표본을 임의추출하였으므로, 표본평균 $\bar{X}$ 의 평균은 모평균과 같은 $20$ 입니다.
> 표본평균 $\bar{X}$ 의 표준편차는 $\frac{\sigma}{\sqrt{n}}$ $= \frac{5}{\sqrt{16}}$ $= \frac{5}{4}$ $= 1.25$ 입니다.
> 그러므로 $\bar{X}$ 는 정규분포 $\mathrm{N}(20, 1.25^2)$ 을 따르게 됩니다.

## [Step 2] 표준정규분포를 이용하여 확률 구하기

- **Trigger**: [정규분포 조건] $19 \leq \bar{X} \leq 22$ 인 확률
- **Action**: [12확통03-03] 확률변수 $Z = \frac{\bar{X} - 20}{1.25}$ 로 표준화하여 표준정규분포표 값 적용
- **Result**: $\mathrm{P}(-0.8 \leq Z \leq 1.6) = 0.2881 + 0.4452 = 0.7333$

> **📝 해설 (Explanation)**
> 구하고자 하는 확률 $\mathrm{P}(19 \leq \bar{X} \leq 22)$ 를 표준화 변수 $Z$ 로 바꿉니다.
> $Z$ $= \frac{\bar{X} - m}{\frac{\sigma}{\sqrt{n}}}$ $= \frac{\bar{X} - 20}{1.25}$ 이므로,
> $$\mathrm{P}\left(\frac{19 - 20}{1.25} \leq Z \leq \frac{22 - 20}{1.25}\right)$$ $= \mathrm{P}\left(\frac{-1}{1.25} \leq Z \leq \frac{2}{1.25}\right)$ $= \mathrm{P}(-0.8 \leq Z \leq 1.6)$ 입니다.
> 정규분포의 대칭성에 의해 이 확률은 $\mathrm{P}(0 \leq Z \leq 0.8)$ $+ \mathrm{P}(0 \leq Z \leq 1.6)$ 와 같습니다.
> 표준정규분포표에서 주어진 값을 더하면 $0.2881 + 0.4452 = 0.7333$ 입니다.
> 따라서 정답은 **②**입니다.
