## [Step 1] 정규분포의 표준화를 통해 누적 확률을 단일 변수 z의 함수로 표현하기

- **Trigger**: [정규분포 조건] $X \sim N(t, (1/t^2)^2)$, $G(t) = P(X \le 1.5)$
- **Action**: [12확통03-04] $Z = \frac{1.5 - t}{1/t^2} = t^2(1.5-t)$ 로 표준화하여 $G(t) = P(Z \le 1.5t^2 - t^3)$ 유도
- **Result**: $Z \le 1.5t^2 - t^3$ (표준화된 범위 확보)

## [Step 2] 이차/삼차식의 극값 분석을 통해 누적 확률의 최댓값 결정하기

- **Trigger**: [함수의 증가·감소 조건] $h(t) = 1.5t^2 - t^3$ 의 최댓값 산출
- **Action**: [12미적Ⅰ-02-07] $h'(t) = 3t - 3t^2 = 0$ 으로부터 $t=1$ 일 때 최댓값을 가짐을 확인하고 $h(1)$ 및 대응하는 $P(Z)$ 연산
- **Result**: $h(1) = 0.5 \implies P(Z \le 0.5) = 0.5 + 0.1915 = 0.6915$

> **📝 해설 (Explanation)**
> $X \sim N(t, (1/t^2)^2)$ 이므로 $Z = \frac{1.5-t}{1/t^2} = t^2(1.5-t) = 1.5t^2 - t^3$ 입니다.
> $G(t) = P(Z \le 1.5t^2 - t^3)$ 이 최대가 되려면 $h(t) = 1.5t^2 - t^3$ 이 최대여야 합니다.
> $h'(t) = 3t - 3t^2 = 3t(1-t)$ 이므로 양의 실수 $t$ 에 대하여 $t=1$ 에서 극대이자 최대가 됩니다.
> $h(1) = 1.5(1)^2 - (1)^3 = 0.5$ 이므로 $G(t)$ 의 최댓값은 $P(Z \le 0.5)$ 입니다.
> 표준정규분포표에서 $P(0 \le Z \le 0.5) = 0.1915$ 이므로, $0.5 + 0.1915 = 0.6915$ 입니다.
> 정답은 **③**입니다.
