## [Step 1] 로그의 성질을 이용하여 수열의 일반항을 함숫값으로 확정하기

- **Trigger**: [수열 조건] $a_n = f(6^n) - f(3^n)$, (단, $f(n) = \log_2 n$ ($n$은 짝수), $f(n) = \log_3 n$ ($n$은 홀수))
- **Action**: [12대수01-04] $6^n$ 은 짝수, $3^n$ 은 홀수임을 활용하여 $a_n = \log_2 6^n - \log_3 3^n$ 식을 로그의 성질로 정리하기
- **Result**: $a_n = n \log_2 3$ (일반항 확보)

> **📝 해설 (Explanation)**
> $6^n$ 은 짝수이므로 $f(6^n) = \log_2 6^n = n \log_2 6$ 입니다.
> $3^n$ 은 홀수이므로 $f(3^n) = \log_3 3^n = n \log_3 3 = n$ 입니다.
> 그러므로 $a_n = n \log_2 6 - n = n(\log_2 2 + \log_2 3) - n = n + n \log_2 3 - n = n \log_2 3$ 입니다.

## [Step 2] 합의 기호의 성질과 자연수 거듭제곱의 합 공식을 이용하여 수열의 합 산출하기

- **Trigger**: [시그마 연산 조건] $\sum_{n=1}^{15} a_n$
- **Action**: [12대수03-04] $\sum_{n=1}^{15} n \log_2 3 = \log_2 3 \sum_{n=1}^{15} n$ 으로 변환하고 자연수의 합 공식 $\frac{n(n+1)}{2}$ 을 적용하여 연산하기
- **Result**: $120 \log_2 3$ (최종 도출 값)

> **📝 해설 (Explanation)**
> $\sum_{n=1}^{15} a_n = \sum_{n=1}^{15} (n \log_2 3) = \log_2 3 \sum_{n=1}^{15} n$ 입니다.
> 자연수의 합 공식에 의해 $\sum_{n=1}^{15} n = \frac{15 \times 16}{2} = 120$ 이 성립합니다.
> 결과적으로 구하는 값은 $120 \log_2 3$ 입니다.
>
> 따라서 정답은 **④**입니다.
