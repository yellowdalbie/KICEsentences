## [Step 1] 접선의 방정식을 수립하고 $y$ 절편의 절댓값으로 정의된 함수 $g(t)$ 의 구조 파악하기

- **Trigger**: [함수 정의 조건] 곡선 $y=f(x)$ 위 $(t, f(t))$ 에서의 접선의 $y$ 절편까지의 거리 $g(t)$
- **Action**: [12미적Ⅰ-02-05] 접선의 방정식 $y = f'(t)(x-t) + f(t)$ 를 구하고 $x=0$ 일 때의 $y$ 값에 절댓값을 씌워 $g(t) = |f(t) - tf'(t)|$ 도출하기
- **Result**: $g(t) = |2at^2 + t^3 - (3t^3 + 2at^2 + bt)|$ ? 아니지, $f(x)=x^3+ax^2+bx$ 에서 $f(t)-tf'(t) = -2t^3-at^2$

> **📝 해설 (Explanation)**
> $f(x) = x^3 + ax^2 + bx$ 이므로 $f'(x) = 3x^2 + 2ax + b$ 입니다.
> 접점 $(t, f(t))$ 에서의 접선의 방정식은 $y = (3t^2+2at+b)(x-t) + (t^3+at^2+bt)$ 입니다.
> $x=0$ 일 때 $y = -t(3t^2+2at+b) + t^3+at^2+bt = -2t^3 - at^2$ 입니다.
> 원점에서 이 점까지의 거리는 절댓값이므로 $g(t) = |-2t^3 - at^2| = |2t^3 + at^2|$ 입니다.

## [Step 2] 함수 $g(t)$ 가 실수 전체에서 미분가능할 조건을 통해 미지수 확정하기

- **Trigger**: [미분가능 조건] $g(t)$ 는 전 구간 미분가능
- **Action**: [12미적Ⅰ-02-02] 절댓값 내부의 함수 $h(t) = 2t^3 + at^2$ 가 부호가 바뀌는 지점에서 미분계수가 0이거나 중근을 가져야 함을 이용하여 $a=0$ 도출하기
- **Result**: $a=0, f(x) = x^3 + bx$

> **📝 해설 (Explanation)**
> $g(t) = |t^2(2t+a)|$ 입니다. $t=0$ 에서는 $t^2$ 에 의해 미분가능하지만, $2t+a=0$ 인 $t = -a/2$ 에서 부호가 전환됩니다.
> 절댓값 함수 $|h(t)|$ 가 실수 전체에서 미분가능하려면 부호가 바뀌는 근이 중근 이상이어야 합니다.
> 따라서 $-a/2 = 0$ 이어야 하므로 $a=0$ 입니다.

## [Step 3] 제시된 함숫값 조건을 대입하여 나머지 미지수를 구하고 최종 결과 산출하기

- **Trigger**: [함숫값 조건] $f(1)=2$
- **Action**: [12미적Ⅰ-02-01] $f(1) = 1 + b = 2$ 에서 $b=1$ 을 구하고 $f(3)$ 계산하기
- **Result**: $f(3) = 30$ (최종 수치)

> **📝 해설 (Explanation)**
> $f(x) = x^3 + bx$ 에서 $f(1) = 1+b = 2$ 이므로 $b=1$ 입니다.
> 따라서 $f(x) = x^3 + x$ 이고, $f(3) = 3^3 + 3 = 27 + 3 = 30$ 입니다.
> 정답은 **④**입니다.
