## [Step 1] 주어진 실생활 관계식에 관측 수치를 대입하여 비례 상수 결정하기

- **Trigger**: [로그의 성질 조건] $\log(b/a) = -1 + k \log c$, 조건 $\{a=10, c=8, b=4\}$
- **Action**: [12대수01-04] 수치를 대입하여 $\log(4/10) = -1 + k \log 8$ 식으로부터 $k \log 8 = \log 4$ 관계 도출 및 $k$ 산출
- **Result**: $k \cdot 3\log 2 = 2\log 2 \implies k = 2/3$

## [Step 2] 확정된 상수를 활용하여 새로운 상황에서의 목표 수치 계산하기

- **Trigger**: [로그의 성질 조건] $\{a=20, c=27, k=2/3\}$ 일 때 $b$ 구하기
- **Action**: [12대수01-04] $\log(b/20) = -1 + \frac{2}{3} \log 27$ 식에 $k$를 대입하여 로그 대수 연산 수행
- **Result**: $\log(b/20) = -1 + 2\log 3 = \log(9/10) \implies b = 18$

> **📝 해설 (Explanation)**
> 첫 번째 조건 $a=10, c=8, b=4$ 를 대입하면 $\log \frac{4}{10} = -1 + k \log 8$ 입니다.
> $\log 4 - 1 = -1 + k \log 2^3 \implies 2\log 2 = 3k \log 2$ 이므로 $k = \frac{2}{3}$ 입니다.
> 두 번째 조건 $a=20, c=27$ 을 대입하면 $\log \frac{b}{20} = -1 + \frac{2}{3} \log 27$ 입니다.
> $\log \frac{b}{20} = -1 + \frac{2}{3} \log 3^3 = -1 + 2\log 3 = -1 + \log 9$
> $\log \frac{b}{20} = \log \frac{1}{10} + \log 9 = \log \frac{9}{10}$
> 따라서 $\frac{b}{20} = \frac{9}{10} \implies b = 18$ 입니다.
> 정답은 **⑤**입니다.
