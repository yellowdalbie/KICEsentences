## [Step 1] 전확률의 성질을 이용하여 확률 변수의 정의 구간과 상수의 관계식 수립하기

- **Trigger**: [정규분포 조건] $P(0 \le X \le x) = kx^2$ ($0 \le x \le a$) 및 전체 확률 $1$
- **Action**: [12확통03-01] 전체 구간에 대한 누적 확률 $P(0 \le X \le a) = ka^2 = 1$ 식 수립하기
- **Result**: $k = 1/a^2$ (상수간 관계 확보)

## [Step 2] 확률밀도함수를 유도하고 기댓값 정의를 활용하여 미지수 확정하기

- **Trigger**: [기댓값 계산 조건] $E(X) = 1$ 및 확률밀도함수 $f(x) = \frac{d}{dx}P(0 \le X \le x)$
- **Action**: [12확통03-02] $f(x) = 2kx$ 를 도출하고 $\int_0^a x f(x) dx = \int_0^a 2kx^2 dx = 1$ 적분 방정식 풀기
- **Result**: $a = 3/2, k = 4/9$ (미지수 확정)

> **📝 해설 (Explanation)**
> 확률변수 $X$ 의 누적분포함수가 $F(x) = kx^2$ 이므로 전체 확률 $F(a) = ka^2 = 1$ 에서 $k = \frac{1}{a^2}$ 입니다.
> 확률밀도함수는 $f(x) = F'(x) = 2kx$ (단, $0 \le x \le a$) 입니다.
> 기댓값 $E(X) = \int_0^a x f(x) dx = \int_0^a 2kx^2 dx = [\frac{2}{3}kx^3]_0^a = \frac{2}{3}ka^3$
> $ka^2 = 1$ 을 대입하면 $E(X) = \frac{2}{3}a$ 입니다.
> 조건에서 $E(X) = 1$ 이므로 $\frac{2}{3}a = 1 \implies a = \frac{3}{2}$ 입니다.
> 따라서 $k = \frac{1}{(3/2)^2} = \frac{4}{9}$ 입니다.
> 정답은 **②**입니다.
