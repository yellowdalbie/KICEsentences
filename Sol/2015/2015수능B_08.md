## [Step 1] 정규분포의 성질을 이용하여 목표하는 확률의 표준화 경계값 설정하기

- **Trigger**: [정규분포 조건] $X \sim N(m, \sigma^2)$, $P(X \le 10) = P(X \ge 22) = 0.5 - P(0 \le Z \le 1.5)$
- **Action**: [12확통03-04] 대칭성에 의해 평균 $m = (10+22)/2 = 16$ 을 구하고 표준화 식 $Z = \frac{10-16}{\sigma} = -1.5$ 활용
- **Result**: $\sigma = 4$ (모수 확정)

## [Step 2] 표준정규분포표를 활용하여 특정 구간의 확률 산출하기

- **Trigger**: [정규분포 조건] $P(X \le 18)$ 산출
- **Action**: [12확통03-04] $P(Z \le \frac{18-16}{4}) = P(Z \le 0.5)$ 연산 수행
- **Result**: $0.5 + 0.1915 = 0.6915$

> **📝 해설 (Explanation)**
> $P(X \le 10) = P(X \ge 22)$ 이므로 정규분포의 대칭성에 의해 평균 $m = 16$ 입니다.
> $P(X \le 10) = P(Z \le \frac{10-16}{\sigma}) = P(Z \le -\frac{6}{\sigma}) = 0.0668$ 입니다.
> $P(0 \le Z \le 1.5) = 0.4332$ 이므로 $P(Z \ge 1.5) = 0.5 - 0.4332 = 0.0668$ 입니다.
> 따라서 $\frac{6}{\sigma} = 1.5 \implies \sigma = 4$ 입니다.
> $P(X \le 18) = P(Z \le \frac{18-16}{4}) = P(Z \le 0.5) = 0.5 + P(0 \le Z \le 0.5) = 0.5 + 0.1915 = 0.6915$ 입니다.
> 정답은 **③**입니다.
