## [Step 1] 부등식 경계를 특정 점에 대입하여 경계값 일치를 확인하고 함숫값 결정하기

- **Trigger**: [다항함수 부등식 조건] 모든 양의 실수 $x$에 대해 $6x - 6 \leq f(x) \leq 2x^3 - 2$이고 $f(0) = -3$
- **Action**: [12미적Ⅰ-01-01] $x \to 0^+$에서 샌드위치 정리를 적용하여 $f(0)$ 조건과 결합
- **Result**: $x = 1$에서 양쪽 부등식의 경계값이 같음: $6(1)-6 = 0$, $2(1)^3 - 2 = 0$. 따라서 $f(1) = 0$.

> **📝 해설 (Explanation)**
> $x = 1$ 대입: $6(1)-6 = 0 \leq f(1) \leq 2(1)^3 - 2 = 0$.
>
> 따라서 $f(1) = 0$.
>
> $f(x)$는 최고차항 계수 1인 다항함수이고 $f(0) = -3$, $f(1) = 0$.
>
> 부등식 $6x - 6 \leq f(x) \leq 2x^3 - 2$에서 $x = 1$에서 접촉하므로 $h(x) = f(x) - (6x-6)$은 $x=1$에서 최솟값 0을 가짐(양수 범위에서).
>
> 또한 $k(x) = 2x^3 - 2 - f(x)$도 $x=1$에서 최솟값 0을 가짐.

## [Step 2] 도함수 부호 분석으로 f(x) 계수 결정

- **Trigger**: [극값 조건] $h(x) = f(x) - (6x-6)$이 $x=1$에서 극솟값 0을 가짐
- **Action**: [12미적Ⅰ-02-07] $x=1$이 $h(x)$의 극값점이므로 $h'(1) = 0$, 즉 $f'(1) = 6$
- **Result**: $f(x)$는 최고차항 계수 1인 다항함수. $f(1) = 0$, $f'(1) = 6$, $f(0) = -3$. $f(x) = x^3 + ax^2 + bx + c$ 설정 후 연립방정식으로 계수 결정.

> **📝 해설 (Explanation)**
> $h(x) = f(x) - (6x-6)$이 $x > 0$에서 $x = 1$에서만 0이 되고, 이것이 극솟값이면 $h'(1) = 0$.
>
> $h'(x) = f'(x) - 6$이므로 $f'(1) = 6$.
>
> $f(x) = x^3 + ax^2 + bx + c$로 놓으면:
> - $f(0) = c = -3$
> - $f(1) = 1 + a + b - 3 = 0 \Rightarrow a + b = 2$
> - $f'(x) = 3x^2 + 2ax + b$, $f'(1) = 3 + 2a + b = 6 \Rightarrow 2a + b = 3$
>
> $a + b = 2$, $2a + b = 3$에서 $a = 1$, $b = 1$.
>
> $f(x) = x^3 + x^2 + x - 3$.
>
> 검증: $f(1) = 1 + 1 + 1 - 3 = 0$ ✓, $f(0) = -3$ ✓.

## [Step 3] 확정된 삼차함수에 특정 값 대입하여 함숫값 산출

- **Trigger**: [삼차함수 계수 확정 후 특정 값 계산] $f(x) = x^3 + x^2 + x - 3$ 확정, $f(3)$ 산출
- **Action**: [12미적Ⅰ-02-03] 확정된 $f(x)$에 $x=3$ 대입하여 $f(3) = 27 + 9 + 3 - 3 = 36$ 계산
- **Result**: $f(3) = 36$. 정답은 **(1) 36**.

> **📝 해설 (Explanation)**
> $f(3) = 3^3 + 3^2 + 3 - 3 = 27 + 9 + 3 - 3 = 36$.
>
> 따라서 정답은 **36**입니다.
