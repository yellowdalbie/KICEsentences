## [Step 1] 주기함수 정적분의 한 주기 면적 계산

- **Trigger**: [주기함수 정적분 조건] $f(x+3) = f(x)$를 만족하는 주기 3인 함수의 $\int_{-a}^{a} f(x)dx = 13$
- **Action**: [12미적Ⅰ-03-04] 한 주기 $[0, 3)$에서의 정적분 값 계산
- **Result**: 한 주기 면적 $S = \int_0^3 f(x)dx = \int_0^1 x\,dx + \int_1^2 1\,dx + \int_2^3 (-x+3)dx$

> **📝 해설 (Explanation)**
> 한 주기 $[0, 3)$에서의 면적을 계산한다:
>
> $$\int_0^1 x\,dx = \left[\frac{x^2}{2}\right]_0^1 = \frac{1}{2}$$
>
> $$\int_1^2 1\,dx = 1$$
>
> $$\int_2^3 (-x+3)\,dx = \left[-\frac{x^2}{2}+3x\right]_2^3 = \left(-\frac{9}{2}+9\right) - \left(-2+6\right) = \frac{9}{2} - 4 = \frac{1}{2}$$
>
> $$S = \frac{1}{2} + 1 + \frac{1}{2} = 2$$

## [Step 2] 주기함수 정적분의 주기 분해와 나머지 구간 적분으로 a 결정

- **Trigger**: [정적분 대칭 조건] $\int_{-a}^{a} f(x)dx = 13$, 구간이 원점 대칭
- **Action**: [12미적Ⅰ-03-03] 주기함수의 정적분: $n$ 주기 분량의 면적 + 나머지 분량으로 표현
- **Result**: $a = 3q + r$ ($0 \leq r < 3$) 형태로 분해하면 $\int_{-a}^{a} f(x)dx$를 계산할 수 있음.

> **📝 해설 (Explanation)**
> $\int_{-a}^{a} f(x)dx$에서 $f$는 주기 3이므로 구간 $[-a, a]$에서의 적분을 주기별로 분해한다.
>
> $f$가 $[0,3)$에서 양수이고, 주기적이므로 전체 $2a$ 길이의 구간에서:
>
> $$\int_{-a}^{a} f(x)dx = \int_{-a}^{0} f(x)dx + \int_0^a f(x)dx$$
>
> 변수 치환 $t = -x$로 $\int_{-a}^0 f(x)dx = \int_0^a f(-t)dt$.
>
> $f(-t)$를 계산하자. $f(-t+3) = f(-t)$이므로 $f$는 음수 범위에서 주기적으로 이어짐.
>
> 구체적으로 구간 $[-3, 0)$에서 $f(x)$:
> - $x \in [-1, 0)$: $x+3 \in [2, 3)$이므로 $f(x) = f(x+3) = -(x+3)+3 = -x$.
> - $x \in [-2, -1)$: $x+3 \in [1, 2)$이므로 $f(x) = 1$.
> - $x \in [-3, -2)$: $x+3 \in [0, 1)$이므로 $f(x) = x+3$.
>
> 즉 $\int_{-3}^0 f(x)dx = \int_{-3}^{-2}(x+3)dx + \int_{-2}^{-1}1\,dx + \int_{-1}^0(-x)dx = \frac{1}{2}+1+\frac{1}{2} = 2$.
>
> 따라서 $\int_{-3n}^{3n} f(x)dx = 2n \cdot 2 = 4n$.
>
> $\int_{-a}^{a} f(x)dx = 13$에서 $13$이 홀수이므로, 주기 $2S = 4$가 아닌 나머지 부분이 있음.
>
> $a = 3n + r$ ($0 < r < 3$)으로 놓으면 $\int_{-a}^a = 4n + \int_{-r}^r f(x)dx$.
>
> $\int_{-r}^r f(x)dx = 13 - 4n$.
>
> $n = 3$이면 $4n=12$, $\int_{-r}^r = 1$. $r < 3$에서 $\int_0^r f(x)dx = \frac{1}{2}$이면 $\int_{-r}^r = 1$.
>
> $\int_0^r f(x)dx = \frac{1}{2}$:
> - $r \leq 1$: $\int_0^r x\,dx = \frac{r^2}{2} = \frac{1}{2} \Rightarrow r = 1$.
>
> $a = 3 \cdot 3 + 1 = 10$. 정답은 **(1) 10**.

> 검증: $a = 10$, $\int_{-10}^{10} f(x)dx = 4\cdot3 + \int_{-1}^1 f(x)dx = 12 + (\frac{1}{2}+\frac{1}{2}) = 13$ ✓
