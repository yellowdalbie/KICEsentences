## [Step 1] 정규분포의 표준화를 통한 제트점수 산출하기

- **Trigger**: [정규분포 확률구간 조건] 변수 $X$ (토마토 줄기의 길이)가 $\mathrm{N}(30, 2^2)$ 을 따를 때 확률 $\mathrm{P}(27 \le X \le 32)$ 계산
- **Action**: [12확통03-02] $Z = \frac{X - m}{\sigma}$ 표준화 식을 이용해 양 끝 지점들인 $27$ 과 $32$ 의 $Z$ 값을 탐색
- **Result**: $Z_1 = \frac{27-30}{2} = -1.5$, $Z_2 = \frac{32-30}{2} = 1.0 \implies \mathrm{P}(-1.5 \le Z \le 1.0)$

> **📝 해설 (Explanation)**
> 문제에서 임의로 선택한 토마토 줄기의 길이가 평균 $30$, 표준편차 $2$ 인 정규분포를 따른다고 하였으므로 $X \sim \mathrm{N}(30, 2^2)$ 입니다.
> 우리가 구해야 하는 확률의 구간은 $\mathrm{P}(27 \le X \le 32)$ 입니다.
> 표준정규분포표를 사용하기 위해서 $Z$ 값으로 표준화(Standardization)를 해야 합니다.
> $x=27$ 의 경우: $Z = \frac{27 - 30}{2} = -\frac{3}{2} = -1.5$
> $x=32$ 의 경우: $Z = \frac{32 - 30}{2} = \frac{2}{2} = 1.0$
> 즉 $\mathrm{P}(27 \le X \le 32) = \mathrm{P}(-1.5 \le Z \le 1.0)$ 이 됩니다.

## [Step 2] 표준정규분포표를 이용하여 확률 구하기

- **Trigger**: [정규분포 조건] $\mathrm{P}(-1.5 \le Z \le 1.0)$ 의 넓이 연산
- **Action**: [12확통03-02] 정규분포의 대칭성에 의해 넓이를 $0$ 기준으로 두 개로 합산하여 쪼갬
- **Result**: $\mathrm{P}(0 \le Z \le 1.5) + \mathrm{P}(0 \le Z \le 1.0) = 0.4332 + 0.3413 = 0.7745$

> **📝 해설 (Explanation)**
> 표준정규분포 곡선은 $Z=0$ (중앙)을 기준으로 완벽하게 좌우 대칭입니다.
> 그러므로 $-1.5$ 부터 $1.0$ 까지의 넓이는 $0$ 부터 $-1.5$ 까지의 넓이와 $0$ 부터 $1.0$ 까지의 넓이의 합과 같습니다.
> 대칭성으로 인해 $\mathrm{P}(-1.5 \le Z \le 0) = \mathrm{P}(0 \le Z \le 1.5)$ 와 같습니다.
> 결국 전체 구하는 확률은 $\mathrm{P}(0 \le Z \le 1.5) + \mathrm{P}(0 \le Z \le 1.0)$ 이 됩니다.
> 표에서 값을 찾아보면 $0.4332 + 0.3413 = 0.7745$ 가 산출됩니다.
> 따라서 정답은 **②**입니다.
