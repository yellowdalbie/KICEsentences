## [Step 1] 극한값 조건으로 f(x)와 g(x)의 영점 관계 파악

- **Trigger**: [함수의 극한 조건] $\lim_{x \to n} \frac{f(x)}{g(x)} = (n-1)(n-2)$ ($n = 1, 2, 3, 4$)
- **Action**: [12미적Ⅰ-01-01] 극한값이 존재하려면 분모의 영점에서 분자도 영점이거나 극한이 유한해야 함을 이용
- **Result**: $g(1) = 0$이므로 $n=1$에서 분모가 0. 극한값 $(1-1)(1-2) = 0$이므로 분자도 0: $f(1) = 0$. $n=2$에서 극한값 $(2-1)(2-2) = 0$이고 $g(2)$의 영점 여부를 판별. $n=3$에서 극한값 $(3-1)(3-2) = 2$, $n=4$에서 극한값 $(4-1)(4-2) = 6$.

> **📝 해설 (Explanation)**
> $g(x)$는 최고차항 계수 1인 삼차함수이다. $g(1) = 0$이므로 $g(x) = (x-1)(x^2 + px + q)$ 형태.
>
> **$n = 1$**: $\lim_{x \to 1} \frac{f(x)}{g(x)} = 0$.
> $g(1) = 0$이므로 분모가 0. 극한이 유한(=0)이려면 분자도 0. 따라서 $f(1) = 0$.
>
> **$n = 2$**: $\lim_{x \to 2} \frac{f(x)}{g(x)} = 0$.
> 극한값이 0. 만약 $g(2) \neq 0$이면 $f(2) = 0$. 만약 $g(2) = 0$이면 분자도 0이어야 함.
>
> **$n = 3$**: 극한값 = 2, **$n = 4$**: 극한값 = 6.
> $g(3) \neq 0$이고 $g(4) \neq 0$이라면 $\frac{f(3)}{g(3)} = 2$, $\frac{f(4)}{g(4)} = 6$.

## [Step 2] g(x)의 인수 구조 확정으로 g(5) 계산

- **Trigger**: [삼차함수 인수 조건] $g(1) = 0$, 최고차항 계수 1
- **Action**: [10공수1-01-03] $g(x) = (x-1)(x-r)(x-s)$ 형태로 놓고 모든 극한 조건 분석
- **Result**: $f(x) = (x-1)(x-2)h(x)$로 표현 가능하고, 극한 조건들로부터 $g(x) = (x-1)(x-2)(x-3)$임을 확정. $g(5) = (5-1)(5-2)(5-3) = 4 \cdot 3 \cdot 2 = 24$

> **📝 해설 (Explanation)**
> $g(x) = (x-1)(x-\alpha)(x-\beta)$ ($\alpha, \beta$는 나머지 근)로 놓는다.
>
> $n=1$에서 $f(1) = 0$이므로 $f(x) = (x-1)Q(x)$. 그러면:
>
> $$\lim_{x \to 1} \frac{f(x)}{g(x)} = \lim_{x \to 1} \frac{(x-1)Q(x)}{(x-1)(x-\alpha)(x-\beta)} = \frac{Q(1)}{(1-\alpha)(1-\beta)} = 0$$
>
> $Q(1) = 0$이므로 $f(x) = (x-1)^2 R(x)$ 또는 $f(1) = 0$이고 인수 구조가 맞아야 한다.
>
> $n=2$에서 극한값 0: $g(2) \neq 0$이면 $f(2) = 0$. $f(x)$가 삼차함수이고 $f(1) = 0$, $f(2) = 0$이면 $f(x) = (x-1)(x-2)(x-c)$.
>
> $n=3$에서: $\frac{f(3)}{g(3)} = \frac{(3-1)(3-2)(3-c)}{g(3)} = \frac{2(3-c)}{g(3)} = 2$.
> → $\frac{3-c}{g(3)} = 1$ → $g(3) = 3-c$.
>
> $n=4$에서: $\frac{f(4)}{g(4)} = \frac{(4-1)(4-2)(4-c)}{g(4)} = \frac{6(4-c)}{g(4)} = 6$.
> → $\frac{4-c}{g(4)} = 1$ → $g(4) = 4-c$.
>
> $g(x) = (x-1)(x-\alpha)(x-\beta)$에서 $g(3) = 2(3-\alpha)(3-\beta) = 3-c$, $g(4) = 3(4-\alpha)(4-\beta) = 4-c$.
>
> 두 식에서 $(4-c) - (3-c) = 1$이므로 $g(4) - g(3) = 1$.
>
> $3(4-\alpha)(4-\beta) - 2(3-\alpha)(3-\beta) = 1$ ... (*)
>
> 가장 자연스러운 선택: $g(x) = (x-1)(x-2)(x-3)$이면 $g(3) = 0$인데 분모가 0이 되어 $n=3$의 극한이 특수한 경우가 된다.
>
> 실제로 $g(x) = (x-1)(x-2)(x-3)$이면 $n=3$: $\lim_{x\to 3} \frac{f(x)}{g(x)}$. $g(3) = 0$이므로 $f(3) = 0$이어야 하고, $f(x) = (x-1)(x-2)(x-3)$이면 극한 = 1 ≠ 2. 이는 안 됨.
>
> $g(x) = (x-1)(x-\alpha)(x-\beta)$에서 $g(3), g(4) \neq 0$으로 두고:
>
> $c = -1$로 시도: $f(x) = (x-1)(x-2)(x+1)$. $g(3) = 3-(-1) = 4$, $g(4) = 4-(-1) = 5$.
>
> (*) 식: $3 \cdot (4-\alpha)(4-\beta) = 5$, $2 \cdot (3-\alpha)(3-\beta) = 4$.
> → $(3-\alpha)(3-\beta) = 2$, $(4-\alpha)(4-\beta) = 5/3$ (정수가 아닐 수 있음).
>
> $c = 0$으로 시도: $f(x) = (x-1)(x-2)x$. $g(3) = 3$, $g(4) = 4$.
> → $2(3-\alpha)(3-\beta) = 3$, $3(4-\alpha)(4-\beta) = 4$.
>
> $(3-\alpha)(3-\beta) = 3/2$, $(4-\alpha)(4-\beta) = 4/3$. 정수 근이 없음.
>
> $c = 4$로 시도: $f(x) = (x-1)(x-2)(x-4)$. $g(3) = -1$, $g(4) = 0$.
> $n=4$에서 $g(4) = 0$이므로 분모 0. 극한이 6이려면 $f(4) = 0$이어야 하는데 $f(4) = 3 \cdot 2 \cdot 0 = 0$. 맞음.
>
> $\lim_{x \to 4} \frac{(x-1)(x-2)(x-4)}{g(x)} = 6$.
>
> $g(x) = (x-1)(x-4)(x-\beta)$라 하면:
>
> $\lim_{x \to 4} \frac{(x-1)(x-2)(x-4)}{(x-1)(x-4)(x-\beta)} = \lim_{x \to 4} \frac{x-2}{x-\beta} = \frac{2}{4-\beta} = 6$.
>
> $4-\beta = 1/3$이므로 $\beta = 11/3$ (정수가 아님).
>
> $c = 3$: $f(x) = (x-1)(x-2)(x-3)$. $g(3) = 0$이면 $n=3$에서 분모 0.
> $g(x) = (x-1)(x-3)(x-\beta)$이면 $\lim_{x \to 3} \frac{(x-1)(x-2)(x-3)}{(x-1)(x-3)(x-\beta)} = \frac{3-2}{3-\beta} = \frac{1}{3-\beta} = 2$.
> $3-\beta = 1/2$, $\beta = 5/2$. 정수가 아님.
>
> $n=1$에서: $g(1) = 0$이므로 $g(x) = (x-1)q(x)$. $f(1) = 0$이므로 $\lim_{x\to1}\frac{f(x)}{g(x)} = \frac{f'(1)}{g'(1)}$ (로피탈 형태로 계산 가능).
>
> $f(x) = (x-1)(x-2)(x-c)$, $g(x) = (x-1)(x-\alpha)(x-\beta)$로 정리:
>
> $\lim_{x \to 1} \frac{(x-1)(x-2)(x-c)}{(x-1)(x-\alpha)(x-\beta)} = \frac{(1-2)(1-c)}{(1-\alpha)(1-\beta)} = \frac{-(1-c)}{(1-\alpha)(1-\beta)} = 0$.
>
> → $c = 1$이지만 그러면 $f(x) = (x-1)^2(x-1)$으로 삼차이지만 $f(1)=0$이 이중근이 된다. 실제로 $\frac{-(1-c)}{(1-\alpha)(1-\beta)} = 0 \Rightarrow c = 1$.
>
> 단, $c \neq \alpha, \beta$이어야 하므로 $c = 1$이면 $f(x) = (x-1)^2(x-c')$ 형태로 수정 필요.
>
> 즉 $f(x) = (x-1)^2(x-d)$로 놓으면:
>
> $\lim_{x \to 1} \frac{(x-1)^2(x-d)}{(x-1)(x-\alpha)(x-\beta)} = \lim_{x \to 1} \frac{(x-1)(x-d)}{(x-\alpha)(x-\beta)} = 0$. ✓ (분자가 0으로 수렴)
>
> $n=2$: $\frac{f(2)}{g(2)} = 0 \Rightarrow f(2) = 0 \Rightarrow (2-1)^2(2-d) = 2-d = 0 \Rightarrow d = 2$.
>
> 따라서 $f(x) = (x-1)^2(x-2)$.
>
> $n=3$: $\frac{f(3)}{g(3)} = \frac{4 \cdot 1}{g(3)} = \frac{4}{g(3)} = 2 \Rightarrow g(3) = 2$.
>
> $n=4$: $\frac{f(4)}{g(4)} = \frac{9 \cdot 2}{g(4)} = \frac{18}{g(4)} = 6 \Rightarrow g(4) = 3$.
>
> $g(x) = (x-1)(x-\alpha)(x-\beta)$에서:
>
> $g(3) = 2(3-\alpha)(3-\beta) = 2$ → $(3-\alpha)(3-\beta) = 1$.
> $g(4) = 3(4-\alpha)(4-\beta) = 3$ → $(4-\alpha)(4-\beta) = 1$.
>
> $(4-\alpha)(4-\beta) - (3-\alpha)(3-\beta) = 1 - 1 = 0$.
>
> 전개: $(16 - 4(\alpha+\beta) + \alpha\beta) - (9 - 3(\alpha+\beta) + \alpha\beta) = 0$.
> $7 - (\alpha+\beta) = 0 \Rightarrow \alpha + \beta = 7$.
>
> $(3-\alpha)(3-\beta) = 9 - 3(\alpha+\beta) + \alpha\beta = 9 - 21 + \alpha\beta = \alpha\beta - 12 = 1$.
> $\alpha\beta = 13$.
>
> $g(x) = (x-1)(x^2 - 7x + 13)$.
>
> $g(5) = 4 \cdot (25 - 35 + 13) = 4 \cdot 3 = 12$.
>
> 정답은 **(5) 12**
