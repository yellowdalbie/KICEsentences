## [Step 1] 확률밀도함수 조건으로 a 결정

- **Trigger**: [연속확률변수 조건] $P(x \leq X \leq 3) = a(3-x)$ ($0 \leq x \leq 3$), 전체 확률 = 1
- **Action**: [12확통03-01] $x = 0$ 대입으로 전체 확률 조건 활용
- **Result**: $P(0 \leq X \leq 3) = a(3-0) = 3a = 1$이므로 $a = \frac{1}{3}$.

> **📝 해설 (Explanation)**
> $X$는 구간 $[0, 3]$의 모든 실수 값을 가지는 연속확률변수이므로:
>
> $$P(0 \leq X \leq 3) = 1$$
>
> $x = 0$ 대입: $P(0 \leq X \leq 3) = a(3-0) = 3a = 1$
>
> $$a = \frac{1}{3}$$

## [Step 2] 확률밀도함수 도출 및 목표 확률 계산

- **Trigger**: [확률밀도함수 미분 조건] $P(x \leq X \leq 3) = \frac{1}{3}(3-x)$를 미분하여 확률밀도함수 확정
- **Action**: [12확통03-02] 누적분포함수를 미분하여 확률밀도함수 $f_X(t)$ 결정 후 적분으로 확률 계산
- **Result**: $P(t \leq X \leq 3) = \frac{1}{3}(3-t)$를 $t$에 대해 미분하면 $-f_X(t) = -\frac{1}{3}$, 즉 $f_X(t) = \frac{1}{3}$. $a = \frac{1}{3}$이므로 $P(0 \leq X < a) = P\!\left(0 \leq X < \frac{1}{3}\right) = \frac{1}{3} \cdot \frac{1}{3} = \frac{1}{9}$.

> **📝 해설 (Explanation)**
> $P(x \leq X \leq 3) = \frac{1}{3}(3-x)$를 $x$에 대해 미분하면:
>
> $$\frac{d}{dx} P(x \leq X \leq 3) = \frac{d}{dx} \int_x^3 f_X(t)\,dt = -f_X(x) = -\frac{1}{3}$$
>
> 따라서 $f_X(x) = \frac{1}{3}$ (균등분포).
>
> 구하는 확률:
>
> $$P\!\left(0 \leq X < \frac{1}{3}\right) = \int_0^{1/3} \frac{1}{3}\,dt = \frac{1}{3} \cdot \frac{1}{3} = \frac{1}{9}$$
>
> 따라서 $\frac{q}{p} = \frac{1}{9}$이므로 $p = 9$, $q = 1$ (서로소인 자연수).
>
> $$p + q = 9 + 1 = \boxed{10}$$
