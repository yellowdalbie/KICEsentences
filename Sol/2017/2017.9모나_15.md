## [Step 1] 정규분포의 표준화를 통하여 특정 구간의 확률 산출하기

- **Trigger**: [정규분포 조건] $X \sim N(18, 2^2)$인 상황에서 무게가 $16 \le X \le 22$일 확률 요구
- **Action**: [12확통03-04] 표준화 변수 $Z = \frac{X-18}{2}$를 대입하여 주어진 범위를 표준정규분포로 변환하고 표준정규분포표를 참조하여 값 산출
- **Result**: $0.8185$ (구하고자 하는 확률)

> **📝 해설 (Explanation)**
> 수하물의 무게를 확률변수 $X$라 하면, $X$는 평균 $m=18$, 표준편차 $\sigma=2$인 정규분포 $N(18, 2^2)$을 따릅니다.
> 구하는 확률 $\mathrm{P}(16 \le X \le 22)$를 표준화하여 $Z$에 관한 식으로 변환합니다.
>
> $\mathrm{P}\left(\frac{16-18}{2} \leq Z \leq \frac{22-18}{2}\right) = \mathrm{P}(-1 \leq Z \leq 2)$
>
> 표준정규분포의 대칭성과 성질을 활용하여 확률표의 값을 대입합니다.
> $\mathrm{P}(-1 \leq Z \leq 2) = \mathrm{P}(-1 \leq Z \leq 0) + \mathrm{P}(0 \leq Z \leq 2)$
> $= \mathrm{P}(0 \leq Z \leq 1) + \mathrm{P}(0 \leq Z \leq 2)$
>
> 표에 제시된 값을 대입하면 다음과 같습니다.
> $0.3413 + 0.4772 = 0.8185$
>
> 따라서 수하물의 무게가 $16\,\text{kg}$ 이상 $22\,\text{kg}$ 이하일 확률은 $0.8185$입니다.
> 정답은 **④**입니다.
