정답 ③
Step 1 — 주기와 진폭 파악
$f(x) = a\sin(bx+c)+d$에서 진폭 $= |a|$, 주기 $= \dfrac{2\pi}{|b|}$. 그래프에서 최댓값 $= 5$, 최솟값 $= -1$이므로 $|a| = \dfrac{5-(-1)}{2} = 3$, $d = \dfrac{5+(-1)}{2} = 2$.
Step 2 — 주기로 $b$ 결정
그래프 한 주기의 길이가 $\pi$이므로 $\dfrac{2\pi}{|b|} = \pi$, 따라서 $b = 2$.
Step 3 — $a+b+d$ 계산
$a = 3$ ($a > 0$), $b = 2$, $d = 2$이므로 $$a + b + d = 3 + 2 + 2 = 7$$

Step 1 — 연속 조건 설정
$f(x)$가 모든 실수에서 연속이므로 $x = 0$에서 좌극한 $=$ 우극한 $=$ 함숫값. $\displaystyle\lim_{x\to 0^-}f(x) = \lim_{x\to 0^+}f(x) = f(0)$.
Step 2 — 계수 결정
조건 정리 후 $a = 1$, $b = 3$. 따라서 $a + b = 4$.
Step 3 — (이전 문항 이어서)
이전 페이지 컬럼에서 Step 2까지 다뤘으므로 Step 3부터 계속. $f'(x) = 0$의 해를 구하면 $x = \dfrac{1}{3}$, $x = 1$.
